　　在江凌若带队引开丧尸的时间里，其他小队全体增加了移动速度，朝着前方的原定路线快速前进，进度最快的那一个小队，甚至已经到达了指定地点，占领了圆圈一处。
<br/>　　其他小队也在拼命望前赶，原本需要小心注意和缓慢进行的任务，一下子被迫提升了速度，让他们所有人都紧急了起来。
<br/>　　就这样，一个个小队陆续到达圆圈内，快速将其占领而后清理。
<br/>　　何峰也没料到忽然的变故竟然造就了这一任务提前完成，已经随着大流到达圆圈内的他立即联系后续队伍，开始安排其他精英队成员的前进路线。
<br/>　　“A区已经清理完毕。”一人来到何峰面前问道。
<br/>　　“把东西放进去。”何峰头也不抬的吩咐，一边执笔在写着什么，一边时不时看向旁边地图。
<br/>　　旁边的各小队成员人来人往，在黑夜里忙的不可开交，漆黑一片之中他们还带着提前到达的兴奋神情。
<br/>　　时间在战争中往往就是转机！
<br/>　　“第一小队在哪里？”忽的，何峰停下笔问道。
<br/>　　第一小队是徐锋为队长，虽然三战队的成员都打散了与各个精英队成员结合，但有着三战队队长的小队依旧为最强的三个小队，尤其是徐锋所在的第一小队，那明显的杀羽战队风格在这次作战任务中最为显著。
<br/>　　“找不到。”路冰泽的声音从旁边响起，带着显而易见的暴躁：“这两人一个比一个不按规矩办事，等刘玉定来了让他们好看！”
<br/>　　这两人自然就是指的徐锋和江凌若，路冰泽的神隐在这次任务中难得的低调，近乎就是跟着作战计划一步步来，没有任何吐出的地方，原因也很简单。
<br/>　　一是敌军没有人类，神隐战队的优势发挥不出来。
<br/>　　二是此处没有动物，路冰泽的能力跟失效没区别。
<br/>　　所以在这只有丧尸和狼牙战团的地方，路冰泽只能听命行·事，所以对比江凌若的大放异彩和不知道跑去哪的徐锋，路冰泽那叫一个羡慕和不爽。
<br/>　　听到路冰泽的声音，何峰却难得的有些迟疑，顿了顿道：“尽快联系上第一小队。”
<br/>　　路冰泽一愣，问道：“他们难道会有什么问题？”
<br/>　　何峰摇摇头苦笑：“埋骨之地，谁知道会发生什么，但大部分小队都已经到达的情况下徐锋还没到，这不正常。”
<br/>　　路冰泽皱起眉思考起这个问题，与第五小队已经被全员知道了行踪不同，第一小队就好似消失在了这处地方，一片黑暗里无影无踪，对比第一小队的战力和风格，的确太不相符。
<br/>　　路冰泽也不禁认真起来，皱起眉看向前方远处的黑暗，圆圈内的埋骨之地一片漆黑好似什么都没有，却不住的传来阵阵恐怖的声音，分不清是什么，但路冰泽知道那不是丧尸的嘶吼……
<br/>　　与此同时随着时间的一点点推移，第五小队身后的丧尸数量越来越多，到最后近乎整个这一圈大范围内的丧尸都察觉到动静向此处聚集，各类丧尸从四面八方而来，天空中还有持续增多的飞行丧尸，场面一度扩张再这么下去就将失控。
<br/>　　第五小队的成员一路疾走中只觉得丧尸的嘶吼声越来越大，后方时不时利爪能抓到他们后背衣服的情况也增多，甚至还有丧尸开始有了自主意识行动，不像之前显得尤其蠢笨。
<br/>　　江凌若第一时间察觉到这一状况，忽的一跃而起，在半空中踩着旁边一头追上来的丧尸脑袋，‘刷’的一下借力调头朝后方奔去。
<br/>　　紧接着，银色的巨大镰刀垂落在脚下，随着她一步步踩着丧尸的脑袋行走，刀刃狠狠的刺破划走再刺破，或是破开了一只只丧尸的脑后，或是划开了耳后又削起大·片血骨，就此在半空中带出一条黑色的血线。
<br/>　　一下子十只丧尸就此死亡，不过是一瞬间！
<br/>　　江凌若的脚步轻灵，走在一只只丧尸的脑袋上犹如蜻蜓点水，伴随着持续的一刀一划轻轻松松的收割着更多的丧尸性命，十阶新人类的碾压状态彰显到极限。
<br/>　　而当她走到中段的时候，收割的场景终于被制止，因为丧尸已经分散不成队形，江凌若无法在轻松的踩在高处随手收割。
<br/>　　与此同时她的正前方，一只身躯比之周围其他丧尸矮小很多，看上去就像是个人类身材的丧尸拦住了她的去路。
<br/>　　这是一只超级丧尸，更是这一批丧尸中的首领！
<br/>　　丧尸首领并非是狼牙第一次遇到，早在南方尸潮战的时候，楚涵就带着陈少爷与丧尸首领有过一战，而那时候的那只丧尸首领还更为强大，登场更是震撼了在场所有的军官。
<br/>　　人脸黑发，四肢健全甚至还有皮肤与眼神，那已经异常接近人类的样貌，实在让人心慌不已。
<br/>　　当下江凌若眼前的这只丧尸首领亦是如此，但又总给人感觉级别不够。
<br/>　　毕竟被狼牙浩浩荡荡的这么多队伍闯进来，又炸死了大·片地带的丧尸之后，这会儿能聚集起来的丧尸数量还构不成尸潮规模，对应能出现在此处的丧尸首领也不是来自埋骨之地深处，而是边缘地带的小首领一只。
<br/>　　只是哪怕是小首领，其一双森白瞳孔内的神色，也看的不少第五小队的成员心惊胆战。
<br/>　　丧尸首领，莫非真的进化出了视觉？
<br/>　　刷！
<br/>　　江凌若站定在地上，周围的丧尸群都让开一片区域，给江凌若和丧尸首领留下了充分的空间，它们竟然已经懂得避让和尊敬强者。
<br/>　　顿时黑风飒飒之中，就只有一抹银色的巨大镰刀闪着流光，映照出江凌若在黑夜里并不明显的身躯。
<br/>　　而对面则是丧尸首领的一双森白眼瞳，被那柄巨大的银色镰刀的流光反射，如同一双鬼瞳般的明显又阴森可怕。
<br/>　　两人对视相望，像极了两个人类的交锋！
<br/>　　噌——
<br/>　　江凌若紧握手中镰刀，流光炸闪忽然绽放出银色光华！
<br/>　　奔走了一夜又消耗了大量的能量维持领域，江凌若的战力早已下降很多，此时对战眼前的这是丧尸首领却依然气势十足。

A大校长？
<br/>
<br/>        吴校本来站在最边上，毫无存在感，但是他突然被陆寒霆给点名了。
<br/>
<br/>        刷一下，所有人的目光都落在了他的身上。
<br/>
<br/>        大家都不明白他怎么突然被陆总cue了。
<br/>
<br/>        吴校脑袋也是懵的，不过他反应迅速的上前，来到了陆寒霆的身边，“陆总，你好，A大校长在这里，我就是，我姓吴。”
<br/>
<br/>        周副校迅速道，“陆总，你怎么突然找吴校了，这A大学习不行，娱乐也是万年老二，难道这一次A大闯了什么祸？”
<br/>
<br/>        所有人都是这样猜测的，其实吴校自己也有点心虚，难道是他手下哪个小王八羔子给他惹了什么大祸，现在这位陆总要找他算账了？
<br/>
<br/>        陆寒霆深邃的狭眸落在吴校的脸上，嗓音低沉磁性的开腔道，“吴校不用紧张，没什么事，就是叫你过来拍照的。”
<br/>
<br/>        陆寒霆指了指自己身边的位置。
<br/>
<br/>        天哪。
<br/>
<br/>        本来吴校只是站在最边上，那也是他的专属位置，但是现在陆寒霆点名要他站在他身边！
<br/>
<br/>        发生了什么？
<br/>
<br/>        众人倒吸一口冷气，周副校也是脸色大变，不可置信，这个吴校何德何能，竟然得到了陆氏总裁陆寒霆的青睐，他这是中了什么大运？
<br/>
<br/>        吴校也一僵，花了好长时间才明白过来陆寒霆根本就不是找他麻烦，而是抬举他的，不行了，他整个人都开始飘了。
<br/>
<br/>        “吴校，请，我们开始拍照了。”主办方和颜悦色的看着吴校，还做了请的手势。
<br/>
<br/>        大家纷纷往旁边挪，让出了位置，吴校的腰杆突然就直了起来，他咳嗽一声，大摇大摆的站在了陆寒霆的身边。
<br/>
<br/>        陆寒霆左手边是郁闷的周副校，右手边是开心的吴校，“咔嚓”一声，合照拍好了，被挂在了纪念馆的橱窗里。
<br/>
<br/>        吴校很满意，他春风得意的看了一眼周副校。
<br/>
<br/>        周副校气的心口疼，他实在不知道这个吴校在背后使了什么手段，竟然让陆寒霆高看了他一眼，不过他自己有底气，“陆总，待会儿各大高校的拉拉队就要出场跳舞了，我们T大抽到了1号签，第一个出场，还请陆总留下来观看。”
<br/>
<br/>        陆寒霆精致的俊容上并没有什么情绪波澜，他淡淡的回了一个字，“好。”
<br/>
<br/>        周副校又将目光落在了吴校身上，“老吴，听说你们A大出了一个校花夏夕绾，今天也要领舞拉拉队，这个夏夕绾曾经想进我们T大，但是我给拒绝了，你怎么把她收下了？”
<br/>
<br/>        吴校整个人神清气爽，靠着陆寒霆这尊大佛，他觉得自己的战斗指数都蹭蹭上涨了，“老周，有一个词不是叫做有眼无珠吗，我收下夕绾就是让你知道后悔的滋味是什么。”
<br/>
<br/>        “哼！”周副校不屑的冷哼了一声，那个夏夕绾有什么好的，他怎么可能后悔，“历年来我们T大的拉拉队都是最好看的，也是所有人最期待的，老吴，你不要夸大海口，免得到时被打脸很难堪。”
<br/>
<br/>        吴校，“老周，你也知道那是历年，那是以前了？不要将过去的辉煌拿出来炫耀，人要向前看。”
<br/>
<br/>        “你！”周副校气死了。

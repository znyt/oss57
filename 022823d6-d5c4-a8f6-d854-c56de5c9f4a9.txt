　　“怎么小子，有本事开口说要杀尽我们七大宗门的弟子，怎么无力做到？现在本尊就在你的面前，倒是要看看，你如何能杀本尊？”
<br/>　　突然之间，一道神秘的声音响起，这声音，似乎便是从周围之中的各个角落中响起。
<br/>　　令人听见，都无法分辨出，这声音，到底是来自何方。
<br/>　　“躲躲藏藏，鼠辈罢了，既然你不自己出来，那我就将你抓出来！”
<br/>　　听到这神秘声音，楚风眠也不动怒，只是冷笑一声，突然大手一挥，便是化为战龙之爪，直接向着冰谷的方向，抓了过去。
<br/>　　在这战龙之爪之下，突然冰谷之上，一道金色身影，突然出现。
<br/>　　这是一尊身着金衣的男子，被一爪抓出，他的脸上都是露出了几分惊讶，不过很快便是平息过来，露出几分淡淡冷笑。
<br/>　　“看来敢杀我们圣王宗的人，倒是还有些本事。”
<br/>　　这金色身影一举一动之中，都蕴含无上武道真理，稍微一动，都是极为纯粹的武道展示，如同武神一般。
<br/>　　他的身影周围，便是被一道武神虚影所笼罩着。
<br/>　　“又来一位高手。”
<br/>　　看到这金色身影的一刻，楚风眠便是看得出，此人的实力，不在剑祖荒之下。
<br/>　　尤其是他的一身武道，倒是年轻一代中，楚风眠所见到过的最强者。
<br/>　　“圣王宗，鹿战神！”
<br/>　　寒月漓看到这身影，眼神中闪过一丝杀意，冰冷开口道。
<br/>　　“我刚刚来到这天羽神域中，便是被你们圣王宗的人追杀，今日你还敢追到这冰裂谷来。”
<br/>　　“哦？这鹿战神，也追杀过你？不用动怒，今日这些蝼蚁，来一个我就杀一个，谁敢来，谁就得死。”
<br/>　　楚风眠坐在宝车上，听到寒月漓的话，缓缓的开口道。
<br/>　　这话，虽然是对寒月漓诉说，却是在场所有人，都是听得一清二楚，其中的杀机，丝毫不假。
<br/>　　“哼，蜉蝣撼树，不足量力，小子，自信不错，但是自信过了头，可就是狂妄了。”
<br/>　　鹿战神冷冷的开口道，他的目光突然看向一边的剑祖荒。
<br/>　　“剑祖荒，没想到你们天剑宗，居然会被一个小辈，逼迫到了如此地步，你根本不配跟我相等。”
<br/>　　“不过念在你跟我，同是七大宗门弟子的份上，今日我可以救你一次，让你有机会归来，再来找着小子一战。”
<br/>　　鹿战神一边说着，一边便是猛然抓起剑祖荒，要将剑祖荒带走。
<br/>　　“放肆，鹿战神？你算是一个什么东西，自封战神？也敢在我的面前带走人，你想要来就这剑祖荒，正好，那你就跟他一起死了，让你们在黄泉路上有个陪伴！”
<br/>　　说完之间，楚风眠突然出手了，他的大手一挥，一掌只见猛然冲着鹿战神劈了过去。
<br/>　　今日楚风眠的心中，早已经是愤怒至极，要打开杀戒，谁敢来救人，楚风眠便是要杀了谁，更何况这鹿战神，曾经追杀寒月漓。
<br/>　　于情于理，这鹿战神，今日都得死。
<br/>　　“一个不知天高地厚的小子，也敢向我出手？”
<br/>　　鹿战神的眼神中，都是露出了几分杀机，似乎也有些吃惊，楚风眠居然敢直接跟他出手。
<br/>　　在这天羽神域中，鹿战神的名号，无人不知无人不晓，无数武者见到他，都是恨不得退避三舍，从未有人敢直接对他出手。
<br/>　　但是这鹿战神的名气再大，在楚风眠的面前，也不过只是一个死人罢了，只要是跟楚风眠为敌之人，无论实力再强，背景再深，都是一个死人。
<br/>　　这鹿战神，也不例外。
<br/>　　他现在杀了二十多位天剑宗弟子，所有精血，都被楚风眠凝聚，又是再度凝聚三千滴远古战龙精血，足足已经是凝聚了三万六千滴远古战龙精血，随时可能再进一步。
<br/>　　这鹿战神，在楚风眠看来，也就是来送精血的，一样会被楚风眠轰杀，吞噬。
<br/>　　“战龙震怒！”
<br/>　　楚风眠一掌横空劈下，直接化为了漫天战龙，当空冲起，轰击鹿战神。
<br/>　　“龙族武技？看来传闻你得到颜古龙帝洞府的事，倒是真的了，颜古龙帝，无愧于是曾经位于九域之巅的人物，一个默默无闻的小子，得到了他的传承都能一飞冲天。”
<br/>　　“只可惜小子，今日你碰到我，便是你最大的不幸，今日我就杀了你，也算是为我圣王宗弟子报仇！”
<br/>　　鹿战神看到楚风眠出手，冷笑一声，凝聚身上灵力，猛然一拳，便是冲着空中打出。
<br/>　　他的一拳打出，在他身上笼罩的武神，都是同样，凝聚力量，一拳轰出。
<br/>　　看似平常的一拳，其中蕴含的却是武道真谛，一拳向着楚风眠轰击过来，好像是要将楚风眠都给硬生生的贯穿。
<br/>　　天上群龙，都在这一拳下，尽是灰飞烟灭。
<br/>　　“不过如此。”
<br/>　　鹿战神冷喝一声，再度凝聚灵力，又是一拳冲着楚风眠轰击过来。
<br/>　　“既然如此，那你就死吧！”
<br/>　　这一拳的力量，比起刚刚那一拳，还要强大几分，两拳的力量合二为一，直接向着楚风眠轰击过来。
<br/>　　“轰！”
<br/>　　一声巨响，这一拳的力量，直接向着楚风眠轰击而来，但是在接近到楚风眠身旁三尺的一刻。
<br/>　　在这战龙宝车上，罗伞打开，七彩霞光便是笼罩下来，这一拳的力量，全部都被七彩霞光所抵挡，根本无法撼动这战龙宝车。
<br/>　　“恩？居然足以抵挡住我的一拳，这是什么灵器？宝车？天魔宝车？”
<br/>　　鹿战神看向楚风眠所座的战龙宝车，都是满脸惊讶，有些震惊。
<br/>　　他的全力，就算是一件天级灵器，都很难全部抵挡住，可是面对这战龙宝车，他的力量，居然都无法撼动分毫。
<br/>　　“剑祖荒，怪不得你们居然败的这么惨，这小子的手中，居然还有这等至宝？”
<br/>　　鹿战神有些惊讶的说道，似乎也有些明白过来，剑祖荒为何败的如此之惨。
<br/>　　“不止是天魔宝车，此人的手中，还有龙族至宝，祖龙至尊剑。”
<br/>　　剑祖荒听到鹿战神的话，也是出言提醒道。
<br/>　　“天魔宝车，祖龙至尊剑，这两件可都是天级灵器中的至宝，这小子何德何能，居然独占？”
<br/>　　鹿战神听到这两个名字，眼神中也是露出了几分贪欲。

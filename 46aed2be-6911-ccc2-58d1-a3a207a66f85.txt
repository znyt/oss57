        她发现，他每次生气都会对她动粗，对她毫不客气，几乎到了冷酷无情的地步。
<br/>
<br/>        只不过，到现在，她都没想明白，他为何要生气？
<br/>
<br/>        明明，这段时间，他们都相处得好好的。
<br/>
<br/>        “大哥……”你到底是怎么呢？
<br/>
<br/>        童筱颖的话还未说完，他俊脸阴沉，霸道地将她扼制在床上动弹不得。
<br/>
<br/>        “你背着我吃了什么？”他疾言厉色地质问。
<br/>
<br/>        傅夜沉的语气里充斥着愤怒与不满，恼火得脸色格外铁青。
<br/>
<br/>        童筱颖这才恍然大悟，双手试图推开他的身子，但他压根就不给她挣脱的机会。
<br/>
<br/>        她被痛得挤出了眼泪，柳眉拧成一团，贝齿咬着下唇，心慌又心痛地与这个男人四目相对。
<br/>
<br/>        每次看到她的眼泪，他的心都会像被针扎了般刺痛难耐。
<br/>
<br/>        可是，她为什么每次都要用她自己的方式来折磨他？
<br/>
<br/>        他想要她怀上他的孩子，就这么难？
<br/>
<br/>        当初，她在他爷爷面前那般信誓旦旦的话，敢情都是用来糊弄他的吗？
<br/>
<br/>        所有不好的情绪因为愤怒而涌上心头，让他在瞬间变得格外暴戾不安。
<br/>
<br/>        童筱颖哭得梨花带雨，双手不停地捶打着傅夜沉那结实的胸膛。
<br/>
<br/>        但他完全没有要放开她的意思，直接抓住她的双腕，重重地按向了两边，而被他抓着的地方，肌肤已经开始失去了血色。
<br/>
<br/>        可想而知，他下手有多重。
<br/>
<br/>        童筱颖的眼中充盈着泪水，瞳仁里是傅夜沉那张帅得无可挑剔的脸，但此刻却让她觉得他很陌生。
<br/>
<br/>        他很疯狂地对待她，她只能呜咽着控诉：“大哥，你居然派人监视我！”
<br/>
<br/>        不然，他又是怎么知道她偷吃了事后药。
<br/>
<br/>        对，她的重点与他的重点，不在一个问题上。
<br/>
<br/>        她在乎的是他居然派人暗中监视她，而他在乎的是她居然背着他偷吃事后药。
<br/>
<br/>        “如果不监视你，我还真不知道，你打心底地就不愿意怀上我的孩子！”他单手刚放开她的手腕，下一秒又钳住她的下颚，将她的面庞与自己拉近。
<br/>
<br/>        那他为什么要派人监视她？
<br/>
<br/>        说到底，他还是不愿意相信她吧！
<br/>
<br/>        他不相信，她爱他，对她回到他身边充斥着各种怀疑！
<br/>
<br/>        明明她把身心都给了他，为什么他还要这么对她？
<br/>
<br/>        童筱颖的脑海里，全是这些糟糕的设想。
<br/>
<br/>        心痛与心塞并齐，她矢口否认，说了不该说的气话：“是！我就是不愿意怀上你的孩子！”
<br/>
<br/>        “很好！你倒是承认得爽快！那么，童筱颖，你给我记住了，从今天开始，你不用上班了。你的任务，就是生下我傅夜沉的继承人！一个不够，生两个！生到我满意为止！”他的眼睛里瞬间布满了血丝，愤怒令他像嗜血般疯狂地折磨起她。
<br/>
<br/>        童筱颖只觉自己痛不欲生，她不停地摇头，不停地推搡，不停地抗拒，甚至哭哑了嗓音，苦苦求饶：“不要！你放开我！放开我！大哥，我求你放开我！求你！我求求你！我好痛！真的好痛！大哥，我求你放开我……呜呜……唔——”
<br/>
<br/>        他突然吻了下来，将她的哭喊声堵回了肚子里，直到这一切结束。
<br/>
<br/>        童筱颖满脸泪痕，披头散发，衣衫不整地抱膝坐在床上。
<br/>
<br/>        她的目光涣散，找不到焦点。
<br/>
<br/>        他不愿意面对这样的她，整理好了自己身上的着装后，二话不说，便毅然转身下了楼。
<br/>
<br/>        傅夜沉的气还未消，而童筱颖的身心都在痛着。
<br/>
<br/>        她和他之间，果然是相爱容易，相处难。
<br/>
<br/>        甚至，她还觉得，其实，她与他压根就是性格不合。
<br/>
<br/>        他不生气的时候，世界祥和，他生气的时候，就好像是世界末日。
<br/>
<br/>        她身上的衣物被他撕破了，她只能老老实实地待在床上，哪都去不了。
<br/>
<br/>        所以，她在这床上一坐，就呆呆地坐了一上午。
<br/>
<br/>        脑子里只有一个问题，那就是大哥到底在堤防她什么，以至于要派人对她暗中监视？
<br/>
<br/>        直到落以柔捧着一套里里外外都有干净的新衣服走上来，轻轻地放在了她的床边。
<br/>
<br/>        童筱颖才面无表情地问道：“他去哪呢？”
<br/>
<br/>        “傅总上午就出去了，到现在还未回，只是电话吩咐我，说私宅的刘婶送衣服来了，让我去楼下接衣服上来送给你。”落以柔慢条斯理地说，双眼有意无意地打量着童筱颖。
<br/>
<br/>        童筱颖现在这副模样，披头散发，两眼无神，衣物破烂地挂在身上，手腕和大腿上还有几道淤青，狼狈得像是被谁强暴了一样。
<br/>
<br/>        落以柔难以想象，刚刚傅夜沉是怎么对待她童筱颖的。


<br/>　　<br/>　　    奈何桥下，忘川河旁站着道俏丽的身影，这人伸出两根手指扫了下额头前的秀发，淡淡的说道：“谁让你们动我弟弟的了······”<br/>　　<br/>　　    望乡台上，鬼王，夜游和孟婆同时顺着声音的来源望去，就看见了奈何桥下的王冬至，和她一同来的还有个青年，看见他俩后孟婆到没什么反应，鬼王和夜游同时脸色大变。<br/>　　<br/>　　    王冬至顺着忘川河上了奈何桥，又来到望乡台上，她旁边的菜刀文看见晕在地上的王惊蛰，叹了口气说道：“这就是你让我来阴曹地府的原因了？”<br/>　　<br/>　　    菜刀文是先一步进入阴曹地府的，但进来之后他人却消失了，和他前后进来的没人看见他去了哪，没想到此时却忽然和王冬至出现了。<br/>　　<br/>　　    上了望乡台，看着已经喝了孟婆汤后倒在地上的王惊蛰，冬至瞥了一眼鬼王和夜游，说道：“你们也就这点小伎俩了，拿不了人家的魂就灌了一碗孟婆汤，现在人也倒了，你们还能怎么样？真要把人给送到十八层地狱里关着么？呦呵，你看把你们厉害的，是不是还得叉个小腰，在牛逼一会啊？”<br/>　　<br/>　　    菜刀文捂着脸小声说道：“这嘴，也太损了”<br/>　　<br/>　　    “你有意见？”王冬至斜了着眼睛问道。<br/>　　<br/>　　    菜刀文连忙摇头说道：“不敢，不敢，你说的全对，就是这么回事”<br/>　　<br/>　　    鬼王，夜游被她挤兑的脸红一阵白一阵的，百天后他俩才咬牙问道：“你怎么又来阴间了？”<br/>　　<br/>　　    “想来便来了，怎么样？”王冬至伸出双手递了过去，嘲讽着说道：“要不你们也把我给抓了，看看能不能把我的魂魄也给抽出来，或者实在不行也给我喝一碗孟婆汤？”<br/>　　<br/>　　    孟婆皱眉说道：“阴间什么时候这么随随便便的了，说来就随便能来的了？”<br/>　　<br/>　　    “对别人不行，是规矩，但对我来说却不一定”王冬至冲着鬼王和夜游问道：“你们说呢？”<br/>　　<br/>　　    “你别太过分了······”夜游皱眉说道。<br/>　　<br/>　　    “过分？过分又如何呢，过分也是建在实力上的，当初那个叫余秋阳的单人一剑闯你们阴曹地府，你不也一样拦不住么？到我这里也没有例外”王冬至指了指地上的王惊蛰说道：“怎么样，我想把人带走，有没有问题？”<br/>　　<br/>　　    夜游不吭声了，鬼王似乎也拿不定主意，王冬至眯着眼说道：“你们拿不了他的三魂七魄，就给他灌了一碗破汤喝，就算他擅闯阴间这也算是受了责罚了吧？还想怎么样，真的把人送进大牢里去才算作罢？我可事先提醒你们一声，孟婆汤虽然喝了，也不见得能把他的魂魄怎么样了，到时候小心自己别进退两难”<br/>　　<br/>　　    孟婆怒声说道：“我的汤，什么时候要你来做评判的了？”<br/>　　<br/>　　    王冬至冷冷的笑了一声，也不说话，忽然蹲下身子，伸出一根手指在王惊蛰的眉心上快速的写了一串符箓。<br/>　　<br/>　　    “唰”悄然间，王惊蛰的三魂七魄从他体内渐渐浮现出来，鬼王，夜游和孟婆还有菜刀文看见之后，全都豁然呆住了。<br/>　　<br/>　　    寻常人的魂魄看起来就像是一道虚影，但也是人形，可王惊蛰飘出来的这道魂，从上到下都布着印记，仿佛被人用笔在其上勾画出来的图案一样。<br/>　　<br/>　　    “你确定，你们拿下他能有用？不怕给自己招来一身麻烦么，阴曹地府也脱不了大道因果吧·····”<br/>　　<br/>　　    鬼王和夜游还有孟婆都惊异不定的看着王惊蛰的魂魄，那是被上天所诅咒的因果，难怪先前无法拘拿他呢，确实阴帅也不够资格来干涉这天谴。<br/>　　<br/>　　    虽然说阎王让人三更死，就不会留你到五更天，但阎王再大也大不过天啊，王惊蛰身受天谴活不过三十岁，那在这之前谁也没有能力让他早走一步，阴帅自然也就没办法抽出他的魂魄了，毕竟一个人要是没了魂，那不就是一个死人了么。<br/>　　<br/>　　    鬼王和夜游有些进退两难了，带走喝了一碗孟婆汤的王惊蛰也没用，魂抽不出来，就不能让一个生人进入炼狱关押。<br/>　　<br/>　　    但是不缉拿他的话，又对阴间律法没有了交代。<br/>　　<br/>　　    王冬至接着说道：“你们啊，是不是想的太多了？他左右都是一死，还差这几年时间么？我先把人带走了，等到他天谴到来之时人不是还得进入阴间么，到时你新账旧账一起算不就得了？非得拘泥于现在么？何苦呢，呵呵······再说了，你们真要是和我死扛到底，我也不怕留在阴间和你们周旋，人家都叫我扶弟魔，这话说的肯定是有原因的，为我这个弟弟，我真不介意狠狠的搅合你们”<br/>　　<br/>　　    “两位大人，睁一只眼闭一只眼吧？”菜刀文催促道。<br/>　　<br/>　　    孟婆叹了口气，转过身说道：“这事，本来和我也无关，你们商量去吧”<br/>　　<br/>　　    鬼王和夜游面面相觑，一阵头疼，先前他们还对拿下王惊蛰抱着一本正经的态度呢，现在却是真不太好办了。<br/>　　<br/>　　    “把人背上”王冬至说道。<br/>　　<br/>　　    “啊？”<br/>　　<br/>　　    王冬至说道：“我叫你来干什么来了？我用得到你保驾护航还是怎么着？背人啊·····”<br/>　　<br/>　　    菜刀文无语的把地上的王惊蛰给搀了起来背到背上，鬼王张了张嘴欲言又止，实在不知道话该从哪说起了。<br/>　　<br/>　　    王冬至笑眯眯的说道：“你们也别觉得为难，这次我带他走，大不了以后我向你们保证再也不来阴曹地府就是了”<br/>　　<br/>　　    夜游顿时说道：“你说的？”<br/>　　<br/>　　    “呵呵，唯小人和女子难养也，这也是我说的········”<br/>　　<br/>　　    王冬至和菜刀文带着昏倒了的王惊蛰走了，没有朝着良水河的那道鬼门，而是奔着另外一个方向走了。<br/>　　<br/>　　    路上，菜刀文背着王惊蛰，诧异的问道：“你以前总来阴曹地府不成？”<br/>　　<br/>　　    “来过好几次了，每年都跑一趟，每次都闹腾他们一顿”王冬至叹了口气。<br/>　　<br/>　　    这些年，王冬至每年都会走一两趟阴曹地府，为的就是王惊蛰身上的那道天谴，她是真操碎了心，父女两人，王冬至和王仙芝几乎踏遍了阴阳两地，一直都在琢磨到底用何种方式，才能驱除掉王惊蛰身上的天谴。<br/>　　<br/>　　    王仙芝曾经说，也许可以为王惊蛰逆天改命，但可惜改命的方式如今无人得知，只有多年前古井观门人改过一次，但也是受了一道天道气运之后才成的，更可惜的是，据说那逆天改命的风水大阵已经被埋在了楼兰古国的地宫下，世上似乎再也无人得知了。<br/>　　<br/>　　    最后，王冬至想到了一个法子，让王惊蛰喝上一碗孟婆汤，试试让他忘记今生之前的所有，重新捋一遍，看看会对其魂魄产生什么影响。<br/>　　<br/>　　    所以，当一天之后，在朝歌王村的家中，王惊蛰醒来问的一句话就是。<br/>　　<br/>　　    “我是谁······”<br/>　　<br/>　　    “我在哪里？”<br/>　　<br/>　　    是的，王惊蛰什么也不记得了。<br/>　　<br/>　　    言情剧里，最狗血最烂大街的桥段出现了，男主角失忆了！

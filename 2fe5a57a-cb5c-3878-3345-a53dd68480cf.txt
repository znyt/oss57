　　第578章
<br/>　　李婉儿的声音并不大，但在此刻却足够被所有人听见，因为大家的注意力都集中在她身上。
<br/>　　密密麻麻围着山头的敌人出现短暂的停顿，人群哗然，开始低头议论着什么。
<br/>　　“不可能，帝国已没有公主，明玉公主更是早已死了。”张明理大声喊了一句。
<br/>　　他说的是实话，当初，林少羽率领起义军攻入长安，为了复仇林少羽亲手用长枪杀了明帝。帝国天子尚且如此，更何况皇宫中的嫔妃公主。能够自杀了结性命的，反而是少数的幸运儿。至于一些来不及自尽的，难免就成了男人胯间的玩物。
<br/>　　这是帝国皇族的耻辱，又何尝不是每个帝国子民的耻辱。所以，大家都不愿意聊起这件事。
<br/>　　而明玉公主，她当初赴草原和亲，很是在民间收割了一波声望。多少读书人为她写诗立传，赞扬她的牺牲精神。但后来戎族王死了，大家也以为明玉公主死在了草原上。
<br/>　　至于她被程大雷劫走的事情，却是很少有人知道。帝国朝廷当然也不会大力宣传这件事，毕竟也不怎么露脸。
<br/>　　李婉儿站在山头，迎风而立，目光坚定：“我是大武李氏之后，先帝的第十七个孩子，帝国的第三位公主，你们……谁敢杀我。”
<br/>　　人群慌慌乱乱，陷入犹豫中。李婉儿的态度让人不得不相信，她口中所言是真实的。世上不缺乏冒充公主想要混吃混喝的人，可不是每个人都有李婉儿身上这种气质。
<br/>　　这便是皇家气派，令人匍匐，令人心生敬意。
<br/>　　皇权衰落，各路诸侯都窥视着天下神器，其实很多人都不再将皇族当会事。但今天场上最不缺的就是读书人，读书人每天嘴里说着忠孝礼智信，这『忠』可是排在第一位。
<br/>　　难道我们今天要杀的，真是一位帝国公主。
<br/>　　哪怕仅仅是一个猜测，也令太多人不寒而栗，握剑的手不自觉的颤抖，让他们行事与过往的信念所悖，并不是一件容易的事。
<br/>　　“尔等是想做那乱臣贼子么？”
<br/>　　李婉儿又喝了一声，这一刻没人敢与她的目光直视，书院的弟子纷纷都垂下头去。
<br/>　　不过，有人不想当乱臣贼子，但场上也有人根本不将所谓皇族看在眼里。
<br/>　　这些人，有一批是东海一地的江湖豪强。江湖人将脑袋绑在裤腰带上干活，什么王法朝廷，他们眼里根本没这些东西。
<br/>　　二便是孟玄青和夫子少数几个十九国之后。他们当初的目的就是推翻李氏王朝，重新恢复十九国的荣光。就算到今日野心淡了，也不会太把皇族当真。
<br/>　　说也奇怪，夫子弟子满天下，各个口里说的都是忠君爱国，但教授他们的夫子，本人对这些东西是不相信的。
<br/>　　还有一方，就是戎族。
<br/>　　戎族很多次把帝国军摁在地上摩擦，对于手下败将，他们向来缺乏敬意。
<br/>　　这最后一方的人数最少，只有一人，脾气也是相当暴躁。
<br/>　　他是燕不归。
<br/>　　在程大雷突破剑阵的时候，燕不归也趁机闯了出来。李婉儿的话他自然也听见了，但根本没往心里去，拎着大剑便闯了过来。
<br/>　　程大雷单枪匹马接住了他，匹夫剑甩开，独斗他的重剑。刚才已短暂的交过手，燕不归并非程大雷的对手，如今突破剑阵，程大雷更不会怕他。
<br/>　　李莫愁手摁银针，正寻找机会出手，用冰魄神针送燕不归下地狱。没理由这冰魄神针对孟玄青无效，对燕不归也无效。
<br/>　　这个时候，孟玄青和夫子同时向李婉儿攻来。二人心中想得明白，程大雷杀他们一位公主，他们也杀一个帝国公主，如此一了百了。
<br/>　　让李莫愁独立应战二人，的确有些强人所难。程大雷将燕不归逼退后，赶忙来到李婉儿身边护驾。
<br/>　　他和李莫愁联手，对抗燕不归三人。不过，燕不归三人之间的关系，也并非是合作。燕不归也很想杀掉二人，但是，他有这个想法，却没有这个力量。
<br/>　　五人之间完全是一团混战，打得难解难分，也是极其凶险。
<br/>　　李婉儿却是太高看了自己，如果大家真的将皇族威严当会事，也就没有今天帝国二帝十八王的局面。
<br/>　　程大雷暗暗心忧，这群读书人回过味来，再次联手攻上来，自己可未必能应对。他想要擒贼先擒王，诛杀夫子立威。但夫子这老滑头，也绝对不是程大雷想杀就能杀死的。
<br/>　　更何况，身边还有个搅屎棍一样的燕不归。从来英雄难过美人关，无论他以前是不是英雄，但在黑贞死后，他都算不上英雄了。他只是个想杀死所有人的疯子。
<br/>　　“冲啊，杀了这恶贼！”
<br/>　　不知谁喊了一嗓子，只是稍微有些犹豫的书院弟子和江湖豪强都冲了上来。
<br/>　　程大雷现在的情况不是很好，不过，他确实还有东西没拿出来。实在不行，自己还能兑换几个顶级人物出来，把战局搞乱。
<br/>　　并州一战，程大雷积累了数量不少的恐惧值，现在还未用尽。在上岛之前，他就将这当做最后一个法子，绝世人物出现与否还需要一定概率，兑换几个顶级人物是没有问题的。
<br/>　　但因为隐藏属性的缘故，每出现一名顶级人物，就要相应实力的敌人，这才是程大雷一直忍着不用的原因。
<br/>　　可此时此刻，也没有什么其他办法，程大雷只能如此做了。
<br/>　　“呔！”
<br/>　　程大雷一剑将三人逼退，口中大喝一声，先将三人唬住。
<br/>　　“本当家杀戎族，闹长安，闯草原，什么大风大浪没见过，凭你们几个江湖上的软脚虾也想杀了本当家。”程大雷大喝道：“你们以为本当家是个武夫，本大家其实他妈的是个法师。”
<br/>　　“看我五雷法啊！”
<br/>　　不得不说，所有人都被程大雷搞蒙了。关于他呼风唤雨，撒豆成兵的事情，江湖上的确也有些传言。但是真是假，大家也就是听说得多，见过的少了。
<br/>　　正这时候，一艘大船靠岸，一群人哗啦啦冲上海岛。
<br/>　　为首之人抬起眼睛一瞧：咦，怎么这么多人。
<br/>(本章完)

        &lt;h3&gt;第三卷第一百五十章  爱你，我愿意解开心结&lt;/h3&gt;
<br/>
<br/>        凌辰心里一直很不舒服，但是这几天他倒是想通了，宁雅有男朋友也是很正常的。
<br/>
<br/>        想了想这么多年，就像是刘诚之说的，他们一直都是在暖昧的状态，似乎他就没有对宁雅正儿八经的表白过，翻开手机，看了看那么多的情话，连一句“我爱你！”都没有说过。
<br/>
<br/>        他们俩个人就是处在一种尴尬的的地步，谁也不肯向前走一步。
<br/>
<br/>        他怕宁雅会拒绝他！
<br/>
<br/>        而宁雅，大概是因为家庭的原因吧！
<br/>
<br/>        宁雅有了男朋友也是应该的，而宁雅有能力也是有目共睹的，他不允许任何一个人说宁雅的不好。
<br/>
<br/>        他的性格冷情，对任何事情都看得比较透彻，要不然那一干年龄比他大的老狐狸能对他服服帖帖。
<br/>
<br/>        所以在通过这一段时间之后，他反而想通了一下事情，从心底不去抱怨宁雅了。
<br/>
<br/>        现在，他只能通过自己的努力，提示自己的价值，假如有一天能够碰到像宁雅这样的女孩子，才有资格站在她的身边。
<br/>
<br/>        他们已经定好了机票，新年已经快要到了。
<br/>
<br/>        和董亚子分别之后，刘诚之似乎是满腹心事。
<br/>
<br/>        “诚之，你怎么了，一副无精打采的样子？舍不得离开这里了？”凌辰看着刘诚之。
<br/>
<br/>        “也不是啦，就是想到要走了，心里还有一点乱乱的感觉。”
<br/>
<br/>        “是舍不得董亚子？”
<br/>
<br/>        “那有的事情啊？你别胡说了。”刘诚之急了。
<br/>
<br/>        “没有？没有你着急什么？这一趟贵州之行还很有意义哦，我们诚之的恋爱开启了新篇章。”
<br/>
<br/>        “我天天吃你喂的狗粮吃饱了。”刘诚之委屈的说。
<br/>
<br/>        “哦，那这是准备给我喂狗粮了？”
<br/>
<br/>        “你还需要别人给你喂，天天表白小雅姐，太阳很温暖，风儿很温柔，身边没有雅儿，可是雅儿在我的心头。”刘诚之撇着嘴说。
<br/>
<br/>        “我有这么肉麻吗？”
<br/>
<br/>        “怎么没有？我耳朵都堵住了。”
<br/>
<br/>        “哈哈哈，以后怕是不会再说的了，所有的情话都说给了宁雅。”
<br/>
<br/>        “我觉得吧，小雅姐不可能那么轻易的找男朋友。她身边没有优秀的人是不可能呢，如果有男朋友了不可能和你天天聊天打电话。”刘诚之皱着眉头分析着。
<br/>
<br/>        凌辰听说刘诚之的分析心里一动。
<br/>
<br/>        “不过没有看见情况谁也说不清楚，不如我们去北京看看？”刘诚之提议。
<br/>
<br/>        “你机票已经都买好了，怎么去？”凌辰被刘诚之说动了。
<br/>
<br/>        “改签啊，或者回去了再去，你去不去？”刘诚之看着凌辰。
<br/>
<br/>        “算了吧，先回去再说，马上要放假了，这么久都没有在公司，还是回去看看，再说家里已经催得紧了。”凌辰想着在放假前的一天可以去看看。
<br/>
<br/>        “放假了，那就得你一个人去了，我要回家。”刘诚之放假是要回家的。
<br/>
<br/>        “行，放假了我去看看吧，再说我们还是老同学。”凌辰终于是给自己找到了一个借口，准备放假去一趟北京，看看宁雅，如果宁雅还推说她忙的话，那么他也就彻底放下。
<br/>
<br/>        凌辰想了想，并没有给宁雅打电话，也没有发信息，而是找陈子昂。
<br/>
<br/>        “陈总，好久不见！”
<br/>
<br/>        “凌总，好久不见。”陈子昂已经从宁雅的口中知道了凌辰有女朋友的事情了。
<br/>
<br/>        “陈总，春节休几天假啊？”
<br/>
<br/>        “常规假期，凌总呢？”
<br/>
<br/>        “我们也是常规假期。陈总休假是在北京吗？”
<br/>
<br/>        “还不一定呢，看情况吧？凌总是要来北京吗？”
<br/>
<br/>        “哦，我也不一定呢，那个，宁雅也是和你们一起放假吗？”
<br/>
<br/>        “呵呵，是啊，这几天特别忙！凌总为什么不直接给宁雅打电话？”陈子昂对那种藕断丝连的感情特别厌恶，知道凌辰有女朋友还撩宁雅从心里有点反感，但是凌辰又帮过她。
<br/>
<br/>        “啊，我也就是随口问问。”凌辰心里有些酸楚，他和陈子昂、江俞轩偶有联系，怕大家知道他追宁雅追了这么多年，最后是竹篮打水一场空。
<br/>
<br/>        “我和宁雅是同学。”凌辰又补了一句。
<br/>
<br/>        这句话在陈子昂听来就是要和宁雅撇清关系了。
<br/>
<br/>        宁雅送资料到陈子昂的办公室，陈子昂张了张嘴，准备说凌辰打电话问宁雅的情况，想让宁雅接电话，但是听到凌辰的话就什么都没有说。
<br/>
<br/>        她不希望宁雅受到任何伤害。
<br/>
<br/>        放下电话，凌辰想此次去一趟北京，在宁雅放假的前一天飞过去，第二天和宁雅联系，放假了她应该不会推脱忙了吧？
<br/>
<br/>        心里有了打算，凌辰就觉得轻快了起来。
<br/>
<br/>        刘诚之说得对，宁雅这么多年都没有男朋友，不可能这几天就谈上了，按照宁雅什么话都给他说的那种性子，如果要是有男朋友肯定也会告诉他的。
<br/>
<br/>        凌辰自动了忽略了宁雅那天接电话的声音，他想去求证一下，不管结果如何也算是给自己一个交代，毕竟这么多年心里一直都是宁雅一个人。
<br/>
<br/>        后来冷静下来想一想，宁雅每次都会告诉他，相亲的结果，她没有一个满意的人，虽然他们俩个人从来都没有确定过俩个人之间的关系，但是心里肯定是会有彼此的。
<br/>
<br/>        如果，他勇敢一点，告诉宁雅，他爱她，那么会是什么样的结果？
<br/>
<br/>        凌辰突然就不想自己有遗憾了，他已经确定要去见一见宁雅。
<br/>
<br/>        “诚之，帮我订一下去北京的机票。”心情好起来了，见谁都开心。
<br/>
<br/>        “想通了？”刘诚之看着凌辰那万年不变的脸有了一点点变化。
<br/>
<br/>        “哪里来的那么多废话？让你订你就订，另外酒店也定一下。”
<br/>
<br/>        “好嘞！”刘诚之将凌辰的行程看了一下，定了节前一天的机票和酒店。
<br/>
<br/>        “我得提前两天回苏州，老爷子要我回去相亲。”刘诚之有些无奈，他还想玩几年好不好？
<br/>
<br/>        “你多大一点？就要想相亲了？”凌辰有些奇怪的问。
<br/>
<br/>        “家里条件一般般，老爷子怕我找不到媳妇。”刘诚之无奈。
<br/>
<br/>        “刘诚之，你家里条件一般？你穿定制的衣服？”凌辰冷冰冰的说。
<br/>
<br/>        “你怎么知道？”
<br/>
<br/>        “董亚子那天被你搂住的时候我发现呢。我就说呢，平时你的举动就不是一般家庭能教育出来的，果然不错！”
<br/>
<br/>        “从学校毕业我就溜出来了，要不然就被老爷子逼着和他一起，天天见一帮老古董，那样生活是多么的无趣啊。”
<br/>
<br/>        “跟着我就有趣了？”
<br/>
<br/>        “也很无趣，一个冰坨子，不过学到了在老爷子身边没有学到的东西。”
<br/>
<br/>        “我觉得，董亚子和你很配哦。”
<br/>
<br/>        “你是怕董亚子黏着你吧？如果是为你和小雅姐的幸福我倒是可以牺牲一下我自己，但是我还是很害怕，董亚子就是一只母老虎。”
<br/>
<br/>        凌辰被刘诚之逗得哈哈大笑，心情就像是那碧空如洗的天。
<br/>
<br/>        天很蓝，你很甜，心里永远都有你的颜，凌辰心里突然就蹦出一句话很想对宁雅说。

他当即将速度减了下来，路边停车。

<br/>
<br/>        他将一块白色的方巾递给了她。

<br/>        陆寒霆看着她泛红的眼眶，那纤长的羽捷上甚至沾了几滴晶莹的泪珠，翩跹之下要掉不掉的，楚楚可怜。
<br/>
<br/>        陆寒霆心里像是有一把刀在绞着，很疼很疼，将薄唇勾出了一道浅浅的弧线，他嗓音低哑的笑道，“我不是故意将车开的这么快的，刚才就像是控制不了自己，你就当我…又发病了吧。”
<br/>
<br/>        夏夕绾看出了他眉心里的自嘲之色，她迅速拿出了一粒药丸递到他的唇边，“那你现在还好么，先把今晚的药吃了吧。”
<br/>
<br/>        陆寒霆扭头，避开了。

<br/>        “你为什么不吃？”夏夕绾问。

<br/>
<br/>        “哎，你干什么，你怎么能将药丢了？”

<br/>        夏夕绾秀眉一拧，迅速打开了垃圾桶的盖子，她将两只小手伸进垃圾桶里，赶紧去找药丸。
<br/>
<br/>        陆寒霆迅速扣住了她纤软的腰肢将她扯进自己的怀里，“应该是我问你想干什么，你的手能去找垃圾？”
<br/>
<br/>        “陆寒霆，那是药，你不要丢药也不要自暴自弃好不好，你应该配合我的治疗！”


<br/>　　<br/>　　    王惊蛰也夹杂在大昭寺的祈福队伍里，他倒是并不一定需要什么祈福，但就想瞻仰一下这位活佛的尊容，这世上沽名钓誉之辈不要太多，可大昭寺里的这一位，绝对是实打实的第十二世班禅活佛，藏区里首屈一指，官方认证的。<br/>　　<br/>　　    论神秘的话，天坑下苯教神庙里的扎西降泽肯定最神秘，可要说盛名，那还得是这一位了，活佛是受藏区万人敬仰的。<br/>　　<br/>　　    王惊蛰身边是一群来拉sa旅游的年轻大学生们，朝气蓬勃，洋溢着青春，和他们年纪差不多的王惊蛰在这群人中，却显得有点老气横秋了，脸上有着和他年龄不相称的城府，老脸和成熟。<br/>　　<br/>　　    岁月蹉跎了他的人生，在他的年轮上，写满了沧桑。<br/>　　<br/>　　    “我想求大师赐缘，我喜欢白嘉学长，希望大师能为我们配上一对”一个满脸花痴，长相还算及格的女生捧着小手，眼睛里全是小心心的憧憬着说道。<br/>　　<br/>　　    “你低调一点行么？大师是活佛，又不是月老，祈福也不是为你点鸳鸯谱啊，你为家人求一些安康为自己求一求好运也行啊，至于把祈福浪费在男女之事上么？”花痴姑娘旁边的同伴笑嘻嘻的挤兑着她。<br/>　　<br/>　　    “那我不管，我就要白学长，唉？小蜜蜜你要求什么福啊？”<br/>　　<br/>　　    旁边的女人沉吟着说道：“我也要白学长……”<br/>　　<br/>　　    “哈哈，你看你那贱气逼人的样子，刚才还装的一本正经的，原来也是不怀好意啊”<br/>　　<br/>　　    “哎呀，能者居之么，谁让白嘉长得那么好看，谁不想咬一口啊”两个姑娘肆无忌惮的调侃，引来了旁边一群人的哄堂大笑，王惊蛰站在队伍里不禁感叹了一声，年轻真好。<br/>　　<br/>　　    其实他也一样年轻，只是他的生活和经历却没有年轻该有的朝气，作为现代人来讲，王惊蛰过的可能和绝大部分同龄人都是不相干的生活，他既没有上过学，也没去打过工，更没有年龄相仿的同伴，无疑，他的生活里少了很多的色彩。<br/>　　<br/>　　    “各有各的活法吧，何必强求？”这是王惊蛰安慰自己的一句话。<br/>　　<br/>　　    祈福的队伍进行的不算太快，因为人太多了，正在缓慢的朝前移动着，旁边的青年男女都有点着急了，有人说活佛的赐福是有时间限制的，可不是一整天都在这里的，不然几百上千号人岂不是把活佛给累吐血了。<br/>　　<br/>　　    又等了一段时间，王惊蛰随着队伍移动了一段距离后，他远远的看见了十二世班禅活佛，一个中年男子身体前倾着，活佛正在为他摸顶。<br/>　　<br/>　　    十二师班禅活佛难以想象的年轻，大概只有三十岁上下，浓眉大眼，皮肤黝黑，长相很是普通，一点都没有那种在人群中你能把他一眼就找出来的感觉，要不是他披着黄肩穿着僧袍，你很难想象如此普通的年轻人居然会是第十二世班禅活佛。<br/>　　<br/>　　    似乎感觉到有人在看自己，班禅活佛抬起头，在千百人中一下子就把眼神落在了王惊蛰的身上，然后点了点头，王惊蛰也轻点了下头示意。<br/>　　<br/>　　    只是因为在人群中多看了你一眼。<br/>　　<br/>　　    没过多久，今日大昭寺的赐福活动就结束了，一众信徒没有什么可失望的，对于他们来说以后总归还是有机会的，来大昭寺朝圣又不是一次两次，想被活佛赐福并不难。<br/>　　<br/>　　    可对于那些来此旅游的人来说，没被活佛摸摸脑袋，真的很失望，因为他们可能一辈子就只会来这一次拉sa了。<br/>　　<br/>　　    有喇嘛礼貌的在人群里告诉众人，今日活佛赐福结束，请各位回去，王惊蛰也夹杂在队伍里要慢慢的退出去，这时那位喇嘛忽然快步走了过来，到王惊蛰面前说了一句话。<br/>　　<br/>　　    “活佛说，请失主留步”<br/>　　<br/>　　    王惊蛰没有觉得有什么诧异的，毕竟两人在人群中对了一眼，对方是活佛能看在他身上看出门道，并不稀奇。<br/>　　<br/>　　    王惊蛰道了声谢，就随着喇嘛进入了大昭寺，往后面的一间厢房去了。<br/>　　<br/>　　    房间里面，就班禅活佛一人，见到王惊蛰他伸出手掌示意对方坐下。<br/>　　<br/>　　    “我叫罗桑丹珠”<br/>　　<br/>　　    “王惊蛰”<br/>　　<br/>　　    一句简单的介绍之后，活佛说道：“施主有恙”<br/>　　<br/>　　    王惊蛰一点都被诧异的点头说道：“上师可有解惑？”<br/>　　<br/>　　    活佛摇了摇头，王惊蛰依旧不觉得诧异。<br/>　　<br/>　　    “死，不过是另外一种方式的重生，施主不必介怀，再生，依旧是你，只是轮回罢了……”<br/>　　<br/>　　    王惊蛰心里腻歪的呵呵了一句，一阵腹诽，你真是站着说话不腰疼，道不同不相为谋，你们藏传佛教那一套在我这里行不通。<br/>　　<br/>　　    你们崇尚转世，重生。<br/>　　<br/>　　    而我，只想活好当下。<br/>　　<br/>　　    不只是我，还有王仙芝，王冬至，王风饶，还有那棵小草。<br/>　　<br/>　　    我和他们都想的是当下。<br/>　　<br/>　　    王惊蛰忽然失去了和活佛见面的心思，至于赐福什么的，他自然更不感兴趣，没有什么福可以解得了他的难题，所以赐或者不赐都无关紧要。<br/>　　<br/>　　    正当王惊蛰想要起身告辞，屁股刚刚挪开椅子的时候，罗桑丹珠又开口了。<br/>　　<br/>　　    他一开口，就让王惊蛰屁股又重新回到了椅子上，坐的是相当牢固了。<br/>　　<br/>　　    “曾经有一位施主，推算过我的转世……”活佛平淡的说道：“虽然我们不讲究什么人情，可能是我的修行不太够吧，我总觉得会欠了点什么，正好你和那位施主有关”<br/>　　<br/>　　    王惊蛰忽然有种很古怪的感觉，他觉得藏区的喇嘛要比内地的和尚好多了，中土那些和尚不管老少说话的时候特别喜欢打禅机，每一句话都说的恨不得跟哑谜似的然后才能显示出自己的佛法高深来。<br/>　　<br/>　　    这一点上，藏传佛教做的就比较好，你看看这位班禅活佛嘛，他说话就通俗易懂有什么说什么，简明扼要，一点都不禅。<br/>　　<br/>　　    这么交流，才不会累的！<br/>　　<br/>　　    王惊蛰身子往前探了一下，点头道：“那行，聊聊吧”

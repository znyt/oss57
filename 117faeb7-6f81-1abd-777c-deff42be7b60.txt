　　这圣院宗的护山大阵，足以承受数位九劫古帝的联手轰击。
<br/>　　在这护山大阵的保护之下，这无家的九劫古帝，有恃无恐。
<br/>　　“小子，你现在要是有能耐就直接杀了我啊，你以为我们圣院宗没有底蕴？等到圣地强者降临的一刻，就是你的死期了，以为得到些小小机缘，就可以毁灭我们圣院宗？简直白日做梦！”
<br/>　　那无家古帝又是开口讥讽道。
<br/>　　“就这小小的护山大阵，也想要阻拦住我？简直可笑，给我破开！”
<br/>　　楚风眠根本懒得回答那无家古帝的话，冷喝一声，突然间楚风眠手中的祖龙至尊剑一剑出鞘！
<br/>　　耀眼的剑锋，笼罩在整个圣院峰之上。
<br/>　　楚风眠出手了。
<br/>　　无数圣龙秘境弟子都将目光看了过来，注意着这每时每刻的变化。
<br/>　　只见那一道剑锋，斩在圣院峰之上的一刻，从圣院宗的深处浮现出了一道道古老的纹路，这些纹路凝聚而出，化为了上千层阵法，抵挡在了剑锋的前方。
<br/>　　这上千层阵法，几乎都是历代圣院宗诞生而出的强者，布置下来的。
<br/>　　圣院宗之中诞生的强者无数，这些强者都将自己的力量，加持在了圣院宗的护山大阵上。
<br/>　　现在这护山大阵，几乎相当于经历过上千位古帝的加持，其中的防御力，堪称逆天。
<br/>　　轰！
<br/>　　楚风眠剑锋的力量，斩在了这无数阵法之上，只是爆发出了一阵阵剧烈的响声，掀起了一阵波澜，却是没有将这些阵法打破。
<br/>　　“圣院宗，不愧是圣院宗，底蕴之深，的确不亚于圣龙宗了，这楚风眠还是太自大了，以为灭了一个六绝宗，就可以灭圣院宗。”
<br/>　　“可圣院宗的底蕴，远远不是六绝宗可以媲美的。”
<br/>　　“看来今日这小子要栽了，他就连圣院宗的防御都无法打破，等到圣地的强者到来，这小子必死无疑。”
<br/>　　一尊尊隐藏起来的强者，纷纷议论交谈道。
<br/>　　圣龙秘境八堂的势力，虽然远不如四宗，但是在数万年的时间内，也是培养出了各自的强者。
<br/>　　像是那执法堂之中，最为耀眼的法理古帝，可就是即将踏入半步天人境的强横存在。
<br/>　　八堂之中，虽然位列第二，第三，第四的万龙堂，悟剑堂，执法堂，都已经被楚风眠重创。
<br/>　　但是八堂第一的武霸堂，却是一直未曾现身过。
<br/>　　当然还有斩鬼堂，向修堂等堂口的强者。
<br/>　　这些八堂强者，聚集起来，也是一股不小的力量。
<br/>　　八堂的无数强者，都聚集在了圣院宗附近，准备看这一战的结果，如果楚风眠强势无敌，横扫圣龙秘境，他们也会选择臣服楚风眠以自保。
<br/>　　可如果楚风眠不是圣龙秘境的对手，那今日的事，也只是闹剧罢了。
<br/>　　本来无数强者，还都在观察这，可是看到这一幕，一些人便是蠢蠢欲动起来。
<br/>　　这个时候出手相助圣院宗的话，等到日后圣龙宗被除，圣院宗为尊的时候，现在相助圣院宗的人都足以得到巨大的好处。
<br/>　　“楚风眠，你狼子野心，胆敢攻打同门，已经是犯下了滔天罪孽，我虽不是圣院宗弟子，可也不能容忍你如此行为！”
<br/>　　嗖！
<br/>　　一道身影，突然降临，向着那圣院峰的方向飞了过去，降临在了圣院峰之上，跟圣院宗弟子，站在了一起。
<br/>　　“是斩鬼堂的太上长老狞断。”
<br/>　　看到这突然现身的人影，一些人喃喃自语道。
<br/>　　“看来八堂的人，要帮助圣院宗，来对付楚风眠了。”
<br/>　　“那是自然，这楚风眠毕竟独身一人，就算是加上圣龙宗的人，也不过小猫两三只罢了，在整个圣龙秘境，不算什么，现在出手相助圣院宗，未来有着大好处，这八堂的人，自然会出手。”
<br/>　　一名圣龙秘境弟子开口道，突然指向一个方向。
<br/>　　“你看。”
<br/>　　嗖！嗖！
<br/>　　随着狞断的现身，又有数道身影，从远处飞了过来，无一例外，都是降临在了圣院宗之中。
<br/>　　“那是武霸堂的太上长老，霸司！”
<br/>　　“还有向修堂的白山古帝。”
<br/>　　越来越多的身影现身，那些之前还在保持中立，观察双方局势的八堂强者，许多都是加入到了圣院宗之中。
<br/>　　“多谢诸位相助，等剿灭了这小子跟圣院宗这两个宗门孽障之后，少不了诸位的好处。”
<br/>　　那无家的无不源，看到到来的众人，看着楚风眠冷笑一声道。
<br/>　　“小子，你以为你一个人就可以抗衡整个圣龙秘境，未免太天真了些，现在乖乖跪下臣服，是你唯一的生路！”
<br/>　　“一群乌合之众，土鸡瓦狗，联合在一起，又有何妨？”
<br/>　　楚风眠看到聚集起来的众人，眼神之中却是露出冰冷之色道。
<br/>　　“既然都来了，那正好也免得我一个个去找了，今日圣院峰之上的人，都得死！”
<br/>　　“哼，小畜生，你不用在这里说大话，你先打破了这护山大阵再说。”
<br/>　　方才出言讥讽的那无家古帝，再度出声讥讽道。
<br/>　　“就这么一小小的护山大阵，你以为真的可以抵挡住我？我不过是想要将那些废物都引诱出来罢了，既然都出来了，那你们就可以死了。”
<br/>　　突然间就在开口的一刻，楚风眠身上的气息，俨然之间已经是跟之前截然不同，冲天的剑意从楚风眠的背后爆发而出。
<br/>　　滚滚剑意，凝聚在了楚风眠的剑锋之上，从祖龙至尊剑上，更是燃烧起了一股无色的火焰。
<br/>　　三阳烈炎剑！
<br/>　　“给我破！”
<br/>　　轰隆隆！
<br/>　　同样是剑锋，可是这一道剑锋的力量，却是比起刚刚，要强横数十倍。
<br/>　　楚风眠的声音落下的一刻，轰隆！那剑锋也随之斩在了护山大阵之上。
<br/>　　千层阵法，再度凝聚而出，这一次，突然那千层阵法之上，都是燃烧起了一种无色的火焰。
<br/>　　一道道阵法，在这无色火焰之下，都被无情的燃烧殆尽。
<br/>　　护山大阵也好，上面的无数古老纹路也好，在这无色火焰的面前，都只有被灼烧的份。
<br/>　　整个护山大阵，都在崩溃，其中的无数阵眼，都被燃烧成了灰烬。
<br/>　　“咔嚓！”
<br/>　　随着一道清脆的声音，这圣院峰之上的护山大阵，破裂出了一个裂痕，随后更是如同接连不断的开始破碎，刹那间，护山大阵崩塌了。

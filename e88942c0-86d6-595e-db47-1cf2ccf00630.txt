　　“在阳光医院，我哥哥是刚住进去的，我还不知道具体的位置。”
<br/>　　讪讪的笑了一声，刘哔回答了小胖的问题。
<br/>　　“阳光医院啊……是不是市中心的那个？我倒是知道，就带你去吧。”小胖合计了一下，开口将事情包揽了下来。
<br/>　　居住了这么长时间，他对这个城市自然是无比熟悉，一般只要不是特别偏僻的建筑，他都知道。
<br/>　　“那就麻烦恩人你了。”刘哔脸上带着感激的表情，道谢了一句。
<br/>　　小胖摆了摆手，没多说什么，转身回屋准备去了。
<br/>　　片刻，穿戴整齐的小胖从屋里走出来，顺手还扔给刘哔一个背包，道：“这里面有一些压缩食物，还有一把菜刀，你注意好自己的安全。”
<br/>　　点了点头，刘哔打开背包，将菜刀握在手中，两人一前一后从楼道中走了出去。
<br/>　　小胖居住的地方有些偏僻，相对来说也比较安全，街道上零散几只尸兄漫无目的的走动着。
<br/>　　两人小心翼翼的从门口走出，没惊动这些尸兄，而是选择从旁边绕开，向市中心的方向走去。
<br/>　　一路上没有碰见什么危险，对于街道上的尸兄，两人能躲便躲，实在躲不开的，就只能尽量在其它尸兄没有发觉的情况下，迅速出手解决。
<br/>　　在城市里，和尸兄战斗可不是一件明智的事情，一旦动手的话，很容易就会引来尸群。
<br/>　　如果那样的情况发生，即使是小胖也无法保证自己能逃脱出去。
<br/>　　在这样寸步难行的条件下，两人到达阳光医院的时候，已经是傍晚了，当最后一点太阳落下，刘哔终于看到了阳光医院有些残破的牌子。
<br/>　　似乎要下雨，天空很阴沉，一大片乌云笼罩在阳光医院上空，残留的一些灯光闪烁着，让整座医院显得死气沉沉。
<br/>　　可诡异的是，自从进入了这里，尸兄的数量明显变少了许多，一直到医院门口，两人都没发现哪怕一只尸兄。
<br/>　　要知道这可以算得上是市中心了，是最为接近重灾区的地方，按常理说，这里的尸兄数量绝对不会太少。
<br/>　　事实上在进入这里之前，两人也不时碰见规模庞大的尸群，甚至有好几次，还差点惹出事情。
<br/>　　“小心点，这个地方……有点诡异。”
<br/>　　从自己身后的背包中抽出一把唐刀，小胖的语气有些严肃，他可不认为没有尸兄是一件好事情。
<br/>　　末世中，最可怕的并不是丧尸，而尸兄世界，也是同理。
<br/>　　在这样的地界，周围如果没有尸兄，那很有可能你是落入变异兽的地盘了，所以小胖才会如此小心。
<br/>　　“喔，喔，不过，这里看起来，好像没什么危险啊。”手中紧握着菜刀，刘哔跟在小胖身后，语气有些疑惑。
<br/>　　对于他的疑问，小胖并没有解答，而是做了一个噤声的手势。
<br/>　　他不想惊动可能潜伏在阳光医院内部的那些东西，如果真的有变异兽，两人的处境将会十分危险。
<br/>　　和尸兄不同，那些动物经过变异之后，都会觉醒一些特性，再加上原本就比人形更适合战斗的身体，故此这些变异兽的危险程度，比尸兄要高了许多。
<br/>　　嘀咕了一声，刘哔闭上嘴巴，安静跟随在了小胖身后。
<br/>　　两人小心翼翼的进入了医院。
<br/>　　……
<br/>　　医院里的电灯大都已经破碎了，偶尔几个尚能使用的，也一闪一闪的，极不稳定。
<br/>　　标注着应急通道的牌子，绿油油的，让人心底莫名有些发怵。
<br/>　　小心起见，两人都没有使用电梯，而是来到了步行通道，沉闷的脚步声在空旷的楼道里有些刺耳。
<br/>　　太安静了。
<br/>　　即使他们已经可以放轻了脚步，可脚步声依旧十分清晰。
<br/>　　“你哥哥住在几楼？”
<br/>　　小胖压低声音，在刘哔耳边询问了一句。
<br/>　　心中一动，刘哔伸出手掌，做了一个五楼的手势，如果他没记错的话，那个博士就在居住在五楼。
<br/>　　只是不知道现在这个时间段，主角有没有来过这里，博士要不再了的话，他就只能再寻找其他的办法了。
<br/>　　点了点头，小胖带着刘哔，两人径直来到了五楼，期间楼道外面偶尔会传来一些诡异的声响，但两人并没有去查看的心思。
<br/>　　谁也不知道外面有什么东西，医院这种地方，总会让人联想到一些不好的事情。
<br/>　　小胖不想惹麻烦。
<br/>　　一路上畅通无阻，即使两人再小心，区区五楼，几分钟的时间也就爬上去了。
<br/>　　步行通道的大门紧闭着，谁也不知道门后面是怎么样的场景，至少到现在，小胖预料中的危险还没有出现。
<br/>　　他可没将这个地方当做什么安全的宝地，这只能是更加危险的一个魔窟。
<br/>　　“咔嚓。”
<br/>　　一声轻响，小胖将刘哔挡在伸手，扭动了大门的把手，开门的一瞬间，刘哔猛地拉了小胖一把，两人重心不稳，跌倒在楼道的拐角处。
<br/>　　当看清门后的场景时，小胖头皮猛地一麻，一咕噜从地上爬了起来。
<br/>　　半截尸体吊在了门口，因为小胖的动作，微微摇晃着，被大门遮挡，两人都看不见尸体的上半身。
<br/>　　宽大的病号服遮盖在尸体身上，松松垮垮的，在昏暗的灯光下鼓动着。
<br/>　　可是，医院内部，哪里，来的风？
<br/>　　“诶嘿嘿哈哈哈……”
<br/>　　一股刺耳的笑声，突兀从楼道内响起，两人心中一惊，顺着声音传来的方向看去，发现了墙角上方悬挂着的扩音器。
<br/>　　两人的目光皆是阴沉了下来，他们知道，自己是中了埋伏。
<br/>　　“又有新的猎物了呢……正好那你们实验一下，试验品的威力。”
<br/>　　刺耳的声音从扩音器中传出，语气中带着一股狂热，随后一段轻柔的音乐从扩音器中播放了出来。
<br/>　　“咦……放错了，不对啊，我的启动音频放哪去了。”
<br/>　　在两人阴沉的目光下，扩音器中一阵骚乱，随后，音乐声戛然而止。
<br/>　　“诶嘿嘿哈哈哈，找到啦！”
<br/>　　狂热的声音从播放器中传出，有些癫狂，随后狂躁的音乐声从中传来，门口悬挂着的尸体，猛然颤动了一下。

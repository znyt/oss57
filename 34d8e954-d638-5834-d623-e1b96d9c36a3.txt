　　不周山！
<br/>　　距离元始上次离开不周山，时间已经过去近百个元会，这号称洪荒第一神山的不周山也大变模样，变得不再如元始第一次见到的那般贫瘠。
<br/>　　元始三人来到不周山，放开神识，元始分明感觉到，天柱不周山的威压已经减弱了许多，不再如他之前他见到的那般强大了。如今，就算是一个普通的大罗金仙修士，也能够轻松到达天幕罡风层。
<br/>　　他微微一愣，便知道这是一种无法避免的现象，随着盘古身化洪荒的时间的延长，盘古的气息就会越来越少，直至消失不见。不周山的威压亦相同，甚至元始猜测，最后这不周山的威压更甚至会变得完全没有。这也说得通，共工能够以区区祖巫之躯，便撞断天柱不周，也许这方面的原因占了很大一部分。
<br/>　　传闻不周山宝物多多，随手可捡灵根，随地可收仙草。如今虽然没有传说中那般夸张，但也不差到哪里。这里的仙草灵跟的确很多，可以说是随地都是，但这些灵根仙草大多都实属一般，这对天仙以下的修士来说，的确是天堂，但对于金仙或者大罗金仙一般的修士，这些仙草灵根就未必起多大作用了。
<br/>　　而且，此处的灵根，大多都是一些下品或者不入品级的后天灵根，对于元始等人来说，他连看都懒得看一眼，更不用说是去捡了。
<br/>　　不过，偶尔中还是能够遇到几个例外的，这里一样有先天灵根，只不过它们不如这些一般的后天灵根多而已。
<br/>　　一路上，若是遇上不错的灵根灵宝灵药，都会被他收了起来，种到身体之中的洞天空间之中，为洞天增加点色彩生机。本着绝不放过一样好东西的原则，元始三人一路之上，倒是收获了不少灵根，到了半山腰，看着布满洞天的各种灵根，元始心中一阵满足。而且越到后面，能够遇到的好东西就越多，这想必是盘古威压所导致的原因吧。
<br/>　　很快，三人便穿过层层天幕的阻碍，来到了不周山顶。不周山顶开阔了许多，而且顶级灵根更多了，元始大手一挥，便将一片一片地灵根收了起来。
<br/>　　元始带着老子通天两人来到他之前布置的阵法之处，打开阵法，便可见到里面的先天灵根。一株先天芭蕉树，长着黄、蓝、赤、青四色的四柄芭蕉叶，树上还挂着几串金黄色的芭蕉。
<br/>　　这正是顶级先天灵根先天芭蕉树！
<br/>　　“好东西啊！”
<br/>　　通天见此，眼睛一亮，他的眼光何等毒辣，一路之上见惯了普通的灵根，现如今见到顶级灵根，顿时觉得不虚此行。而且，他也看出了，那四柄芭蕉叶，为地水火风四象属性，蕴含四象本源，都是顶级的先天灵宝。
<br/>　　“来来来，每人一柄芭蕉叶，不要嫌弃，闲时拿来扇扇凉还是可以的。”
<br/>　　元始大手一招，瞬间便将四柄芭蕉叶摘了下来，准备将它们分配了。
<br/>　　“二弟你喜欢炼丹，这柄芭蕉火扇就是你的了。至于三弟，嗯，你性子直，就自己选一柄。”
<br/>　　元始说着，一边将赤色火扇递给老子，一边将另外三柄送到通天面前，供他挑选。
<br/>　　“多谢大兄，我就选这柄蓝色的水扇吧，一扇扇出，大水涛涛，我喜欢！”
<br/>　　通天说着，拿起蓝色芭蕉水扇，对着不远处的一个大石就是一扇子。
<br/>　　顿时，虚空之中，出现一股浓郁的太阴之水，水还没有落下，大石便被冻住，等水落下，大石早已破碎一地，就连周围的大地，都被冻结。
<br/>　　“好！不错，不错！”通天拿着水扇，夸赞道。
<br/>　　“好了，既然你们已经选好，那么，我们也该去下一处目标了。”
<br/>　　元始收起另外两柄芭蕉扇，顺便将这一株芭蕉树移进洞天之中，取出一些三光神水、天一真水浇灌到树上。如若不然，这颗芭蕉树就快要死了，你没看见在元始取走芭蕉扇后，芭蕉树便变得萎缩起来了吗。
<br/>　　神水与真水一入芭蕉树，芭蕉树瞬间便变得精神起来，一股强烈的灵气风暴不到片刻便掀了起来。无数的灵气灌入到芭蕉树内，芭蕉树重新长出了嫩芽，眼看用不了多久，它便会再次成长，元始便不再关注。
<br/>　　之前元始本打算用芭蕉四扇中的本源来修炼《玄黄不灭》的第三重，但如今他早已突破第七重，它们却是用不上了。既然用不了，拿了当做一般灵宝来用，还是可以的。
<br/>　　“来来来，吃点芭蕉，尝尝味道如何？”
<br/>　　元始取出之前芭蕉树上接的芭蕉，递给两人，一边还往自己嘴里送了一根。
<br/>　　“嗯，不错不错，不愧是先天芭蕉，这果香味的确不错，话说，我好久没有吃过灵果了……”
<br/>　　“谁让你不吃的，这又怪得了谁？”
<br/>　　“哈哈哈！”
<br/>　　“……”
<br/>　　不周山顶另一处，一株仙藤长得极为茂盛，而现在更是灵气汇聚而来，注入其中，原来是藤上长得葫芦就快要成熟了。这株仙藤可不简单，它上面竟然挂着七个葫芦，而且颜色不一，各具威能。
<br/>　　这是十大先天灵根之一的先天葫芦藤！
<br/>　　而葫芦藤前，此时却站着三个人，他们自然是元始三人了。得到了先天芭蕉树后，三人便一路马不停蹄地来到了另外一处阵法前，打开阵法，便见到了这一幕。
<br/>　　灵气汇聚而来，注入到先天葫芦藤***七个葫芦吸收。
<br/>　　“葫芦娃吗？”
<br/>　　元始轻声嘀咕道。看着眼前七个颜色各异的葫芦，这一刻，他竟然想到了葫芦兄弟的故事。
<br/>　　“什么？”
<br/>　　通天没听清，就想要再问元始。
<br/>　　“没什么。”元始摆了摆手，将这个话题抛开。
<br/>　　看着眼前正在不断吸收灵气的七个葫芦，眼看不知道还要等多久，元始有些不耐烦了，便对着通天道：
<br/>　　“三弟，布下一道聚灵阵。”
<br/>　　“如此甚好。”
<br/>　　接着，通天立刻布下一道聚灵阵。瞬间，灵气大涨，蜂拥而来，汇聚成液，浇灌到七个葫芦之上。
<br/>　　元始皱了皱眉，还是不太满意，接着，他取出一些三光神水与天一真水，混合灵气，浇灌到先天葫芦藤上。
<br/>　　“轰隆隆！”
<br/>　　立刻，光彩大涨，葫芦威势在迅速增长，各种彩光绽放出耀眼的光芒，眼看这七个葫芦就要成熟之际，元始抬头看了看不远处的虚空，皱了皱眉，轻声道：
<br/>　　“有人要来了。”
<br/>　　话音刚落，便见远处虚空一阵波动，几道流光闪过，远远地就可看见有六道身影出现这片虚空。
<br/>　　这六人正是伏羲与女娲兄妹、帝俊与太一两位兄弟、红云与镇元子两位基友。
<br/>　　六人听完道祖讲道，本是要回道场修炼的，但在途中元神一阵抖动，他们感觉自身的机缘所至，一番推算后，算到这机缘竟然在不周山。于是，才有了六人结伴来到此处的一幕。
<br/>　　“咦！竟然已经有人了？”
<br/>　　“是三清！”
<br/>(本章完)

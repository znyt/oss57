　　就在刘哔即将得手的时候，一柄闪烁着寒芒的刀尖刺向他的太阳穴，让他无奈收手，用拳套挡下了有马的进攻。
<br/>　　经过先前的战斗，有马贵将也发现了他的弱点，逐渐，有马的大部分攻击，都开始转向刘哔的头颅。
<br/>　　叮叮当当的声音不断响起，两人手中的武器都是舞的密不透风，剩余的两名搜查官，根本插不进手来。
<br/>　　两人不断交手，战斗逐渐开始胶着了起来，刘哔的进攻没有固定章法，他所有的招式，都是经过十年战斗的洗礼总结出来的。
<br/>　　只要出手，必是杀招，对于他来说，根本不存在任何多余的动作。
<br/>　　而有马贵将自幼从白日庭长大，又是上一代的独眼之王，战斗天赋自然是他望尘莫及的，即使是在如此大的实力差距下，依旧和刘哔战了一个旗鼓相当。
<br/>　　两炳长刀式的库克因，在他手上无比灵活，往往都能找到极为刁钻的角度出刀，面对刘哔狠辣的进攻，丝毫不落入下风。
<br/>　　刘哔有些头疼，他从来没有碰见过如此难缠的对手。
<br/>　　这还是首次，面对弱于自己的敌人打的如此艰难，一般情况下，只要实力相差没有太多，他多少都会取得一些优势。
<br/>　　但有马贵将不同，这个家伙，精准的如同机器一样，战斗中的每一个动作，仿佛都被他计算了出来，连一丁点的失误都没有。
<br/>　　反而是自己，一旦出现什么差错，就会被他抓住，瞬间展开反击，往往都会在他身上留下一些伤势。
<br/>　　真是个可怕的家伙，缠斗了一段时间过后，刘哔的动作越来越急躁，终于，在挡下有马进攻的时候，露出了一个微不可查的破绽。
<br/>　　看到这转瞬即逝的机会，有马贵将的眼睛一亮，左手的长刀以一个诡异的角度，插向刘哔的下颚，与此同时，刘哔嘴角勾起一抹笑容。
<br/>　　糟了！
<br/>　　心中升起一抹不好的预感，有马生生止住了进攻，身体急速的向后退去，随后，密密麻麻的赫子，如同箭矢一般将他原本站立的地方覆盖。
<br/>　　S级技能，天灾！
<br/>　　暗红色的赫子，如同一柄柄匕首一般，切豆腐一般的插入了走廊，将地面射出一个个透明窟窿。
<br/>　　有马额头滑落一滴冷汗，脸色终于微微有了一点波动，如果刚才不是他闪避的及时，恐怕就要交代在了这里。
<br/>　　但还不等他缓过劲来，地面突然升起几只骷髅手掌，死死抓住了他的双脚。
<br/>　　“轰！”
<br/>　　原本挂在窗户上的两颗干枯头颅，此时猛然爆开，有马站立的地方，瞬间变成了一片火海。
<br/>　　ZPK300微型炸弹，2000积分一颗，他只兑换了两颗，分别固定到两颗头颅内部。
<br/>　　设立这个陷阱，也是为了以防万一，毕竟他要面对的敌人是有马贵将，果然，现在派上了用场。
<br/>　　随手将激活炸弹的遥控器扔下，刘哔顾不得爆炸的余波，急速冲进了火海，一拳捣碎了有马贵将的头颅。
<br/>　　“叮~恭喜宿主击杀特等搜查官有马贵将，获得积分一万点。”
<br/>　　随着系统的提示声音响起，战斗也彻底落下了帷幕，击杀有马贵将获得的积分，甚至可以和芳村艾特相比了。
<br/>　　不过也没什么奇怪的，毕竟有马的实力，也到达了SSS级，这场战斗，可比猎杀芳村艾特的时候艰难多了。
<br/>　　随手将剩余的两名S级搜查官解决掉，刘哔对地面上的几具尸体，发动了吞噬功能。
<br/>　　瞬间，几具尸体以一个肉眼可见的速度干瘪了下来，系统的提示音，在他脑海中不断响起。
<br/>　　一如既往的，两名S级搜查官并没有给他提供太多的收获，而有马贵将，也只让他的喰种血脉提升了一些。
<br/>　　同为一个级别的能量体，从他身上取得的收获，要比芳村艾特少了很多，更不要说最先吞噬的店长尸体了。
<br/>　　不过，这也是理所应当的事情，随着他的等阶越来越高，这种级别的对手已经满足不了他的需求了。
<br/>　　估计下一次再吞噬这种级别的能量体时，也只会起到提升自己体内的能量总和的作用了。
<br/>　　就和他之前，吞噬S级，或者SS级喰种时一样。
<br/>　　从空间戒指中拿出一块洁白的手帕，将拳套上沾染的碎肉块和血迹擦掉，刘哔转身，向楼下走去。
<br/>　　被鲜血染得通红的手帕，从窗外缓缓飘出，顺着风力飘向远处。
<br/>　　……
<br/>　　“我先冲在前面，旧多一等在我身后待命！”立智站在走廊中央，看着前方手臂缠绕着螺旋状赫子的女人，额头滑下一滴冷汗。
<br/>　　这个家伙，怎么会这么强大，连老师都抵挡不住了吗？
<br/>　　远处的地面上，躺满了搜查官的尸体，一具头颅被电锯分割成两半的尸体，格外突出。
<br/>　　木嶋班的成员，到现在也不敢相信，自己的班长，竟然死在了自己的库克因上面。

<br/>　　旧多二福露出一抹诡异的笑容，伸手将立智拉到了自己身前，用来抵挡松下的进攻。

<br/>　　S级喰种的全力一击，竟然连他身上的西服都没有砍破，不，甚至连一点皱褶都没造成。
<br/>　　漠然的看着松下，旧二多福站在原地微动，反倒是松下被自己进攻力道反震的倒退了几步。

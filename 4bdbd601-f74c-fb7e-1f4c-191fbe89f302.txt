　　围攻靺鞨骑的战斗从早上一直持续到下午，打到最后，黑齿常之他们干脆也不打了，就那么看着靺鞨骑兵拼了命的去渡河。
<br/>　　整个过程异常惨烈，其间淹死的人无数，整条河上漂满了尸体。
<br/>　　毕竟这些靺鞨骑兵是在逃命，身后就是大唐骑兵无休止的攻击，这与平时行军渡河并不一样，紧张、争抢之下难免会有人坠入河中，被战马或挤或撞，最后成为河里的一条亡魂。
<br/>　　看到这样的情况，黑齿常之他们干脆也不打了，除了阻止黑水部突围之外，就那么眼睁睁看着靺鞨骑兵渡河。
<br/>　　……
<br/>　　是夜，已经离开战场三十里之外的大祚荣命令部队停了下来，回身清点之下才发现，能跟着他回来的人不足一万五千之数，其他人不是战死，便是在渡河的时候不小心淹死了。
<br/>　　这是惨败啊，彻彻底底的惨败，五万大军甚至连与唐军交手的机会都没有就折了一多半。
<br/>　　机关算尽的大祚荣羞愤欲死，早知道这样还不哪真刀真枪的与唐军打上一场，就算是输至少也输的清清楚楚。
<br/>　　可是眼下这算什么啊，连敌人的影子都没看到自己就先进了别人的埋伏圈，而后被敌人围上之后竟连还手的能力都没有就被打的落荒而逃。
<br/>　　丢脸，这简直太丢脸了。
<br/>　　……
<br/>　　大唐长安。
<br/>　　刚刚离开没有多久的钦陵再次被李承乾找了回来，看着欲言又止的太子，有些不确定的问道：“殿下突然半路叫钦陵回来，可是有什么要事？”
<br/>　　“嗯……，算是有些事情吧！”李承乾深吸一口气，啧了一声说道：“靺鞨人被调出来了，现在的黑水靺鞨后方一片空虚，你有什么想法？”
<br/>　　“臣一切听从殿下安排！”钦陵眼中闪过一抹兴奋的光芒，不过却没有过多的表态。
<br/>　　他知道既然李承乾把他叫回来，并且与他说起这件事情，那就一定是对他有什么安排，加上此前这位太子殿下还对他说过要调他去辽东这件事，钦陵如何能不明白叫他回来的意思。
<br/>　　“我打算安排你去定襄城，从苏定方那里给你调五千骑兵，你的任务就是从漠北绕道进入靺鞨，直接控制住他们的老巢，怎么样？能做到么？”
<br/>　　李承乾并没有让钦陵多等，很直接的说出了自己的目的，而且他并没有问钦陵是否同意，反而直接问他能不能做到，这分明就是不容拒绝的意思。
<br/>　　而钦陵此时正觉得无聊，想回家去看看老头子又觉得没有什么能拿得出手的‘礼物’。
<br/>　　现在李承乾让他去搞定靺鞨，他自然是乐得接受，打算把这件事搞的漂亮一些，也好回去让老头子高兴高兴。
<br/>　　于是乎钦陵在接了李承乾的命令之后，想都没想，直接去兵部领了兵符将印，便连夜出城向漠北而去。
<br/>　　……
<br/>　　不得不说，李承乾这段时间对老头子的架空工作完成的不错，一些以前需要他去找老头子打招呼的事情，现在只要给下面吩咐一声，下面的人就会就动去办，然后主动找一些理由向老头子反应。
<br/>　　虽然看上去似乎比以前更费事儿了，但却从另一个角度证明了小李子对整个朝庭的掌控力。
<br/>　　甚至在某些特殊的条件下，李承乾现在已经可以做到没有他点头，老头子的命令甚至连大明宫都传不出去。
<br/>　　不过，这样做并没有任何意义，反正现在老头子和他都是站在一条战线上的，如果真把老头子弄成太上皇，不说名声如何，单单每天那些处理不完的公务就能让他痛不欲生。
<br/>　　所以李承乾现在并不着急跟老子摊牌，一切都还保持着原来的样子，甚至在某些情况下李二发现现在的政令比以前更好传达了。
<br/>　　当初有些事情还需要拿到朝堂上议一议，可是现在竟然他在朝党上一说就能通过，也不知道是不是下面这些人突然开窍了。
<br/>　　李承乾很喜欢现在这个样子，自己在暗中策划一些事情，然后交待下面的人去办，老头子负责处理大量公务，处理的意见只要不是太过离谱一般来说小李不会干涉，下面的人照办也就是了。
<br/>　　正是因为这样，对钦陵的安排根本没费什么力气，派小白带着钦陵去兵部走了一趟之后，那家伙就成了五品的朗将，又给他安排了百十个亲卫之后，整件事情就算是搞定了。
<br/>　　……
<br/>　　定襄城，苏定方看着眼前桀骜的青年，与他验了兵符之后，果断的调了五千骑兵给他。
<br/>　　早在数日之前他就已经接到来自长安的飞鸽传说，知道了钦陵的来意，自然不会多说其它，早点打发他上路才是真格的。
<br/>　　毕竟战机一闪即逝，从定襄城绕到靺鞨的北部还需要大量的时间，根本没有时间来扯蛋。
<br/>　　钦陵这还是第一次感受大唐正军的力量，苏定方虽然没有与他多说什么，但那种无形的压力对他来说并不比以前在家时老头子给他的压力小。
<br/>　　不过好在此人并没有与他多说什么，对了兵符之后，便将人马都交给了他。
<br/>　　而当钦陵看到了那在未来数月甚至数年都将数于他的军队时，不由暗暗倒吸了一口冷气。
<br/>　　那是一支怎样的军队啊！
<br/>　　五千人静静立在那里，竟然使大旗无风自动。
<br/>　　阵阵肃杀之气让钦陵站在那里都觉得脸上像刀割一样的疼。
<br/>　　看着眼前的一切，钦陵终于知道为什么李承乾根本不在乎他挑起的各种事端了。
<br/>　　因为不管他搞出什么样的事情，有这样一支军队在，大唐便可以无所畏惧，漫说他在林邑惹出来的那一点点麻烦，就算是现在从高原上拉下一支两万人的军队，估计也很难打败这五千人，而且这还是不算装备之利。
<br/>　　如果再把这五千人的装备算上，估计五千打五万都不成问题。
<br/>　　想到这里，钦陵又开始有了一丝小兴奋，毕竟现在他需要做的不是与这支军队作战，而是带着他们去打击大唐的敌人，他已经迫不及待的想要看到敌人在这支军队的铁蹄之下哀嚎的场面了。

　　　　就在众人以为叶天江的话到这里就要结束的时候，却是没想到叶天江根本没有要结束讲话的意思，手中依旧是拿着话筒。
<br/>　　“另外经过火神总部郑重讨论......”就在众人议论着的时候，叶天江再次开口，而且这次开口他还特意拉了一个长音。
<br/>　　等到现场终于再次恢复安静，所有人都将目光放到他的身上后，叶天江才再次开口。
<br/>　　“经过郑重讨论，决定将叶帝等12人列入英雄联盟名人堂，以此来表达他们12人对英雄联盟做出贡献的鼓励。”
<br/>　　当叶天江这句话开口后，现场经过短暂的沉寂后，刹那间炸起了锅。
<br/>　　“卧槽，12个名人堂啊，这个真的是厉害了！”
<br/>　　“666666666666，名人堂这个早就出来了，没想到今天终于有人进去了，而且还是一次就是12个。”
<br/>　　“我已经跟着激动起来了，12个名人堂选手，这绝对是值得电竞史铭记的时刻。”
<br/>　　“啊啊啊啊啊啊，本来还在想着我家emp能不能进名人堂呢，这下好了，终于进去了。”
<br/>　　直播间里的众多观众这一刻也是在那里狂刷着666。
<br/>　　而一旁的叶帝等人，显然之前也是不知道这个消息，所以骤然听到他们已经入选到名人堂的时候，几人也都是有些激动。
<br/>　　之前火神已经建立了一个英雄联盟名人堂纪念馆了，占地64000平方英尺，至今叶帝还记得那个消息出来后他自己那激动的情绪呢。
<br/>　　名人堂本质上是1个对外开放的英雄联盟博物馆，馆中主要陈列物品包括英雄联盟、图文、录像带、光碟等资料。
<br/>　　名人堂拥有比赛室给参观者体验气氛，还有小型影院可供播放英雄联盟影音资料，当然，还陈列着一张张照片以及照片边上的介绍性文字，照片上的人物就是人们口中的“名人堂成员”。
<br/>　　正式因为它的这个地位，所以就是连叶帝还有金相赫他们这群人也都是想要入选名人堂。
<br/>　　台下的众多选手，还有直播间里的众多选手骤然听闻这个消息后，一个个望着叶帝他们也都是一脸的艳羡。
<br/>　　接着在无数观众的注视下，叶帝一行人便是在叶天江带来的摄影师面前在舞台上开始拍起了照片，他们拍的这些照片将来都会悬挂在名人堂里面的。
<br/>　　到时候粉丝们可以去名人堂那边参观，里面不仅有着他们的照片，还会有着他们的生涯纪录，以及精彩操作的视频集锦。
<br/>　　直到夜里9点钟的时候，叶帝一行人才得以轻松下来，吃了个饭后便是向着基地返去。
<br/>　　接下来的几天，金相赫等人陆续的返回了自己的国家，因为夏季赛马上就要到来了，他们也要抓紧时间回去备战了，尤其是他们看到荣耀蓝白两队这么强之后。
<br/>　　而在这几天的时间，叶帝也是将杨小艺的名字去官方填入了自己的战队中，至于杨小艺小一岁这件事情，在叶帝的眼里那根本就不叫事情。
<br/>　　“卧槽，小叶子你这是作弊好么，我要举报！”蒋念一脸不爽的说道。
<br/>　　“小艺的身份证信息都被我更改了，你举报个毛线，你上哪举报，大家都不会信你的。”叶帝淡定的说道。
<br/>　　听着叶帝的这句话，蒋念顿时没有下文了，为了做到万无一失，将来不被别人拿到把柄，叶帝确实是将杨小意的身份证信息加了一岁。
<br/>　　“小艺现在有多强？”看着比赛室内还在打着游戏的杨小艺，秦霄有些好奇的问道。
<br/>　　“他现在的个人实力是有的，缺的只是对英雄的熟练程度，如果我猜测的不错的话，她现在应该只比我弱上那么一点点，不过明年才是她的巅峰期，甚至接下来3年都将可能在她的统治当中。”叶帝稍稍有些兴奋的说道。
<br/>　　“明天16岁，我怎么感觉20岁之前都将是小艺的统治时期呢，好在我明年就要退役了，真是心疼你们这些在役选手啊。”秦霄乐呵呵的说道。
<br/>　　“那小叶子你呢，明年小艺上场了，你干什么去？”楚南有些好奇的问道。
<br/>　　“明年我转行去做白队的教练吧，我已经感觉出来了，状态在逐渐的下滑，今年将会是我最后的巅峰时段了。”叶帝有些无奈的说道。
<br/>　　这也是他没想到的，怪不得那么多职业选手不找女朋友，原来女朋友真的会影响自己的状态的。
<br/>　　只不过为了让家里安心，他才会这么早和苏沐婉在一起的，要不然按照他的想法怎么也得在推个两三年。
<br/>　　“看来明年我们蓝白两队要有很多人要退役了啊，老秦、、林萧加上小叶子你、其实我也到了退役的年龄了，不过为了蓝队的荣耀，老秦还是让我在打一年。”蒋念有些落寞的说道。
<br/>　　“是啊，明年的时候许多老队友都不会在商场了吧，年龄真的是职业选手的一把杀猪刀啊，到了这个年龄就算是咱们不退役都不行。”林萧也是稍稍有些落寞。
<br/>　　他们热爱这个游戏，但是手速跟不上却是在逼迫着他们退役，而且在见到一个个老朋友都要离开这个舞台，此刻众人的心里都不怎么好受。
<br/>　　“好了，不要想那么多了，明年不是有教练战么，大家完全可以转型教练再战几年啊，要知道职业选手转型教练还是相对简单的。”就在众人压抑的时候，李景禹的一句话却是突然点亮了众人。
<br/>　　“对呀，转型教练，我们还可以继续在赛场上见，看来我们大家要努力了，到时候可千万别丢人了。”叶帝这个时候也是跟着笑了起来。
<br/>　　能够有这么一群兄弟一起投身到喜爱的东西上去，这真的是一件非常幸福的事情。
<br/>　　接下来的一段时间，虽然媒体还在那里争相报道王者战和教练战的事情，但是叶帝他们这边却是已经恢复了正常的作息，每天都是按时训练，调整自己等人的状态。
<br/>　　当然了唯一不寻常的可能就是蓝队在这一段时间也是收刮到了一个非常有天赋的选手。

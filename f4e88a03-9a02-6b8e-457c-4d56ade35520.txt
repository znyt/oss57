　　　　李浩然定睛看着他，道：“你有这等修为，你是何人？”
<br/>　　虽然之前打过照面，李浩然看出他实力不凡，但不知道他是何方神圣。
<br/>　　“昔年曾在祖师门下听讲，后得道飞升，也算有个师徒名分。”广宗天师捏了个子午诀说道。
<br/>　　李浩然一惊，上下打量他，道：“你是张果？”
<br/>　　广宗天师颔首，并不感到意外，从古到今，能够飞升无极的道士两只手数得过来，当年听过青牛祖师开坛传道的人虽然多，但除了张果，不可能还有别人。所以李浩然能够一下子想起来，并不稀奇。
<br/>　　李浩然道：“闻道不分先后。张果证道，往生修罗，福源广深，但你不是张果，你只是张果的一缕神念，生出灵智而已，怎敢在我面前卖弄？”
<br/>　　广宗天师道：“造衅一开，永无宁日，祖师慎重。”
<br/>　　“螳臂当车。”李浩然一笑，再度摇晃铜铃。
<br/>　　“末法未一！！”
<br/>　　又是一声道号，石破天惊。却不是出自广宗天师的口中。他刚气运丹田，一声道号之后，蓄积的真气已破，短时间内不可能再提得起真气。
<br/>　　道号是来自周围某座山的半山腰上。
<br/>　　众人抬头望去，余音缭绕，一时间根本无法确定声音的具体来源，更加看不到人。
<br/>　　“居然有人相助。”李浩然不屑一笑。
<br/>　　叶少阳转头望去，数了一下，自己小伙伴都在身边，而且他们不是鬼就是妖，就算修为再强，也不可能发出纯阳道号，风之谷那边更不用说，只有道风一个是出身道教，声音来自山谷上方，当然跟他没关系，而且道风已经跟杨宫梓、建文帝三人一起，对挡在李浩然前面的黎山老母等人展开了疯狂的攻击，目标自然是阻止李浩然摇铃。
<br/>　　不过这三人实力虽强，随便一个在人间几乎都是无敌，更不用说道风，但对方毕竟人多，最差的也至少都是准地仙、准罗汉的等级。仅仅靠着三个人，根本没用。
<br/>　　在叶少阳的带领下，捉鬼联盟的人也都冲上去，于是，又开始了新一轮的混战。
<br/>　　“什么人暗中帮助到道风？”远处，一座山的山顶上，女魃和后卿一起朝星宿七峰望过来，目光在山野中寻找着刚才发出纯阳道号的人，但没有找到目标。
<br/>　　在某座山峰的半山腰，一片草丛里，站着一个跟叶少阳长的很像的人，正是木子，也就是通玄道人。
<br/>　　通玄道人脸色发白，神色很是疲惫。刚才那声纯阳道号就是他发出的，他虽然修为深厚，但因为距离山谷太远，施展纯阳道号需要的罡气，也远远超过广宗天师在山谷的消耗，一声之后，体内气息翻涌，半天才压下来，直接一屁股坐在了一具尸体上。
<br/>　　是一头修成人形的獐子精，星宿海的门人，本来在这里埋伏守山，被通玄道人杀了，占据了这个居高临下的位置，可以看到山谷中发生的一切。
<br/>　　除了那头獐子精，还有好几个修道的精怪，都被通玄道人杀了，身边一堆尸体，还在不断往外流血。
<br/>　　通玄道人就坐在一堆血污之中，望着下方的叶少阳，喃喃说道：“叶少阳，老祖我也是尽力帮你了，可不能被验证出你鬼童的真身啊，不然你必死无疑，老祖我还上哪里去找合适的身体……”
<br/>　　然而，通玄道人最不想看见的事情，还是发生了——
<br/>　　两帮人正在混战的中间，突然之间，一声清脆的铃声响起。
<br/>　　道风和叶少阳等人猛然怔住，停止了厮杀，朝李浩然望去。
<br/>　　别的人也陆续都停下进攻，一个个目光都锁定在李浩然身上。场面顿时鸦雀无声。
<br/>　　叮铃铃……
<br/>　　清脆的铃声又响了一声。李浩然手里摇着铃，微笑着朝道风望去。
<br/>　　道风没有再试图阻止他。铃声已响，做什么都来不及了。
<br/>　　一时间，所有人的目光也都集中在他身上。
<br/>　　无极钟响，鬼童立现。这是无极天师留下的遗言。这口钟——虽然体积上更像是铃铛，是无极天师用自己的神念倾注祭炼，靠着与无极鬼王之间的某种无形的联系，鬼童一旦听到铃声，会立刻现形。
<br/>　　没有人怀疑这件事的真实性，因此都在等着道风现形。
<br/>　　道风一个纵身来到叶少阳身边，一道玉符贴在他脑门上，“闭眼，念静心咒！”
<br/>　　他给叶少阳贴的是闭气符，不会对身体造成任何影响，但可以隔绝外界的一些惊扰人心的刺激。对无极钟的铃声有没有左右，他也不清楚，但总归要试一试。他不能让叶少阳转世鬼童的身份暴露。
<br/>　　“你在干什么？”叶少阳吃惊地看着他，突然想起什么，推了他一把，“趁着没现身，你快跑！”
<br/>　　道风嘴角浮现出一抹苦笑。
<br/>　　杨宫梓也飞身前来，挡在叶少阳身前，万一有什么情况发生，能够及早应付。
<br/>　　芮冷玉也急忙上前。
<br/>　　他们都知道，其实叶少阳才是真正的转世鬼童。
<br/>　　然而，十秒钟已经过去了，叶少阳什么事也没有。
<br/>　　“你……没事？”芮冷玉吃惊地望着他。
<br/>　　“什么事？”叶少阳吃惊地看着他们，“你们都盯着我干什么。”
<br/>　　空间联盟那边，众人也是面面相觑，最后目光都落在黎山老母脸上，他们都是受到黎山老母的召唤而来，是梨山老母说道风是转世鬼童，他们才参与这次行动的。
<br/>　　如果道风不是鬼童……那这麻烦就太大了，付出的代价也太大了。
<br/>　　黎山老母根本不敢细想后果，脸色微变，朝李浩然望去，喝道：“怎么回事！”
<br/>　　道风是转世鬼童这个消息，就是悬空观的人说的，虽然黎山老母也调查过当日悬空观一战的经过，相信李浩然和苏沫等人没有说谎，但毕竟消息最初是悬空观放出来的。
<br/>　　李浩然面色坚定而刚毅，继续摇铃。
<br/>　　叮铃铃……
<br/>　　众人睁大眼睛，盯着道风不放。而道风等人却是盯着叶少阳。
<br/>　　叶少阳也没念静心咒。但是并没有任何状况发生。

　　“公孙轩辕？好名字！”
<br/>　　元始听了这个名字，自然是微微沉吟起来。公孙轩辕这个名字，现在可能不怎么样，但是以后，它却是会响遍天下，威震洪荒。
<br/>　　公孙轩辕，乃少典氏与附宝之子，本性公孙，后改姬姓，故称姬轩辕，居轩辕之丘，号轩辕氏，建都有熊，亦称有熊氏，也有人称之为“帝鸿氏”。
<br/>　　轩辕一统华夏部落，征服东夷、九黎等部落，一统中华，丰功伟绩！
<br/>　　轩辕一生，杀伐果断，大战不断，却是三皇五帝之中最好杀伐的一位，威名赫赫，震慑万族！
<br/>　　他在位期间，播百谷，种草木，发生展，制衣冠，建舟车，制音律，创医学等，功德无量，乃中华“人文出租”，为三皇之人皇。
<br/>　　当然，这些都只是以后的轩辕黄帝的一生，如今的他，还只是一个五岁大小的小孩，喜爱耍剑，向往飞仙！
<br/>　　这一刻，元始想到了许多，他仿佛能够看到，今后的人族，诸侯相侵伐，暴虐百姓，乱世之争，轩辕挺身而出，征伐众诸侯，灭杀妖魔，除尽暴乱，一统诸侯的浩大局面。
<br/>　　微微一想，元始便不再去关注其他，而是将目光重新放到眼前这位小轩辕身上。未来再如何了得，还是得看眼前，若是他现在就死了，哪里还有什么未来可言？
<br/>　　小轩辕现在还小，需要好好培养一番，如此，说不得我玉虚门下，又可以多出一位绝世无双的人才。
<br/>　　“小家伙，你可愿拜入本座门下，修仙问道！”
<br/>　　略微沉吟，元始直接开口说道，以他圣人之尊，说出这样的话，若是被其他人听见了，那还不羡慕死。
<br/>　　“你能教我什么？”
<br/>　　小轩辕并没有第一时间回答元始的话，而是反问道。他虽然人小，但是却十分聪明。
<br/>　　“呵？”
<br/>　　元始哑然失笑，小家伙那点心思，他一眼便知，不过，他也没有点破，而是直接说道：“你想学什么，本座便可以教你什么。”
<br/>　　“真的吗？”小家伙顿时眼前一亮，“我真的想学什么都可以？”
<br/>　　“那是当然！你看——”说着，元始伸出玉手轻轻朝着虚空一点，顿时，一个巨大的黑洞便出现在元始两人面前，黑洞演变成一个巨大的旋涡，吞噬周围的一切。
<br/>　　元始伸手一抹，顿时黑洞又消失不见，四周瞬间便恢复原样，草木丛生，河流依旧，就好似刚才发生的一切，都没有发生过。
<br/>　　“呃？”
<br/>　　小家伙顿时便瞪大了眼睛，眼中满是不可思议，心中却想着，这人到底有多强，才可以随手破虚空。
<br/>　　“拜见老师！”
<br/>　　小家伙也很聪明，见到这一幕后，便不再迟疑，立刻朝着元始拜下。
<br/>　　“嗯？本座可不是你的老师，我是你的师祖。”元始见此，满意一笑，开口说道。
<br/>　　“师祖？”
<br/>　　小家伙不明白为何不是老师，而要是师祖？不过，他心中很快便释然了，师祖就师祖吧，再说，师祖如此厉害，想必徒弟也很厉害吧。
<br/>　　“走，随我回山修行。”元始开口说道。
<br/>　　“嗯？那我的父母？”
<br/>　　小轩辕有些迟疑道，按理来说，他应该随着自己的老师，啊不！是师祖，随着自己的师祖回山修行，但他又担忧他的父母，怕他们为自己担忧。
<br/>　　“不必担忧。”元始轻轻说了一句，接着他带着小轩辕来到人族，向他父母说明情况，让他们不必担忧。
<br/>　　一番处理之后，他直接带着小轩辕一步踏出，回到昆仑山。
<br/>　　回到昆仑山，元始召来白鹤童子，沉声吩咐道：“去找你广成子大师兄，让他召集门人！”
<br/>　　“是，老爷！”
<br/>　　白鹤童子便是阐教看门童子，类似之前的昊天与瑶池的角色，然而，虽然为看门童子，但阐截的看门童子又岂会简单，他的一身实力也达到了大罗金仙初期境界，不是其他人可比的。
<br/>　　铛铛铛……
<br/>　　没过多久，广成子便敲响了金钟，声音浑厚，形成音波扩散到整个昆仑山，一时之间，在各自洞府内修行的，或者在灵神界中冒险的，或在其他地方游玩的，尽皆奔至玉虚宫。
<br/>　　……
<br/>　　玉虚宫大殿之内，元始高坐九重云床之上，金光万道，霞光冲天，大殿旁，还有一个小轩辕。广成子等十七个弟子，纷纷站在大殿两侧。
<br/>　　广成子等人眼光瞄向小轩辕，尽管他们很好奇这个小孩是何人，怎么会到大殿之中？要知道，能够到大殿之人，都是玉虚门人，这小孩难道是老师新收的徒弟？但他们知道什么该问，什么不该问。他们看了一眼轩辕，便将目光收回，而是向着元始拜道：
<br/>　　“弟子等，拜见老师，愿老师永恒不朽！”
<br/>　　“嗯！”
<br/>　　元始点了点头，看了众人一眼，却见广成子已经突破了大罗壁垒，成为三教弟子中的第一尊准圣高手。其他各自也有不同的突破，如身穿五彩道袍的孔宣也一只脚踏入了准圣，其他人也都有好几人是大罗圆满了。他不由得满意的点了点头。
<br/>　　三教之中，除了太上的人教只有一个传人，不管是截教，还是阐教，都有不少门人。虽然明面上不怎么样，但暗地里，他们也是在各自较劲，比较门人的实力情况，谁也不服谁。
<br/>　　如今广成子乃是三教弟子中除了燃灯之外的第一个突破准圣的，元始虽然面上没有露出丝毫的表情，心里却是对此感到非常高兴的，毕竟谁也不希望自己会输。
<br/>　　“不错，你们的修为都有突破，特别是广成子，你身为大师兄，首个突破准圣，确实起到了带头作用，希望你不要骄傲，以后继续努力。”元始淡淡评价道。
<br/>　　“是，老师！”
<br/>　　众人脸上难得露出一丝喜意，要知道，能够被元始夸赞，这种情况可是很少见的，他们虽然身为元始的弟子，却也是基本见不到元始夸奖一个人。
<br/>　　“嗯，此正逢人道当兴，三皇五帝出世，今天皇已证，神农为人族共主，乃是未来的地皇，只待他时机一到，便可证道为皇！为师等圣商议，三皇五帝之中三教各占一皇一帝，人皇之师当在我阐教！”元始淡淡开口说道。
<br/>　　众弟子一听，顿时面露喜色，一个个皆期盼抬头望去。
<br/>　　人皇之师的功德，谁不想要？功德谁也不嫌多，得到了，哪怕不能成为准圣，但也可以让自己的道行大进了，为自己缩减很多时间。对于像孔宣这样只差一步便可突破准圣的人，那功德更是重要，说不定就因为那点功德，自己就突破呢？
<br/>　　一时之间，广成子、孔宣、惧留孙、慈航道人等众弟子，纷纷抬头看向元始，期盼着自己能够得到此次机会。
<br/>　　“这便是轩辕，广成子，就让他先随你修行百年吧。”元始看向一旁的小轩辕，一语定音道。
<br/>　　“是，老师！”
<br/>(本章完)

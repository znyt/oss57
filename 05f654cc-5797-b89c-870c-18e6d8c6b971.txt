　　太阳星上，元始突然出现在扶桑树下。
<br/>　　此刻，那赤帝火皇与扶桑神树争夺谁为火中至尊的问题也落下了帷幕！
<br/>　　结局不出元始所料，最后是赤帝火皇以绝对的帝威压倒了扶桑神树！
<br/>　　赤帝的优势太大了，此地有源源不断地太阳真火供他吸收，只要不断吸收，他便能不断变强，而扶桑神树的实力却一尘不变，不输才怪。
<br/>　　“哼！”赤帝冷哼一声，霸道开口道：
<br/>　　“既然输了，那便要付出相应代价。”
<br/>　　赤帝不管不顾，伸手便将扶桑神树的一条枝干折断，拿到手中。枝干之上，火光流转，吸收着无数的太阳真火，赤帝一手持大印，一手持扶桑树枝，这一刻，赤帝的形象变得更加圆满了。
<br/>　　“轰隆！”
<br/>　　而此刻，扶桑神树却是变得火光冲天，身形大涨，无数太阳真火瞬间便被它吸收，很快，它那被折断树枝的一处再次长出一支嫩芽。无数的真火涌向嫩芽，被它吸收。
<br/>　　显然，扶桑神树也发怒了。
<br/>　　“嗯！”
<br/>　　见此，元始不再多留，他知道，这扶桑神树出了问题，帝俊与太一定会有所察觉，更何况他刚才突破的动静弄得有些大了，也定会有人来查看。此地不宜久留。
<br/>　　转身便向着洪荒大地而去，甚至他连帝俊的太阳神宫都还没来得及搜刮。之前他还想着趁着帝俊与太一不在太阳星上，他可以到他们的行宫之中搜刮一番，看看能不能捡到什么灵宝之类的东西，如今看来，却是不行了。
<br/>　　他不再耽搁，转身向着洪荒大地飞去。
<br/>　　洪荒大地之上，一处山谷中，两道身影在盘坐在一个灵泉之上修炼。灵泉如同河水一般，涌入到他们身体之内，增强他们的法力。
<br/>　　两人一身穿白袍，一身穿金色帝袍，白袍之人脸色俊朗，金袍之人脸上带有霸气。
<br/>　　此二人，正是两只金乌，太一与帝俊。
<br/>　　太一的头顶庆云之上，一口混沌色大钟在缓缓旋转，镇压这太一的元神，一股厚重，威压，深沉的气息扑面而来，正是混沌第一攻击至宝开天神斧所化的三大先天至宝之一的混沌钟，今后亦称东皇钟。
<br/>　　而帝俊头顶庆云之中，却是两幅图，绘尽了洪荒的山河湖泊，正是极品先天灵宝河图洛书。
<br/>　　两人元神相互交融，互相印证道法。
<br/>　　就在太阳星上元始突破之时，突然，两人睁开眼睛，双眼中爆闪出一股惊人的火焰，正是太阳真火。两人同时大喊一声：
<br/>　　“不好！”
<br/>　　之后，两人头也不回的向着太阳星冲去。
<br/>　　……
<br/>　　回到洪荒的元始并不知道两人的情况，他此刻并没有立刻返回昆仑山谷，而是向着幽冥血海而去。
<br/>　　他此行只是想去看一看，这传说中盘古肚脐的污血所化的幽冥血海到底是什么样？他更想看看，那里为何能连接幽冥界？
<br/>　　元始并不疾行，而是一步一步，悠然而行。他也想看看这洪荒数千年他不在这段时间的变化。
<br/>　　他虽然是一步一步地行走，但他的速度却是非常快速，比起大多数人飞行快多了。毕竟他用的是空间法则，一步踏出，就缩进无数空间，数百里一步就过去了。
<br/>　　也正是因为如此，那些看见他的人，都会识趣的不前去打扰他。这些人也不傻，他们也知道什么人能够招惹，什么人不能招惹？
<br/>　　像元始这样的人，他们一眼就知道不可招惹，要不然，怎么死的都不知道。
<br/>　　洪荒依旧，亘古如此！
<br/>　　要争斗有争斗，要机缘有机缘，要奇遇有奇遇。洪荒每天都在战斗，战斗是洪荒唯一不变的主题。
<br/>　　只不过，如今的洪荒经过一段寂静的时间，再次变得热闹起来。
<br/>　　这是一个众神倍出的时代，各处大神纷纷出世，占山称尊。
<br/>　　不周山女娲与伏羲，西昆仑西王母，万寿山镇元子，火云洞红云老祖，西方灵山接引与准提，北冥鲲鹏，血海冥河……
<br/>　　大罗金仙修为的强者如同潮水一般涌了上来，其余强者更是如雨后春笋般涌现。
<br/>　　但，元始并没有多管这些，完全没必要，这是一种姿态，一种无敌的姿态。以他如今修为虽不敢说此界无敌，但除了几个特别的强者之外，也差不多了。现如今对于洪荒这些涌现出来的强者，他完全没兴趣。
<br/>　　一路悠闲而行，遇到灵根灵药灵材神铁等，就收了起来，随心所欲，随遇而安。
<br/>　　很快，他便到了幽冥血海边缘。幽冥血海是盘古一身污血所化，为众血聚集之地，洪荒大战中流出的鲜血，大多流入汇集到这幽冥血海之中。
<br/>　　幽冥血海大多时间寂静无比，阴森森一片，污浊不堪，不是什么好的地方。但，此地却是也出了一个大神，便是那应运而生的冥河老祖。
<br/>　　冥河一身修为直达大罗中期巅峰，很快便能突破大罗后期，端是神通广大。而且，在他身上先天灵宝众多，端是让人眼红无比。
<br/>　　他有极品先天灵宝十二品业火红莲一座，上品先天灵宝五行旗之一的北方玄元控水旗，元屠、阿鼻两柄杀伐灵剑，杀人不沾因果，端是厉害。
<br/>　　杀人不占因果？元始神秘一笑，杀人不占因果，哪有那么好的事？那是因为他没有遇到能够执掌因果之人，对于像元始这样修炼过因果法则之人，杀人不沾因果简直就是一个笑话。连杀人都不沾因果，那什么才可沾因果？
<br/>　　因果为排名前五的大道法则，岂是一柄小小的上品先天灵宝就可破除的？要是真让它破除了，那才是笑话！
<br/>　　他年我若执因果天罚，我定会让洪荒万物生灵都逃不出因果的制裁！
<br/>　　这一刻，元始想到了很多，他甚至想到了一些前世的因果律武器，是不是什么时候弄两件出来玩玩！
<br/>　　唔！想多了。元始摇摇头，不再多想，而是看着眼前这片无边血海。
<br/>　　似乎是想到了什么，元始心念一动，将身体隐于虚空之中，之后，才踏步向着幽冥血海深处走去。他想看看，这幽冥血海之中到底隐藏着什么，能够与那轮回之地幽冥界相依而生。
<br/>　　元始的隐于虚空可不是一般人的隐身那般简单，一般人的隐于虚空是将自己与虚空融为一体，不可分割，让人发现不了。而，元始的隐于虚空却是进入到另一层虚空之中，与一般人所生存的空间完全不在同一纬度之内。
<br/>　　正所谓，我站在你面前，你无论如何也发现不了我，就是这个意思。
<br/>　　除非是遇到真正精通空间法则的高手，不然，一般大神通者，休想发现元始踪迹。
<br/>　　就这样，元始在无人察觉的情况之下，在这幽冥血海之中大肆搜寻。
<br/>　　……

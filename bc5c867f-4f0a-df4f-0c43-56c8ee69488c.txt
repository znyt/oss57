　　第46章
<br/>　　三天后，黑石城派人将赎金首付送来，包括铜钱一万贯，盐铁粮食若干。负责交付赎金的，竟然就是陆哼。总得来说，交接过程很顺利，在得到寿礼后，陆哼没有再回黑石城，而是直接拉着寿礼奔赴幽州城，如此可见薛半川的焦急程度。
<br/>　　在交接过程中，程大雷能听得到陆哼将牙齿咬得嘎吱嘎吱响，不用怀疑，只要腾出手来，陆哼会第一时间收拾蛤蟆寨。
<br/>　　这段时间差，就是蛤蟆寨的发展机会。
<br/>　　也许是最后机会。
<br/>　　当天下午，程大雷在蛤蟆寨山坡下立了一块用木板拼成的公告牌。上面拓着薛半川亲笔签字盖章的合约，写明凡在蛤蟆寨经过的人，都要交给蛤蟆寨二成利息，所得到的就是由蛤蟆寨护送平安离开青牛山。
<br/>　　同一天，程大雷让徐神机，林少羽几个骑着马在青牛山敲了一圈锣，保证每个山寨都可以听到。
<br/>　　铛铛！
<br/>　　铛铛！
<br/>　　这声音从山下传来的时候，高飞虎正在赶着驴拉磨，心中一激灵，手中的簸箕砸在地上。
<br/>　　“又有好消息！”
<br/>　　连续发生几次这样的事情，高飞虎已经有心理阴影，这次蛤蟆寨又搞什么幺蛾子。
<br/>　　“这次他们贴的是什么？”高飞虎问自己弟弟。
<br/>　　“这次他们没贴……说是都贴在蛤蟆寨外面了，想要知道的，得去哪里看。”高飞豹刚从外面回来。
<br/>　　“派个弟兄去看看，罢了，还是你亲自去，速去速回。”高飞虎明知道不是什么好事，但还是想知道蛤蟆寨又折腾什么。
<br/>　　这几天蛤蟆寨发的财，高飞虎看着眼里，馋在心头。但这也没办法，扪心自问，他是不敢去抢陆哼的，抢也肯定抢不过。可程大雷就做到了，不仅做到了，而且还没有任何伤亡。
<br/>　　这件事无论如何，高飞虎得给程大雷呈上一个服字，自己做不到的事情，人家能做到，你就得服。以前没听说程老大这个儿子怎么样，似乎是个干啥啥不行，吃啥啥不剩的酒囊饭袋，没想到程老大死了，大家都以为蛤蟆寨要完的时候，他却露了出来。连续两场仗，干得真漂亮……
<br/>　　高飞虎看着程大雷的收获也眼馋，有时候也想抢了蛤蟆寨，可陆哼的惨状他还记忆犹新。怎么说陆哼都不该这么惨，可他就是说了，飞虎寨如果想向蛤蟆寨动手，能逃脱这厄运么……
<br/>　　所以如今只是想想，看看以后有没有机会。
<br/>　　想着这些的时候，时间已经过去很久，高飞虎皱起眉头：怎么去这么久都不回来……
<br/>　　高飞豹到蛤蟆岭的时候，看到一座木板下聚了很多人，都是各山各寨的山贼，正议论着什么。
<br/>　　“辣椒……辣椒是什么，和花椒一样吗？”
<br/>　　“收获后每斤辣椒换一斤粮食，这辣椒产量多少……”
<br/>　　“说是至少在千斤以上，一斤辣椒换一斤粮食，那就是亩产上千，不可能！”
<br/>　　高飞豹听了一会儿，又往木板上看了一眼。他勉强认字，所以能读懂上面写的是什么。
<br/>　　意思是，想要种植辣椒的可以到蛤蟆寨领取辣椒苗，等辣椒收获时，蛤蟆寨以一斤辣椒一斤粮食的价格收购。
<br/>　　这……有点扯吧！
<br/>　　平时山寨种田，也就是种些小麦，大豆，粟子……田间地头种些高粱，亩产能收300斤就算高产，如果真按蛤蟆寨所说，岂不是每亩地都能活的上千斤粮食。
<br/>　　“高二当家，你看这件事怎样，靠谱嘛？”
<br/>　　“靠谱，怎么不靠谱，我觉得大家都该信蛤蟆寨的，一亩地上千斤，有几亩，一个山寨就都养活了。”
<br/>　　“那我听二当家的，去取些辣椒苗。”
<br/>　　“回来，没听出来高二当家笑话你呢。”
<br/>　　“哈哈，这娃是那家的，这么天真。”高飞豹愉悦的笑。
<br/>　　“是野猪林的。”有人道：“我觉得也不太真，这种事我做梦梦见过，真发生一次都没见着。”
<br/>　　“对了，有人信他们的嘛？”高飞豹问。
<br/>　　“有啊，怎么没有。杏花岭的小白狼取走了一些，还有秃头坡……其实要不是听二当家这么说，我们几个也想试试呢。”
<br/>　　“去呀，为什么不去，这天上掉馅饼了，你还不张嘴借着。”
<br/>　　“二当家快别说笑了，趁着天还不冷，我们还想种茬大豆呢，可是不能把好田耽搁了。”
<br/>　　“明白这个道理就好。咱们小门小户，可不能跟着蛤蟆寨瞎折腾，至于杏花岭和秃头坡，我们就祝他们发财吧。”
<br/>　　众人哄堂大笑，渐渐人群都散了，最后只有几家取了一些辣椒苗，高飞豹自己也取了些，他不是为了种，只是为了看看这辣椒苗究竟是什么东西。
<br/>　　回到飞虎寨，把发生的事情与高飞虎说了。
<br/>　　“大哥，我去了半亩的苗，要不要找地方种了？”
<br/>　　高飞虎直摇头：“这不是扯淡吗，今年也就能再种一茬豆子，可没有空闲的地让你折腾。”
<br/>　　“要不我就给我一分地，我试试看会长出来什么？”高飞豹。
<br/>　　高飞虎白他一眼，不耐烦道：“堂屋前面有块空地，你收拾一下种在哪里吧。”
<br/>　　今天的事情程大雷也没料得，本以为按一斤粮食一斤辣椒的收购价，辣椒苗肯定会供不应求。结果只有寥寥几个人从自己这里取了辣椒面，让程大雷很失望。
<br/>　　虽然落叶城那边没有消息传来，但程大雷觉得辣椒效果不会错的，不然的话，也不会在短时间内横扫全国。
<br/>　　而问题是，苏樱那边怎么还没消息传来？程大雷觉得，自己该往落叶城走一遭了。第一，是看看苏樱那边情况，第二，是现在山寨有了钱，可以想办法花钱买些奴隶，扩充一下山寨人口，第三点原因么……
<br/>　　自己来这个世界已经有段时间了，从开始就在这穷山沟内，现在都没走出过一步，这城里究竟是什么模样，自己想去亲眼看看。
<br/>　　在一个风和日丽的上午，程大雷换上那套系统奖励的文士服，揣了一把腰刀，骑着瘦马，唤上徐神机。
<br/>　　他上城去了！

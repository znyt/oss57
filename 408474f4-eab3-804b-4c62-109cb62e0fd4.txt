　　　　竖日清晨，最后的比试上演，张小凡运气逆天，在最后一轮比试中，却抽中了一号签，免去比试直接晋级。
<br/>　　相比张小凡，曾书书可就倒霉到了姥姥家，原本他对夺魁就不抱希望，毕竟两人都是高手，陆雪琪手中有天琊神剑，张小凡修为也不俗。
<br/>　　让各位首座目瞪口呆的是，曾书书居然不战而降，风回峰首座，也就是曾书书的父亲，曾叔常却显得异常平静，他本就这个儿子不抱希望。
<br/>　　能够在七脉会武上闯进前四，已经让他脸上有光，要是碰上张小凡或许还有些希望，可碰上陆雪琪，投降并不可耻，众人也都理解。
<br/>　　今日，云海广场上东南西北四座擂台全部拆除，最中央处重新搭建出一座新的擂台。
<br/>　　擂台四周人山人海，数千名青云弟子全都围在此处，各脉首座静坐在高台上，擂台上两人弟子相对而立。
<br/>　　张小凡望着对面冷若冰霜的陆雪琪，眼中忍不住惊艳了一把，这是他第一次如此近距离看青云门的女神，不得不说，确实比田灵儿美太多，让人移不开双眼。
<br/>　　即便如此，他的目光也不敢乱瞄，虽然慕清霄已经答应他，让他夺得桂冠，可心里却依旧直打鼓。
<br/>　　毕竟，对面的绝美师姐脸上冷若冰霜，丝毫看不出她的情绪来，也不知道她会不会放水，若是不放水，他绝对没有胜利的希望。
<br/>　　若不能获得七脉会武的桂冠，他便不能兑现与灵儿的承诺，如此一来，哪怕灵儿愿意，他心中也会有疙瘩。
<br/>　　张小凡思绪万千的同时，陆雪琪也在打量着他，面前这位样貌清秀的少年，就是他的师兄，确实有些实力，难怪能闯入七脉会武前四。
<br/>　　就在这时，一股强大的气息自陆雪琪身躯上爆发而出，张小凡一惊，不敢有丝毫犹豫，玉清境八重巅峰的气息释放而出，两股力量在空中相撞。
<br/>　　张小凡心脏狂跳，暗道：“难道陆师姐与慕师弟没有达成协议，可昨日慕师弟明明告诉他，不用担心......”
<br/>　　以擂台为中心，两股力量相撞所产生的余波向四周扩散，霎时间狂风四起，顿时响起一众弟子倒吸凉气的声音，原本那些以为张小凡纯靠运气闯进七脉会武的弟子，目光都变得复杂起来。
<br/>　　玉清境八重巅峰！
<br/>　　左侧高台上，各脉首座面面相觑，都能看到对方眼中的震惊，看向田不易的目光也变得不同，没想到这家伙门下居然还有资质如此恐怖的弟子。
<br/>　　同时，看向苍松的目光有些戏谑，有些怜悯，他们不得不承认，当年都看走眼了，原本都认为张小凡资质愚钝，争着抢着要收林惊羽为徒。
<br/>　　却不曾想到，当初被他们误认为资质愚钝的弟子，短短数年的功夫，居然已经修炼至玉清境八重巅峰，绝世天才，真正的绝世天才啊！
<br/>　　当初，与他同时拜入青云的林惊羽，现在也不过玉清境七重巅峰。
<br/>　　饶是田不易有心理准备，也是被张小凡身上的气息吓了一跳，气息雄厚，基础扎实不扎实一目了然，没想到自己这个最不起眼的弟子，居然这么强！
<br/>　　苍松道人脸色黑如锅底，这简直就是在狠狠抽他的脸啊，当初他抢到了林惊羽，却失去一个资质更逆天的张小凡，心中羞怒到了极点！
<br/>　　擂台下方，站在人群中，大竹峰一脉弟子则目瞪口呆，眼中满是不敢置信，他们认为最弱的师弟，竟有如此可怕的修为。
<br/>　　田灵儿美目中则满是小星星，纤纤玉手捏着衣角，整一副幸福小女子模样。
<br/>　　此时，面对陆雪琪释放出的浑厚且有些冰冷的内力，张小凡使出全力，苦苦支持，就在他不抱希望的时候，忽然压力全部消散，整个人都轻松了起来。
<br/>　　抬起头来，有些不明所以，陆雪琪行礼道：“师傅，各位师叔，在先前的比试中，雪琪身心疲惫，再无心应战，而且张师资质过人，雪琪自愧不如。”
<br/>　　静，死一样的寂静，一时间整个云海广场变得鸦雀无声。
<br/>　　高台上的各脉首座也都愣住了，刚才陆雪琪的气息，可是完全压制着张小凡，更何况手中还有九天神兵‘天琊’神剑！
<br/>　　水月更是美目大睁，简直不敢相信自己的耳朵，身心疲惫，如此荒唐的理由！
<br/>　　昨日的战斗，她全程看到尾，陆雪琪根本丝毫未损，玉清境九重初期的她，休息一晚上就能恢复全盛状态，现在却选择不战而降，这可是七脉会武桂冠啊！
<br/>　　好气啊！
<br/>　　只是一句身心疲惫就将桂冠拱手让人，脑子里进水了吗？
<br/>　　对于水月来说，自己门下的弟子获得七脉会武的桂冠，心中虚荣心被填满，可现在陆雪琪将触手可及的桂冠拱手相让，她心中的愤怒可想而知。
<br/>　　“陆雪琪！”
<br/>　　水月再也忍不住，猛的站起身来，气的胸脯上下起伏不定，美目盯着自己最喜爱的弟子，望着她那坚定的目光，一时间竟说不出话来。
<br/>　　道玄真人此时也愣住了，不明白这小丫头搞什么鬼，七脉会武桂冠何等荣耀，哪怕是千年后，她的名字也会挂在青云榜上。
<br/>　　只见他站起身来，望着擂台上的陆雪琪，语气平淡道：“雪琪师侄，你确定不争一争吗？”
<br/>　　“三番两次战斗，弟子确实有些倦意，所以......”
<br/>　　说着，陆雪琪还揉了揉太阳穴，做出一副劳累的模样，各脉首座脸上都挂着无奈的表情。
<br/>　　站在大竹峰一脉所在的人群中，慕清霄嘴角勾起一丝笑容，如此看来，自己在她心目中的地位，可远比七脉会武桂冠带来的荣誉要珍贵的多。
<br/>　　望着她的模样，水月则气的浑身颤抖不止，配上她冷艳的模样，让青云门众弟子都产生一种征服欲望。
<br/>　　比起冷若冰霜的陆雪琪，年长的水月对与男性更有诱惑力，当然有这些心思的弟子也只敢在心里丫丫一下，毕竟对方可是小竹峰首座。

陆寒霆看着Dr.川，低沉的嗓音透着几分暗哑，还有茫然，“不知道为什么，我觉得心里空空的，日子过得不好也不坏，对什么都提不起喜欢，睁开眼起身，太阳落下再闭上眼，每天的生活枯燥苍白到近乎机械，今天陆子羡问我，幸福么，我一点都不幸福，甚至连幸福是什么滋味都不知道，我不知道自己想要些什么，却总是感觉到…自己已经失去了生命里最重要的。”
<br/>
<br/>        Dr.川看着男人镀在昏暗光线里的俊脸，他直觉这个男人是受到过极大的心理创伤，但是他现在又活的很平静。
<br/>
<br/>        “陆总，下一次治疗我们试一试催眠术，我走进你的世界去看一看，看看能不能替你找到答案，你究竟失去过什么。”
<br/>
<br/>        ……
<br/>
<br/>        Dr.川走了，陆寒霆回到了自己的卧室里，他冲了一个冷水澡，然后上床睡觉。
<br/>
<br/>        他现在的作息规律很正常，每天晚上都可以入眠，其实他也不知道他的睡眠障碍是什么时候被治好的，总之他再也不会发病，不会暴怒，不会失控，每天晚上也不再做噩梦，一觉可以睡到天亮。
<br/>
<br/>        一切都变好了，他现在变得很好。
<br/>
<br/>        这时“叮”一声，他手机来短信了，是厉嫣然发来的寒霆哥哥，明天你开车送我去学校吧。
<br/>
<br/>        陆寒霆想起Dr.川的建议，没有拒绝，惜字如金的回了一个字好。
<br/>
<br/>        这个“好”字足以让那端的厉嫣然心花怒放了寒霆哥哥，谢谢你啊，我好开心，早点睡，明天不见不散。
<br/>
<br/>        陆寒霆将手机放在了床柜上，然后平静的闭上了眼。
<br/>
<br/>        ……
<br/>
<br/>        此时，帝都城的夏家。
<br/>
<br/>        夏家的大当家夏邦推开了房间门，他这火爆脾气已经上来了，气沉丹田的开口骂道，“绾绾，刚才孙进那个龟孙子是不是给你打电话了，要不要我现在把他套上麻袋给掳来，给你揍上一顿出气！”
<br/>
<br/>        夏夕绾站在雕花阳台上，身上是一件宽松的白裙子，白裙子越发显得她身段玲珑曼妙，清纯的乌发披散肩头，她露出的半个侧颜姣好而绝色。
<br/>
<br/>        三个月没见，她整个人像是脱胎换骨，光站在那里都如明珠一样散发着莹玉般的光泽，那纤尘灵动的气息勾魂夺魄。
<br/>
<br/>        夏邦这大嗓门一吼，夏夕绾迅速将纤白的手指放在唇边做了一个“嘘”，“夏爸，小点声，你吓到我的鸟了。”
<br/>
<br/>        夏夕绾在喂食，她手边有一个鸟笼子，里面有三只毛绒绒的小鸟。
<br/>
<br/>        “绾绾，这都什么时候了，你怎么还在喂鸟，你这鸟都养了三个月了，一直没离身，这不就是天上一抓一大把的鸟儿吗，你要是喜欢我给你抓一林子鸟让你玩过瘾！”
<br/>
<br/>        夏邦生的十分强壮，浓眉大眼就是一个武夫，性格直接又豪爽。
<br/>
<br/>        夏夕绾迅速婉拒，“夏爸，谢谢你，纵然一林子的鸟都不及我这三小只，还有刚才孙进没讨到好，他现在估计气的半死了，我会跟他和平把婚退了的，夏爸，我刚来帝都，请你不要给我闹出太大动静，我们低调点。”

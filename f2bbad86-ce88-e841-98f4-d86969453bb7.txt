　　宫殿之中，所有苍龙宗老祖的目光，都看向了最高处王座之上的中年男子，他就是苍龙宗的宗主，也是做出最终命令的人物。
<br/>　　就是他之前要让金龙卫，擒下楚风眠，带到苍龙宗来的。
<br/>　　“此子挑衅苍龙宗威严，罪无可赦！”
<br/>　　在众人的目光之中，这苍龙宗宗主，冰冷的开口道。
<br/>　　“来人，谁愿意去擒下此子，将此子带到苍龙宗中来，镇压，剥夺灵脉，在带到寰龙城内，斩首示众！以彰显我苍龙宗的威严，不容挑衅！”
<br/>　　“老夫愿意前去。”
<br/>　　那恒龙少主的师尊，第一个开口道。
<br/>　　“我也愿意……”
<br/>　　“我也去。”
<br/>　　其他的几位老祖，也是纷纷开口道。
<br/>　　可就在这议论之时，突然之间一股巨大的冲击，笼罩在整个苍龙宗之内。
<br/>　　苍龙宗内的无数宫殿，都在剧烈的抖动着。
<br/>　　“这，护山大阵被破了？”
<br/>　　一位老祖将你看向天空。
<br/>　　只见在这天空之上，一直笼罩苍龙宗的护山大阵，突然破碎了，被一股巨力，突然轰破了。
<br/>　　这苍龙宗的护山大阵，保佑了苍龙宗数千年，都未曾被人打破过，今日居然是被打破了。
<br/>　　“到底是怎么回事？”
<br/>　　“去看看！”
<br/>　　在这宫殿之中的老祖，纷纷化为遁光，急忙向着外面飞去，想要看看到底是发生了什么事。
<br/>　　“怎么回事，有人来攻打苍龙宗？”
<br/>　　“谁，谁有这个胆子？”
<br/>　　“护山大阵都被打破了！”
<br/>　　不止是这些老祖，所有的苍龙宗弟子，如今都是无比迷茫，一时间吵闹之声不断，不少人都想要弄清楚这到底是发生了什么事。
<br/>　　另一边，就在苍龙宗的天空之上，四道身影，屹立在半空之上。
<br/>　　看着那被打破的护山大阵，整个苍龙宗之内的混乱景象，尹龙仙脸上都是露出了几分苦笑看着楚风眠。
<br/>　　她早就知道，以楚风眠的性格来这苍龙宗，绝不是为了解释什么事来的。
<br/>　　毕竟楚风眠性格的强势嚣张，从楚风眠挥手之间抽了恒龙少主的龙筋，又杀了五位金龙卫的时候，就展现的淋漓尽致了。
<br/>　　楚风眠当初敢抽恒龙少主龙筋，杀了五位金龙卫，就证明从一开始，楚风眠就没有将这苍龙宗，放在眼中过。
<br/>　　如今楚风眠到来，挥手之间，就震破了苍龙宗的护山大阵。
<br/>　　苍龙宗自从成为了苍龙星域的霸主，数千年都没有人敢来攻打过苍龙宗了。
<br/>　　楚风眠已经是数千年以来，第一个打破了苍龙宗护山大阵的人。
<br/>　　“什么人，敢来攻打苍龙宗？”
<br/>　　就在这护山大阵被破的瞬间，一道身影，突然从一座宫殿之中飞了出来，这一道身影的遁光极快，刹那之间他就发现了屹立在苍龙宗上方的四道身影，一个遁光飞了过来。
<br/>　　这一道身影，乃是一名年轻男子，看起来也只有二十岁出头的样子，但是一身实力，已经不在那恒龙少主之下了。
<br/>　　“这是秦龙少主，也是苍龙宗的一位少主。”
<br/>　　尹龙仙在一旁出声道。
<br/>　　这秦龙少主，一个遁光来到了天空之上，他的目光都是看向了楚风眠四人的方向，最终将目光落入到了尹龙仙的身上。
<br/>　　“尹龙仙，怎么回事？你敢带人来攻打苍龙宗？你是吃了熊心豹子胆了？还不快快跪下，跟我去大殿之中，解释解释到底是怎么回事？”
<br/>　　那秦龙少主盯着尹龙仙，直接开口道。
<br/>　　“怎么，攻打一个区区苍龙宗，就算是吃了熊心豹子胆？这苍龙宗实力不怎么样，口气倒是一个比一个大。”
<br/>　　尹龙仙还未开口，旁边的楚风眠，却是赔了撇嘴说道。
<br/>　　“恩？你是何人？”
<br/>　　听到楚风眠开口，那秦龙少主才将目光看向了楚风眠。
<br/>　　他刚刚只是匆匆瞄了楚风眠一眼，便是就没有在乎楚风眠了。
<br/>　　毕竟楚风眠只是一位九劫古帝，根本不被他放在眼中。
<br/>　　他虽然也是九劫古帝，可是这苍龙宗的少主，那一个不是同境界之中堪称无敌的存在，自然没有将楚风眠放在眼中过。
<br/>　　“怎么？你们苍龙宗的人，一直想要见我，现在反倒是不认识我了？”
<br/>　　楚风眠冷笑了一声，打量了一眼那秦龙少主说道。
<br/>　　“滚一边去，不然一会我的手上，又会多一条龙筋。”
<br/>　　“龙筋？你是抽了恒龙少主龙筋的那个人？”
<br/>　　秦龙少主本来还对楚风眠没有任何印象，可听到龙筋两个字，他突然明白了过来，楚风眠的身份。
<br/>　　苍龙宗弟子，在这数千年之间，也就只有恒龙少主一个人，被人抽了龙筋。
<br/>　　听到楚风眠的语气，他一下子明白过来了，楚风眠就是抽了恒龙少主龙筋的那一个人。
<br/>　　秦龙少主看了一眼楚风眠，又看了一眼尹龙仙，怒声说道。
<br/>　　“尹龙仙，看来你是背叛了苍龙宗，居然是跟这小子混在了一起，那好，我就将你们二人一起擒下，带回去审判！”
<br/>　　轰！
<br/>　　在这说完的一刻，秦龙少主已经是凝聚龙力，突然出手了，他一出手，便是龙力凝聚在了掌心之中，化为了一道足有百丈之大的龙爪，横扫而来。

<br/>　　楚风眠刚刚抬起手，想要出手，从楚风眠的旁边，突然间一股灵力迸发而出。
<br/>　　只见在楚风眠身边的老者，一巴掌打了过去，这一道声势浩大的龙爪，在这一巴掌的面前，被震的粉碎。

<br/>　　那秦龙少主脸色大变，没想到站在楚风眠身旁的老者，居然有着如此恐怖的实力。
<br/>　　这一巴掌，根本不是任何的武技，只是简简单单的一巴掌，却是将他全力打出的龙爪，震的粉碎。
<br/>　　“小辈，在公子面前跪下吧。”
<br/>　　狂龙帝看着秦龙少主，平静的说道。
<br/>　　“前辈是谁？为何要来攻打苍龙宗？”
<br/>　　秦龙少主看着狂龙帝，脸色一变的问道。
<br/>　　他看的出来，狂龙帝的一身实力，简直是到了一种深不可测的地步，比起苍龙宗之内的诸位老祖，还要强横。

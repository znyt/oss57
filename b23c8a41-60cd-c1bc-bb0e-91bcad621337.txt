？今天怎么起的这么晚呢，爸你们不上班啊？”吴满弓诧异的问道。<br/>　　<br/>　　“又不是周末，也不放假，你妈可能是昨天累着了睡过去了，你们先吃吧，我过去看看叫她起来”<br/>　　<br/>　　吴国栋回到屋里，来到床边扒拉了两下还在熟睡的许秀琴：“秀琴啊，起来了，吃饭上班了，再不起就晚了”<br/>　　<br/>　　“啊……”睡梦中的许秀琴被招呼醒之后，慢慢的转过身来，吴国栋看见她的脸后顿时就吓了一跳：“你这怎么了，不舒服啊？”<br/>　　<br/>　　许秀琴的脸色很差，一点血色都没有，泛着一股说不出来的灰暗色，四十多岁的年纪瞅着就跟七八十岁老太太的脸一样，倒是没有褶子，就是色太暗了。<br/>　　<br/>　　许秀琴没精神的打了个哈欠，撑起身子晃了晃脖子，说道：“有点累，没劲，我估计是这几天太累了”<br/>　　<br/>　　“行了行了，累也坚持一下吧，这几天加班是有点晚了，后天就周末了到时你好好休息一下”吴国栋催促了几声，就回去吃饭了。<br/>　　<br/>　　十来分钟后，洗漱完的许秀琴强自撑着身子来到了饭桌旁，吴满弓和吴大爷看见后都问她是不是不舒服，许秀琴说可能是累到了。<br/>　　<br/>　　王惊蛰眯着眼睛，看着她的脸色，顿时皱起了眉头。

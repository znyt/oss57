　　　　看到陆雪琪一副疲惫的模样，道玄真人无奈的摇摇头，资质越高就越是任性，后者显然是不想要七脉会武桂冠，可他也没有办法，总不能逼两人接着比试吧。
<br/>　　而且，七脉会武的桂冠已经与他通天峰无缘，至于被其他几脉夺得也无不妥，反正都是青云门的弟子，他们将来都会造福青云。
<br/>　　“既然如此，闫长老就宣布比试结果吧。”道玄真人望着擂台上的愣神的裁判长老说道。
<br/>　　“本次七脉会武决赛，由大竹峰张小凡获胜！”
<br/>　　随着裁判长老宣布结果，擂台四周的弟子陷入短暂的寂静，随即就响起一片谩骂声。
<br/>　　“卧槽，特么的狗.屎.运也太好了点吧，女神怎么会不战而降，不会是看上这小白脸了吧？”
<br/>　　“我可是赌陆雪琪女神获胜，完蛋了，我这些年的积蓄都赔光了！”
<br/>　　“张师兄的修为不弱，我每场比试都在场，真要斗下去，谁输谁赢说不定呢......”
<br/>　　青云门弟子议论纷纷，小竹峰一脉弟子则有些惋惜，陆雪琪获得七脉会武桂冠，她们也跟着沾光，大竹峰弟子依旧处于呆泄状态。
<br/>　　许久回过神来，你看看我，我看看你，眼中都带着不可思议，田灵儿脸颊则有些微红，低着臻首不知道在想些什么。
<br/>　　随着各脉首座各怀心思相继离去，七脉会武最后竟以这种方式告一段落。
<br/>　　......
<br/>　　......
<br/>　　竖日清晨，玉清殿内，各脉首座相继落座。
<br/>　　道玄真人微微一笑，拍了三下掌，殿后立刻有道童走了过来，吩咐了几句，道童点头应了一声，随后向外走去，没有多久，五道身影就被引进玉清殿。
<br/>　　曾书书走在前方，林惊羽与张小凡走在中央，互相攀谈，慕清霄与秦雪琪走在最后面，两人眉来眼去，正是七脉会武前五名弟子。
<br/>　　道玄真人看着殿下的五人，微笑道：“今日让你们五人前来，是有一件事，要让你们下山去历练一番。”
<br/>　　曾书书几人纷纷动容，陆雪琪脸上看不出表情，慕清霄也是一脸平淡。
<br/>　　紧接着，道玄真人便把前往空桑山‘万蝠古窟’一事说了一遍，又道：“此事关系重大，你们五人乃是我门下精英，所以才会派遣你们去查探一番，但魔教妖人奸险毒辣，你们都要小心行事。”
<br/>　　众人齐声道：“是。”
<br/>　　道玄真人满意的点头，继续道：“此外，除了我们青云门外，焚香谷与天音寺都有派出出色弟子前往一同追查，你们在人前不可失礼，但也不可折了我青云门的气势，此外，我门下的萧逸才萧师兄也早已过去空桑山追查此事，你们若找到他，凡事便多商量商量。”
<br/>　　众人皆是对望一眼，齐声答应。
<br/>　　道玄真人细细看了一眼这五个年轻一代的弟子，最后目光落在张小凡身上，招招手道：“张小凡，你上前来。”
<br/>　　张小凡楞了一下，走上前去，道玄真人上下打量了他一番，转头对田不易笑道：“田师弟，你们大竹峰一脉后继有人啊。”
<br/>　　“师兄笑话了。”闻言，就算是满脸严肃的田不易，脸上也露出一丝笑容，倒是对面的苍松道人，眼底闪过一丝微不可查的寒芒。
<br/>　　道玄真人微笑着从袖中拿出一物，递给张小凡，道：“这是本次七脉会武的奖励，收下吧。”
<br/>　　张小凡接过一看，发现是一面小镜子，形状古拙，青铜镂边，上刻着龙，下刻着虎，镜上刻有八卦方位，中间镜片出却不像一般铜镜，反而黄蒙蒙的让人看不清楚。
<br/>　　他心里清楚，这便是慕师弟需要的法宝‘六合镜’。
<br/>　　张小凡还没反应过来，一旁的田不易已经喜形于色，低喝道：“傻小子，还愣着做什么，快跪下谢恩。”
<br/>　　张小凡点了点头，立刻连忙跪下，道：“多谢掌门师伯。”
<br/>　　道玄真人微笑道：“不必，这是你应得的，快起来吧。”说着又向其他人道：“你们也都出去吧。”
<br/>　　众人知道他要传授张小凡六合镜的秘诀，便一起退了出来。
<br/>　　来到殿外，早有一名道童在此等候，田不易无奈道：“清霄，本次你废了龙首峰三名弟子，念你是初犯，掌门真人也是力保你，但小惩还是无法避免的，你便随他去吧，半个月后与小凡他们一起下山历练。”
<br/>　　“是，师傅。”
<br/>　　慕清霄微微行礼，转身跟在道童身后，眼中闪过一丝喜悦，去青云祖师祠堂面壁思过，求之不得。
<br/>　　很快，穿过云海广场，道童带着慕清霄走进通天峰的后山深处。
<br/>　　走来小道上，慕清霄明显察觉到，四周布下了很多阵法，障眼法诸多，普通修士若贸然潜入这里，绝对会惊动道玄真人。
<br/>　　通天峰后山深处有一三岔路口，左边通向青云圣地幻月洞府，为青叶闭关悟道之地，历来之允许掌门进入，相传无名古卷和诛仙剑便是得于此处。
<br/>　　洞府取名幻月，乃是传闻在明月之夜，洞前有奇石缤纷绚丽，如梦如幻。
<br/>　　但更重要的是，洞府石壁上的太极图机关，可凭借太极玄清道上清境界打开右边的幻月之道，人走入之后，如堕入幻梦之中，前尘往事历历在目，非心志坚定者便会深陷其中，不可自拔，一身修行道行毁于一旦。
<br/>　　三岔路口的右边通向供奉青云门历代祖师的祖师祠堂，从开创青云门的青云子到青叶祖师再到历代先辈，都在这祖师祠堂中有着灵位，每日香火不绝。
<br/>　　而且每逢重要日子，青云门都会在掌门带领之下，到此隆重祭祖。
<br/>　　慕清霄深深的看了一眼左边，不知通向何处的道路，嘴角勾起一丝笑容，第五卷天书‘诛仙剑’便在最深处，自己被罚着去祖先祠堂，倒是省了不少麻烦。
<br/>　　走了将近二十分钟后，慕清霄的视线中出现一座建筑，与大竹峰上的守静堂相似，想必这就是祖先祠堂了。
<br/>　　来到门口，道童行礼道：“掌门真人说，师兄在祠堂内面壁半月即可离去。”

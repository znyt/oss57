        虽然这件事情闹得很大，但是犯事的那些学生的信息还是受了保护并未对外公开。
<br/>
<br/>        连星儿拿起筷子，夹起一块刺身在酱油芥末里抹了抹，回答道：“在学校里，也就只有她喜欢跟我这么折腾。”
<br/>
<br/>        “那你还这么淡定？任由她欺负你？”明司寒剑眉微蹙。
<br/>
<br/>        连星儿一口将蘸了芥末的刺身塞入嘴中，被辣得鼻子够呛，连忙捂住了嘴巴。
<br/>
<br/>        直到她把嘴里的食物咽下肚后，才开口回答道：“不是我对她放纵，也不是我任由她欺负。而是我压根就不在乎。再说了，她要是做得过火。我爹地会收拾她的。”
<br/>
<br/>        “怪不得你爹地要我入赘。”明司寒听着连星儿这话，不禁喃喃自语了一句。
<br/>
<br/>        他说话的声音很小，语速又快，连星儿压根就没听清楚他说了什么，于是反问了一句：“你在说什么啊？”
<br/>
<br/>        “没什么。”明司寒会心一笑，也拿起筷子吃了起来。
<br/>
<br/>        反正这件事情，始作俑者的朱小溪被判了三个月的牢狱之灾，其他涉事人员只是被处罚款和思想教育。
<br/>
<br/>        虽然确实挺闹腾，但至少封住了悠悠众口。
<br/>
<br/>        “喂。”连星儿忽然想到了什么，盯着明司寒，一本正经地问道，“你是不是也看了那些照片？”
<br/>
<br/>        “当然。”明司寒毫不避讳地坦白。
<br/>
<br/>        连星儿微微一笑：“那好不好看？”
<br/>
<br/>        她的话一落。
<br/>
<br/>        “咳咳咳——”他便捂着嘴咳了起来，真的是被她这突如其来的一句话给呛到了。
<br/>
<br/>        “你没事吧？”连星儿关问道。
<br/>
<br/>        明司寒顿时脸色阴沉：“你一个女孩子家家的，能不能矜持点，嗯？”
<br/>
<br/>        “肯定有不少男生把那些相片保存了。”连星儿一副无所谓地说道。
<br/>
<br/>        明司寒会心一笑：“你放心，那些相片，除了法院案件档案里的纸质档案，其它地方绝对没有第二份。”
<br/>
<br/>        不过话说回来，连星儿看着明司寒脸上的笑容时，心里还是觉得暖暖的。
<br/>
<br/>        “怎么样，我说话算不算数？”明司寒忽然挑眉，又补充了一句，“我说过，在这学校里，以后没人再敢欺负你。”
<br/>
<br/>        “嗯嗯，真的很算数！”连星儿如小鸡啄米般连连点头，“所以，解除婚约这事交给你去处理，我最放心不过了！”
<br/>
<br/>        每次跟他在一起时，她总是三句话不离解除婚约。
<br/>
<br/>        起初，明司寒听着还会觉得不大舒服，现在，他似乎是对她的这句话已经产生了免疫。
<br/>
<br/>        殊不知，跟她解除婚约这事，他真的从来就没有认真地去考虑过。
<br/>
<br/>        和连星儿吃过饭后，明司寒将她送到了宿舍楼下，转身时，接到了爷爷明护国打来的一通电话。
<br/>
<br/>        “小寒，你老实跟爷爷说，是不是傅夜沉逼迫你答应他，入赘他们傅家？”明护国气呼呼地说道。
<br/>
<br/>        明司寒淡淡地应了声：“没有啊！我自愿的。”
<br/>
<br/>        “什么？”顿时，明护国的气不打一处来，“你是想把你爷爷我气死对不对？你可是你爷爷我最疼爱的小孙子！怎么能入赘他们傅家？他们傅家也配？”
<br/>
<br/>        “爷爷，冷叔叔就星儿那么一个女儿，而我爸妈有三个儿子，您更是有十个孙子。我入赘，又没什么损失。”明司寒不以为然地解释。
<br/>
<br/>        “你入赘就是给我们明家丢脸！”明护国不禁大发雷霆起来。
<br/>
<br/>        明司寒识趣，连忙安慰道：“爷爷，你反过来想想。你拿联姻的事情压制傅家，傅家自然会整出点幺蛾子来挫挫你的锐气，是不是？爷爷这个时候越是生气，岂不是正合了冷叔叔的意？您若是不答应冷叔叔的要求，这婚事十有八九得黄！”
<br/>
<br/>        “你……”明护国欲言又止，毕竟这小孙儿说得并不是没道理。
<br/>
<br/>        明司寒忍俊不禁地接着说道：“爷爷，您真的不要去介意这件事情。入赘又不代表你孙儿我是个废物。更何况，你孙儿我怎么说也是个计算机信息科技公司的老板。就算是入赘了，我也能自食其力。”
<br/>
<br/>        “小寒，你跟爷爷说，你是不是真的喜欢上人家连星儿呢？”明护国忧心忡忡地问。
<br/>
<br/>        明司寒怔了一下，立马矢口否认：“怎么会！我对她喜欢谈不上，但也不讨厌。反正是联姻嘛！将来能过日子就行。爷爷，您跟奶奶不也是这样过来的吗？”
<br/>
<br/>        “爷爷劝你不要投入真感情，傅家和明家的恩怨，真的不是你和连星儿之间的联姻就能消除的。”明护国语重心长地说。
<br/>
<br/>        明司寒脸上的笑容渐敛，心不在焉地应了声：“嗯，我知道的。”

　　第489章
<br/>　　程大雷用目光逼视着奥黛琳娜，心中的怨气未消。这小姑娘为了自己与刘悲在一起，便被李婉儿推了出去。这件事程大雷放不下，非得给她点小鞋穿。
<br/>　　和珅含笑道：“刘军师也的确给有房妻子，这是山寨的大事，以刘军师的身份，不管是谁都是她的福分。”
<br/>　　“喔，和大人有没有中意的人选，定要才貌双佳，还需贤惠淑德。”程大雷有意无意的扫过奥黛琳娜。
<br/>　　刘悲也有些尴尬：“这件事不着急，不着急。”
<br/>　　那个少年不怀春，其实刘悲也一样。他平日是没考虑过这件事，今日被程大雷一句话勾起了心思。
<br/>　　程大雷和和珅你一言我一语，说得分外热闹，看东家的女如何，西家的女怎样。
<br/>　　此刻忽听得一声轻咳，奥黛琳娜昂着头道：“我看我就挺好。”
<br/>　　“咦！”
<br/>　　和珅、程大雷和刘悲都愣住了。程大雷是早知道奥黛琳娜的心思，而和珅是拔根眼睫毛都能当哨吹的人，有什么看不明白的，只不过陪着程大雷打趣奥黛琳娜而已。
<br/>　　二人吃惊的是，奥黛琳娜竟能这样坦率，不免多看了她两眼。
<br/>　　至于刘悲，平日也算机灵，但在这件事上他当真没开窍。一下愣住了，张大嘴巴，不知该说什么好。
<br/>　　程大雷原先还计划着，把孙尚香召唤出来，给刘悲凑个官配。不过世事难预料，谁知道被奥黛琳娜截了胡。
<br/>　　不过，程大雷因为暴恶的属性，想要把孙尚香这种正面角色召唤出来，几乎是不太可能了。
<br/>　　程大雷瞥了奥黛琳娜一眼，又望了望刘悲，心道：便宜你这小子了。
<br/>　　“大当家，这实在是，实在……”
<br/>　　“实在是什么？”程大雷板起脸，这种情况，他觉得自己应该严肃些。
<br/>　　刘悲尴尬道：“属下一心为公，早已无心嫁娶之事，还请大当家收回成命。”
<br/>　　“快别扯淡了。”程大雷盯着刘悲，道：“这天大的好事砸到你身上，你还有什么不乐意的？”
<br/>　　刘悲支支吾吾，程大雷本以为刘悲是一时拉不下脸，到后来渐渐琢磨过来，刘悲可能真的不太情愿。
<br/>　　帝国一直有中央大国的心思，虽然现在已经打得稀巴烂了，仍旧有这心思。对胡人女子帝国上层还是蛮喜欢的，但要娶为正室，却是有点给祖宗蒙羞的意思。就算刘悲现在已经是山贼了，仍看不开这一点。
<br/>　　奥黛琳娜脸若寒霜，怒而怨的瞪了刘悲一眼，然后目光看着程大雷。此刻似乎只有程大雷能帮他了。
<br/>　　“好，这一次我就做回主。”程大雷道：“你们两个的事定了下来，择日就把亲成了，如今的山寨，也该有件喜事了。”
<br/>　　奥黛琳娜眼中一亮，踏踏实实按照帝国礼仪向程大雷磕了个头，道：“多谢大当家成全。”
<br/>　　这还是奥黛琳娜第一次对程大雷这样称呼，看来，现在是把自己当成了蛤蟆寨的人。
<br/>　　而刘悲还有些犹豫不决：“大当家，这……”
<br/>　　“好啦，你不要说话。”程大雷道：“你暂时被剥夺了话语权，这件事就这么定了。”
<br/>　　程大雷使用绝对权力，一句话敲定了这件事。
<br/>　　毕竟山寨还有很多事情要处理，他也没办法为这件事多费心，至少从回来后，他连好好歇一歇都没有。
<br/>　　程大雷从刘悲院子离开后，回了自己的城主府，今夜在苏樱房中歇了。到第二天，程大雷便将刘悲、徐神机、和珅、阿喜、关鱼等山寨主要人物叫到一起，召开了一场会议。
<br/>　　这次会议的主要议题，就是分析帝国未来的形势，以及蛤蟆寨又该在这种形势中如何抉择。
<br/>　　大家分析讨论，甚至伴随着一些争吵，最后得出一个结论：正义教的覆灭，并不能让帝国形势发生转变，不会越来越好，只会越来越糟糕。
<br/>　　总的说来，正义教不是帝国的问题，是问题结出的果，表面上看是林少羽要为父报仇，但根底里却是无数吃不上饭人的愤怒。总而言之，没有正义教，也会有邪恶教，不太正义教，不太邪恶教之类的势力冒出来，反抗帝国的统治。
<br/>　　帝国从根上已经烂了，未来只会越来越糟糕，伴随而来的，是更多的战争，更多的民不聊生。
<br/>　　然后，讨论回到蛤蟆城本身。
<br/>　　蛤蟆城目前的人口平民约有十万，常备军在三万左右，因为前几年风调雨顺，山寨还有些存粮，大概能支撑一年左右。
<br/>　　程大雷这次带回来许多宝藏，都是价值万金的宝物。说是如此说，但盛世古董乱世黄金，在乱世中，这样的宝物是有价无市。
<br/>　　而频繁的战争，必然会对商业产生巨大破坏，日后蛤蟆寨的商路怕是已无法源源不断带来财富。
<br/>　　这场会议从上午开始，一直持续到晚上，众人用过饭后，继续讨论。
<br/>　　最后得出结论，山寨必须自给自足，包括粮食、布匹、纸张、盐、铁等生活必需品。趁着现在年景好，多开垦些荒田，做好粮食储备。另外也许发展养殖和畜牧。
<br/>　　“大当家。”和珅提出了自己的看法：“如果战争打起来，当然，肯定是会打起来的，怕是会有许多流民涌到蛤蟆城，到时，我们该怎么办，接不接受？”
<br/>　　程大雷几乎没有任何犹豫：“这个需要看山寨的发展，如果可以容纳更多人，当然是人越多越好。但如果我们的情况不是很好，那么，我们也不能让流民将我们吃穷了。”
<br/>　　程大雷的看法，得到了所有人的赞成。在这样一个世道，自然是个人顾各人，至于圣人，乱世是不生产圣人的。
<br/>　　“我们放过凉州城不管，也是因为这个考虑。”程大雷道：“战争发生时，凉州城会成为我们的屏障，我们还是抓好生产，只有吃饱了，才能想别的事。当然，对于凉州，以及其他各地的情报千万不可放松。”
<br/>　　阿喜起身答应，蛤蟆城的情报工作一直由他负责。
<br/>　　“那么戎族呢？”刘悲问：“我们和戎族接壤，假若戎族打过来，我们该怎么办？”
<br/>(本章完)

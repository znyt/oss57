
这天夜间，王惊蛰，陈三岁和孔良夜宿在了山上，在山下离他们直线距离不过十五公里的地方，则是三井美奈那一群人在扎营，至于那条獒王在捕猎进食之后，就栖息在了一片灌木丛中。<br/>　　<br/>　　而稍早一些的时候，黄九郎乘坐直升机抵达了帮拉桑村进行调查，找到了三头僵尸被撕碎了的痕迹，也从老桑杰的口中打听到了王惊蛰为何离去的消息，他当时也有点没太想通，对方怎么会为了追一条狗，而如此急促的改变了本来定好的行进方向。<br/>　　<br/>　　这个时候，军分区的三辆卡车也已经开到了距离喜马拉雅山脉三十公里处的地方，车上跳下来一队士兵，荷枪实弹全副武装，这是军分区的某个突击队，常年在边境执行任务。<br/>　　<br/>　　三方人马，汇集在了茫茫的雪域高原上，正在被隐藏的一条线，逐渐的牵连到了一起。<br/>　　<br/>　　深夜时分，三人轮流守夜，这时正是王惊蛰当班的时候，他就随意的坐在离那头獒王不过二十米左右远的地方，望过去就能看见个庞大的身躯睡在草丛里，还能隐约听见一点呼噜声。<br/>　　<br/>　　这狗子睡得相当香甜了，仿佛没有任何戒备的心思。<br/>　　<br/>　　“哈……”感觉有点疲惫，王惊蛰打了个哈欠，然后从身上掏出烟来“啪”的一声点着了火，刚抽了一口，他忽然就看见獒王的脑袋动了一下，黑夜里两只泛着绿光的眼睛正在朝着这边望了过来，一眨一眨的。<br/>　　<br/>　　这要是碰见豺狼虎豹什么的，王惊蛰就得绷紧神经打起精神来了，但寻常的狗智商都相当于一个四五岁的孩子了，一头可能有了灵智的獒王它的智商等级可能就会更高，王惊蛰不信对方会毫无理由的暴起伤人，只要是自己别露出敌意就行了。<br/>　　<br/>　　獒王躺在地上仰着脑袋眨着眼睛看向了他，王惊蛰寻思了下，忽然回过身从地上抓起背包拉开后从里面拿出一袋牦牛肉条来，他撕开包装拽出一根牦牛肉就朝着那边扔了过去。<br/>　　<br/>　　“啪”牛肉条掉在地上，离獒王就两三米的距离，它眼神往下一落，然后嗅了两下鼻子，王惊蛰则是慢慢的试探着往那边挪着步子，然后又拽出一根牛肉条再次扔了过去。<br/>　　<br/>　　“吃吧，吃吧……”<br/>　　<br/>　　王惊蛰走过去后，在离獒王三四米的地方停了下来，这时候一人一狗离的就比较近了，甚至彼此间的眼神都可以清晰可见。<br/>　　<br/>　　离得如此之近，王惊蛰也确信孔良说的应该没错，这绝对是一头獒王无疑了，从外表他不会分辨，但气息他能察觉的出来，这就跟人一样，高手相见，气质决定感觉。<br/>　　<br/>　　王惊蛰又拿出一根牛肉，在手里掂了两下：“来，啧啧啧……”<br/>　　<br/>　　“唰”獒王猛地抬起脑袋，动作幅度之大吓得王惊蛰就往后退了几步，随即把手里的牛肉就扔了过去：“哥们没有恶意，这牛肉味道也不错，你试试？”<br/>　　<br/>　　王惊蛰一边说话一边朝着它点了下头，然后指了指地上的几块牛肉，为了向对方表达自己的善意，他索性直接就坐在了地上，摆了个身心愉悦没有任何防备的姿势，王惊蛰知道如果这獒王的智商颇高的话，是完全可以读懂他这一串动作中的含义来的。<br/>　　<br/>　　果然，獒王有反应了，略微歪着脑袋，眨了眨眼睛，王惊蛰仔细的盯了几眼，觉得对方的眼神似乎有点轻蔑，就跟看着白痴似的，那意思是你让我吃我就吃，朋友你当我是三岁小狗呢？<br/>　　<br/>　　“哎呀我擦，你还挺有脾气的”王惊蛰无语的翻了个白眼，然后撅着屁股往前爬了两下，从地上捡起一块牛肉塞进嘴里“嘎吱，嘎吱”的吃了几口。<br/>　　<br/>　　“真香！”<br/>　　<br/>　　獒王地下脑袋，鼻子嗅了几下后，也找到一块牛肉，先是用舌头舔了舔，似乎感觉味道不错，就谨慎的抬了下眼皮看了王惊蛰一眼，他笑呵呵的又把剩下的牛肉塞进了嘴里。<br/>　　<br/>　　“好吃，好吃……”<br/>　　<br/>　　“唰”獒王一口叼起地上的牛肉，嚼了两下就咽了进去，紧接着又把地上另外两块也给叼了起来。<br/>　　<br/>　　“唉，这就对了！”王惊蛰一股脑的将袋子里剩余的牛肉全都扔了过去，然后一脸傻笑的看着獒王，这种现象无疑是把一人一狗之间的距离给拉近了不少，无疑，狗会吃你给的东西，至少已经对你放下了戒备心。<br/>　　<br/>　　獒王不管多聪明，那也是个没受过训话的牲口，灵智虽然有了，可在某些方面还是差了点，特别是对于食物的诱惑，这是动物的天性。<br/>　　<br/>　　一袋牛肉被獒王给吃的干干净净，这也正好赶上对方睡了一觉醒来之后有点饿了，换成它吃饱的时候，这招可能就差点了，獒王如此巨大的体型对于食物的摄取量是非常大的，它一天中差不多有四分之一的时间都是在进食的，否则很难支撑住身体所需要的热量。<br/>　　<br/>　　王惊蛰眼见最后一块牛肉也被它给吞了进去，就从地上爬了起来，有心思想要在接近点，但是突然之间，从他右后方的草丛里忽然蹿出一道黑影，王惊蛰顿时回头，就看见一只藏羚羊速度很快的在他身后掠过，同时獒王扭着脑袋也看见了。<br/>　　<br/>　　“么的！”这头不知道从哪蹦出来的藏羚羊出现的太是时候了，从他身后一掠过，碰见猎物的獒王紧接着就也追了过去，这明摆着是碰上了夜宵啊，它哪里会就此放过啊。<br/>　　<br/>　　王惊蛰在心里暗骂了一声，藏羚羊出现的太是时候了，不但打断了他和狗子之间刚刚建立起来的熟悉的感觉，同时两头牲口如此快速的离去，就更麻烦了。<br/>　　<br/>　　“起来，快，起来，起来，狗跑了！”王惊蛰冲着正在帐篷里熟睡的陈三岁和孔良喊了一嗓子，随即人就朝着一狗一羊追了过去。<br/>　　<br/>　　在这种山地上，藏羚羊的时速能够达到几十公里以上，獒王同样差不了多少，但是两条腿的人就不行了，王惊蛰狂奔了片刻，还能看见两道黑影，可仅仅咋了几下眼睛，藏羚羊和獒王就全都不见了踪影。

        再次醒来，已经是第二天清晨。

<br/>        诺筱颖脑海里的思绪一滞，猛然想起昨晚的事情后，倏地坐起身来。

<br/>        下身那硬生生撕扯开的痛楚，瞬间让她出了一背的冷汗。
<br/>
<br/>        昨晚，她好像被人给……
<br/>
<br/>        诺筱颖连忙转头打量着床单，在看到床单上的斑斑血迹后，才知道这一切并不是在做梦。
<br/>
<br/>        她的第一次真的就这么稀里糊涂的给没了！
<br/>
<br/>        眼泪滑落了下来，她现在该怎么办，虽然她知道那一张膜并不能代表着什么，可是，她已经有男朋友了啊！
<br/>
<br/>        如果到时候，何明旭发现了她根本不是初次之后，一定会在心里狠狠的唾弃她是个脏女人吧！
<br/>
<br/>        不行，这件事不能这么算了，否则太便宜那个男人了，她要报警！报警把他抓起来！
<br/>
<br/>        诺筱颖拿起手机，刚按下了一个‘1’就停下了手指。
<br/>
<br/>        如果她报了警，那么她被人入室强女干的事情，肯定会人尽皆知，到时候，不仅何明旭会离开她，她的学业也会因此而葬送的！
<br/>
<br/>        就在不知道该怎么办的时候，客厅的门忽然咔哒一声被人从外打开，是她的合租舍友苏漫雪回来了。
<br/>
<br/>        诺筱颖吓得连忙起身，然后把被染红的床单揭了下来。
<br/>
<br/>        脖子上被一块石头样的东西砸的生疼，诺筱颖狐疑的取了下来，发现那竟然是一块白色的玉坠！
<br/>
<br/>        玉坠的质地非常的好，一看就价值不菲，上面刻着一个夜字，旁边缠绕着一道龙纹。
<br/>
<br/>        这样的东西，绝对不会是她或者苏漫雪的，那么……就只能是那个男人戴在她脖子上的。
<br/>
<br/>        想到这，诺筱颖一阵恶心，想都没想就把玉坠拽了下来，准备趁着苏漫雪回来之前，从窗子里扔掉。
<br/>
<br/>        可谁知，手刚抬了一半，苏漫雪就已经推着门进来了，一边进来一边抱怨着：“哎呦，累死了，这样的苦日子什么时候是个头啊，真是够了。”
<br/>
<br/>        苏漫雪说着，抬起了眸子朝诺筱颖看去，看到她奇怪的姿势之后，立马问道说：“你干嘛呢？”
<br/>
<br/>        诺筱颖只好收回了手，心虚的回答：“没，没干嘛。”
<br/>
<br/>        “没干嘛是干嘛？”
<br/>
<br/>        苏漫雪超诺筱颖走了过去，他们两个是老乡，又是同学，现在还一起出来勤工俭学合租在一起，以她们朝夕相处的经验来看，诺筱颖此时一定是有事在瞒着她。
<br/>
<br/>        诺筱颖想要把玉坠藏起来，谁知刚一转身，就被苏漫雪一个跨步抢了过去：“手里藏着什么宝贝，给我看看！”
<br/>
<br/>        下一秒，那个白色的玉坠就在苏漫雪的手心荡漾着。
<br/>
<br/>        苏漫雪看着玉坠，眼睛都发直了，因为她平时最喜欢研究珠宝首饰，所以一眼就看出来了，这块玉坠一定价值不菲。
<br/>
<br/>        “筱颍，这个坠子谁送给你的呀，肯定不是何明旭吧，他自己都靠你打工赚钱养活呢，哪里有钱去给你买这么贵的礼物。”苏漫雪说着，一半羡慕，一半嫉妒。
<br/>
<br/>        诺筱颖这丫头平时清纯的跟什么似得，好像多视金钱为粪土，多高尚一样，结果呢，还不是背地里背着自己的男朋友收了这么贵重的礼物！
<br/>
<br/>        诺筱颖闻言，连忙解释着：“这不是什么贵重的东西，你看错了，就是个赝品而已。”

<br/>        “是吗？”
<br/>
<br/>        苏漫雪重新把目光投向了玉坠上，在看到上面刻着的那个夜字的时候，微微的蹙了下眉头：“这里怎么有字？”
<br/>
<br/>        诺筱颖一听，心脏心虚的狂跳了起来，她向来不会说谎，如果被苏漫雪发现什么端倪，知道她在昨夜失了身，她以后还怎么做人。
<br/>
<br/>        于是，在苏漫雪还想问什么的时候，诺筱颖直接说道：“漫雪，你不是喜欢玉制品吗，这个坠子送你好了，对了，公司有着急的任务找我，所以我就先走了！”
<br/>
<br/>        说完，诺筱颖抱着床单就准备离开，苏漫雪立马问：“你拿床单干什么？”

<br/>        “哦，外卖洒在上面，弄脏了，我拿去丢掉。”说完，诺筱颖就抱着床单头也不回的走了，直至走出了房间，才狠狠松了口气。
<br/>
<br/>        呼，好险。
<br/>
<br/>        屋内的苏漫雪，把玉坠翻来覆去打量了好几遍，然后才走去卫生间，戴到了自己的脖子上。

<br/>        发现这玉确实衬的人肤色好，不过是戴了块玉整个人的气质却完全不一样了，只可惜是个赝品，否则诺筱颖也不会这么大方的送给她。

<br/>        想到这里，苏漫雪撇撇嘴就准备摘下来，恰在此时，门外传来嘟嘟的门铃声。

<br/>        诺筱颖不是说上班了吗？怎么又回来了，自己就不会带钥匙吗！

<br/>        苏漫雪吓了一跳，警惕的问：“你们找谁？”

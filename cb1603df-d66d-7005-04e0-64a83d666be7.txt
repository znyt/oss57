　　听到蝎狮王的询问，楚风眠也是打起了精神，等待着梦烟罗的回答。
<br/>　　少帝可是踏入了半步天人境的强者，如今他发现了这净土佛寺，居然不是一个人独战，而是请了蝎狮王，龙摩罗这种魔族强者前来。
<br/>　　显然是这净土佛寺之中，也有着不少棘手的东西，需要龙摩罗，蝎狮王这样的人物去解决。
<br/>　　“不算什么棘手的东西，一些残留在其中的剑气罢了，这些剑气，经历数十万年时间，威力已经不值一提了，以你们的力量，足以轻松对付。”
<br/>　　梦烟罗轻描淡写的开口道。
<br/>　　剑气？
<br/>　　楚风眠跟蝎狮王，表情之中都是露出了几分惊讶之色。
<br/>　　显然这个结果，似乎是在他们的意料之外。
<br/>　　净土佛寺中，居然有剑气？
<br/>　　血邪魔帝，精通佛门魔道秘术，可没有任何的记载，血邪魔帝，居然懂得剑气。
<br/>　　这净土佛寺之中的剑气，又是从何而来？
<br/>　　难道这血邪魔帝的陨落，还有着其他的秘密？
<br/>　　楚风眠脑海之中推测了半天，也没有想出来一个结果。
<br/>　　毕竟血邪魔帝，净土宗，距离如今的时代之中，足有数十万年了。
<br/>　　在古籍之中的一些记载，也都是寥寥几笔。
<br/>　　像是楚风眠，在古籍之中了解到的，也最多就是知道了血邪魔帝，净土佛祖这两位人物。
<br/>　　而像是这两位人物，居然同是一人这样的秘密，在任何古籍之中都没有记载。
<br/>　　楚风眠还是从龙摩罗的口中，听到过的，以楚风眠的了解，根本难以查清楚其中的秘密。
<br/>　　梦烟罗的话，也是模棱两可，只是一笔带过了，也不愿意详细的去说。
<br/>　　楚风眠也知道，追问下去也没有任何的转移，便是换了一个话题询问道。
<br/>　　“那少帝呢？少帝如今在哪里？”
<br/>　　楚风眠最为担心的，还是少帝，他伪装成的龙摩罗，虽然足以瞒过蝎狮王，梦烟罗等人。
<br/>　　但是能不能瞒得过少帝，楚风眠还依然不清楚，对于少帝，楚风眠必须要时刻保持警惕。
<br/>　　“少帝为了对付一些麻烦，现在并不在，不过等到了净土佛寺开启之时，少帝自然会现身。”
<br/>　　梦烟罗随意的开口道，同时他们的遁光，也是逐渐靠近了净土佛寺。
<br/>　　净土佛寺，金碧辉煌，在万千佛寺之中，都是至高无上的巅峰。
<br/>　　如果不是楚风眠知道内情，谁能想到这样一尊佛寺的主人，居然会入魔，成为一尊杀戮无情的魔帝。
<br/>　　“你们现在这里等待着，佛寺开启，你们就可以进入其中，记住，其中的宝藏，谁得到的，就属于谁。”
<br/>　　梦烟罗带着楚风眠跟蝎狮王，降落到了净土佛寺附近的一处山峰之上，便是消失的无影无踪，。
<br/>　　楚风眠跟蝎狮王的身影落下。
<br/>　　刚刚踏入这山峰之上，楚风眠就发现，在这周围隐藏着不少强横的魔族。
<br/>　　“龙摩罗，蝎狮王，你们也来了。”
<br/>　　一道身影，从山峰之上现身而出，走了过来，开口道。
<br/>　　这是一尊中年男子，看到他的样貌，楚风眠倒是感觉到了几分眼熟，回想一下，正是楚风眠之前从外面，见到进入这山谷之中的几位魔族之一。
<br/>　　“夜魔王。”
<br/>　　楚风眠得到了龙摩罗的记忆，这中年男子的身份，楚风眠便是一眼认了出来。
<br/>　　夜魔一族，也是魔族之中强横的种族之一，这一尊夜魔王的实力，不亚于九劫古帝之中的超级天才。
<br/>　　足以跟凰公子，小耀王寒夜媲美的人物。
<br/>　　当然比起龙摩罗这种九劫古帝中的霸主，还相差一些，他走过来的时候，语气也是无比客气。
<br/>　　毕竟不管是龙摩罗，还是蝎狮王，都是九劫古帝中的霸主。
<br/>　　他虽然也是九劫古帝，可也不敢轻易得罪这二人。
<br/>　　“没想到这一次居然是聚集了这么多的熟人。”
<br/>　　蝎狮王扫视了周围一眼，眼神之中的阴冷之色，丝毫不减，他回头看了一眼楚风眠，冷声道。
<br/>　　“龙摩罗，今日不适合争斗，你我之间的恩怨，改日在清算吧！”
<br/>　　蝎狮王身形一闪，直接离开了，选择跟几尊魔族的身影，靠近在了一起。
<br/>　　显然是他跟这几尊魔族，都是好友，组成了一个小团体。
<br/>　　这一次，净土佛寺开启，按照规矩，所得到的宝藏，谁得到的便是谁的。
<br/>　　如果几尊强者，联合行事，进入净土佛寺之中，也会有更多的机会，得到更多的宝藏。
<br/>　　“龙摩罗，不知道你有没有兴趣，跟我联合？”
<br/>　　那夜魔王看到蝎狮王离开，也不意外，而是看着楚风眠开口邀请道。
<br/>　　“我知道你的实力强横，这一次如果联手，得到的宝物，可以你占大头，不会让你吃亏的。”
<br/>　　“不错，龙摩罗，这一次净土佛寺中的危险，非同小可，不然少帝也不会将净土佛寺的消息，告诉我们的，我们联手行事，也会安全许多。”
<br/>　　又是一尊魔族强者，靠了过来，此人的身影虚幻无比，却是虚魔一族的王者，虚魔王。
<br/>　　他显然是跟夜魔王已经是联合了起来，准备来拉拢楚风眠。
<br/>　　“本帝向来独自一人，不需要跟人联手。”
<br/>　　楚风眠狂傲无比，冷笑连连开口道。
<br/>　　“什么危险，也能难得到本帝？这一次我自己进入其中，不会跟任何人联手，联手？不过是弱者的行为罢了！”
<br/>　　楚风眠拒绝的话无比强硬。
<br/>　　跟这夜魔王，虚魔王联手的话，楚风眠进入净土佛寺之中，也就不能自己行动了。
<br/>　　他的目的可是为了人皇罐碎片而来，至于其他的宝藏，对楚风眠并没有太大的吸引力。
<br/>　　而且一旦是跟人联手，暴露身份的可能性，也会更高。
<br/>　　在加上龙摩罗，本就是性格狂傲，就连朋友都没有几个，做出这样的反应，也是无比正常。
<br/>　　楚风眠说完，便是直接踏步离开，不去管那夜魔王跟虚魔王二人。
<br/>　　“这龙摩罗，真是狂妄，没想到这么多年过去，此人居然越来越发的狂妄了！”
<br/>　　看着楚风眠离开，那夜魔王才脸色不善的开口道。
<br/>　　他可也是魔族强者，来邀请楚风眠，开出了这么丰厚的好处，却是被这样干脆利落的拒绝了让，让他也无比难堪。

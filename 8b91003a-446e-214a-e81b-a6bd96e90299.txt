　　望着满地的尸体，慕清霄道：“血宗没有惊动任何势力，埋伏在这里，煞费心机，绝对不可能让天蛇府有人逃出去。”
<br/>　　此时，青长老在半空中，被迫接下范崂一掌，强横的劲力直接将她打的口吐鲜血，神色萎靡。
<br/>　　范崂则紧追其后，丝毫不给她喘息的机会。
<br/>　　爆退的同时，青长老猛然抬头，原本美丽的脸颊此刻却布满狰狞，玉手一晃，一只寒玉盒便出现在掌心。
<br/>　　“范老鬼，你若是在敢过来，老娘就当场将这阴阳玄龙丹捏成粉末！”
<br/>　　闻言，前扑的范崂止住脚步，阴恻恻的望着青长老，道：“你若是敢毁了阴阳玄龙丹，本宗就废了你的斗气，将你锁在血宗，当成猪狗饲养，专门伺候我血宗弟子！”
<br/>　　平淡的话语，自他口中吐露而出，却是恶毒的令人浑身冰寒。
<br/>　　脑海中不禁浮现某种画面，青长老脸色铁青，握着千年寒冰玉盒的手掌，更是忍不住的颤抖一番。
<br/>　　正当青长老愣神的瞬间，范崂抓住机会，欺身而上，一只鲜血般赤红的手掌，爆射而出，最后狠狠的切在青长老手臂上。
<br/>　　顿时，一道清脆的断裂声凭空响起。
<br/>　　手臂传来剧痛，青长老忍不住发出一道凄厉的惨叫声，手中的玉盒不知何时已经出现在范崂的手中。
<br/>　　望着狂笑后退的范崂，青长老面如死灰。
<br/>　　快速打开玉盒，顿时一道金光射出，范崂面露狂喜，将玉盒合上，对着下方的范凌投掷过去，道：“凌儿，带着它先撤，血宗血卫，立刻护送少宗主回暮之城。”
<br/>　　范崂一声令下，范凌一把将寒玉盒抓在手中，飞快的塞进纳戒，手一挥，几十名血宗血卫脱离战圈，以范凌为中心，护送他向南方撤离。
<br/>　　“啊，范杂种，老娘今日拼了性命，也不会让你好过！”
<br/>　　阴阳玄龙丹被夺走，青长老脸色铁青，仰天发出一声凄厉的尖叫，比先前雄浑数倍的恐怖斗气，自体内铺天盖地的狂涌而出。
<br/>　　站在高坡上，慕清霄搂着小医仙，见她眼中有一丝不忍，拍了拍她的香肩，道：“临死反扑，我们也走吧。”
<br/>　　小医仙微微点头，转身之际，瞥了一眼凄惨的青长老，目光森然的范崂，美眸中满是坚定，她要将这一幕永远记在心中。
<br/>　　弱肉强食，这就是现实。
<br/>　　见状，慕清霄眼底闪过一丝欣慰，周围空间一阵扭曲，两人的身影就消失在原地。
<br/>　　......
<br/>　　......
<br/>　　约莫十分钟后，千米外的山林中，两道身影轻飘飘的落下，后面一支队伍奔袭而来，见状猛的停了下来。
<br/>　　枝繁茂密的森林中，两名身影笼罩在黑袍中，而对面的正是血宗血卫与范凌。
<br/>　　看到慕清霄与小医仙的瞬间，范凌眼神一亮，脸色阴沉，冷声道：“真是上天眷顾我，没想到会在这里遇见你们。”
<br/>　　“少宗主，来者不善，您尽快回暮之城，我等断后。”
<br/>　　其中一名斗灵级别的血卫脑子比较好使，暗骂范凌白痴，两人凭空出现在此处，显然就是冲着他们来的。
<br/>　　“走什么走，天蛇府全都死了，我们行踪神秘，根本不需要着急。”
<br/>　　话语刚落，范凌眼中满是杀意，道：“既然他们主动送上门来，倒是省的我四处搜寻，你们几个别管我，快将他们击杀，把七彩韵灵果给我抢过来。”
<br/>　　听到范凌的话语，慕清霄不屑一笑，道：“想要七彩韵灵果，你觉得自己有命用吗？”
<br/>　　闻言，范凌一愣，愣神过后，脸庞就变得极其狰狞，道：“你找死......”
<br/>　　然而，话语还未落下，一股极其恐怖的气息就以慕清霄为中心，铺天盖地的向范凌等人压迫而去。
<br/>　　霎时间，天地都仿佛倾塌，两名斗灵瞳孔剧烈收缩，浑身冷汗直冒，脑海中一片空白。
<br/>　　回过神来，鲜血飙射而出，陷入黑暗的前一秒，他们所看到的是数具熟悉的无头尸体，尸体正是他们自己。
<br/>　　恐怖的威压一放即收，范凌周围的血卫已经全部倒在地上，眨眼间就成了无头尸体，八颗头颅滚落在草丛中，鲜血染红了的地面。
<br/>　　此时，范凌呆泄的望着周围的尸体，眼中露出浓郁的惊恐，恐惧之余，一下子瘫软在地上，浑身颤抖不止。
<br/>　　这股气息，比他的父亲都不知恐怖多少倍，使得他一丝反抗心都提不起来，随即，一滩黄色液体，不受控制的从他裤裆里流淌而出。
<br/>　　“踏踏...”
<br/>　　轻微的脚步声响起，范凌抬起头来，神色惊恐，望着早已出现在面前的慕清霄，泪流满面：“前，前辈...”
<br/>　　慕清霄伸手掐住他的脖子，宛如拎小鸡崽一样，将他提到空中，语气平淡道：“脑子是个好东西，下辈子出门的时候，记得要带上。”
<br/>　　“咔嚓！”
<br/>　　话语刚落，手掌轻轻用力，伴随着一道清脆的断裂声，范凌的脑袋就无力的歪向一边，临死前，眼中透露着无尽的恐惧。
<br/>　　将纳戒自他的手指上取下，宛如垃圾一样，将范凌的尸体随手丢在一旁。
<br/>　　抹除纳戒上的灵魂印记，稍微看了一眼，慕清霄微微一愣，旋即嘴角牵起一丝笑意。
<br/>　　其中除了阴阳玄龙丹，倒还有不少的好东西，比如凤火磷果，青岩木，培髓根等等，皆是五品灵药，任何一样拿出去，都能让一堆修炼者眼红。
<br/>　　“不愧是血宗的少宗主，好东西就是不少，运气不错，仙儿，其中有几样灵药你能用上。”说罢，慕清霄取出阴阳玄龙丹后，就将纳戒递给了小医仙。
<br/>　　接过纳戒，小医仙就将它收了起来，温柔道：“范崂恐怕要追上来了，我们快些离开吧。”
<br/>　　“嗯，走吧。”
<br/>　　微微点头，慕清霄伸手搂在小医仙纤腰上，周围空间一阵扭曲，两人就凭空消失不见。
<br/>　　两人离开三分钟后，一道强横的气息落在森林中，范崂眉头皱了皱，快步来到森林空地，下一秒身躯猛的一颤。
<br/>　　周围的地面已经被鲜血染红，地上布满血卫的无头尸体，头颅落在一旁，至于范凌则倒在灌木丛中，早已没有了生息！
<br/>　　霎时间，范崂呼吸骤然变得急促起来，双眼赤红，四星斗皇的气息自他体内狂涌而出，竭斯底里，道：“谁，到底是谁杀了我的儿子！”

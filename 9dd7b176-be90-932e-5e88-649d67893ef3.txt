　　幽冥界深处，元始站在神秘的三生石旁边，眼中闪烁着紫金光芒，看向深处。
<br/>　　这是元始修成的一种天眼，名为神痕道眼。
<br/>　　神痕道眼，上，可观九霄，下，可探九幽，中间，可窥大道轨迹，作战之时，可迅速发现敌人的破绽，破阵之时，可以最快速度发现大阵玄机，炼器之时，可快速分解各种器材的构造。更胜者还可以发出道光攻击敌人，可谓是作用多多，妙用无穷，实乃万能天眼。
<br/>　　此刻，元始用上神痕道眼，看向幽冥界深处，他只感觉到了，一方世界的气息，而且还是一个正处于孕育之中的顶级中千世界，其中充斥着一股浩瀚无穷的轮回意境，以及六道巨大的旋涡！
<br/>　　那六道巨大漩涡在不断疯狂旋转着，勾连了整片洪荒大地，一切种种，除非成就大罗金仙之境，挣脱命运长河者，否则都逃不过轮回的束缚。
<br/>　　“果然，轮回早就开始孕育了，后土身化轮回，只是一个契机，使轮回出世罢了。”元始心中暗道。
<br/>　　“不过，洪荒亿万生灵，修士无数，为何会是她呢？难道她不仅仅只是一个祖巫，还有着其他特殊之处？”元始疑惑道。
<br/>　　元始思索良久，也不知其中蹊跷。最后，他在幽冥界之中转了一圈，还是没有发现孕育轮回之地，不过，却是让他发现了十八个巨大的空间正在演化，他们向着小世界的方向演化。元始知道，那是日后的十八层地狱。
<br/>　　最后，元始再次回到三生石旁，因为他总感觉此处不是那么简单。
<br/>　　想了半天，他还是没有想到什么，看着三生石，元始这才醒悟过来，为何此处给他一种奇怪的感觉？
<br/>　　因为之前他看到的幽冥界之中的种种，无论是奈何桥，还是望乡台，它们都是虚幻的，若隐若现，还没有完全孕育出来。而，眼前的三生石却是实实在在的，它已经完全的孕育成功了。
<br/>　　元始看着眼前的三生石，想到传说中的三生石。三生石是一种奇怪的东西，它既可以算作是一件特殊的先天灵宝，也可以算作是天地孕育的灵物。
<br/>　　它可以看见一个人的前世、今生与来世，三生三世，其功效可称逆天。
<br/>　　看着眼前的三生石，元始心中有一个人大胆的想法，这想法让他有些疯狂，因为从来没人这样做过，而且他也不知道此举能否成功。
<br/>　　但，这想法一出现在他的脑海之中，就疯狂的在他脑海之中扎根成长，怎么也甩不开。
<br/>　　“干了！”元始将心一横，决定做了。
<br/>　　面对三生石，元始盘腿坐下，调整好心态，元始进入到识海之中，同时，运转大道之力，用大道以天地之力凝聚成一柄巨大的虚幻道剑。
<br/>　　识海中。
<br/>　　道剑越来越大，越来越强，充斥着大道的韵味。猛然间，虚空震动。元始大喝一声：
<br/>　　“斩！”
<br/>　　顿时，道剑以一种无法阻挡的气势向着元始的元神斩去。
<br/>　　“嗽！”
<br/>　　一声轻响，元始的元始断作两半，他的元神被斩出了三分之一。
<br/>　　“分！”元始的元神猛然分离。
<br/>　　其中，那三分之一的元神快如闪电般地飞出元始肉身。神魂分出的那一刹那，元始感到一阵眩晕，飞行中的那三分之一元神也一阵震颤，三分之一元神之力对于元始来说还是很多的。
<br/>　　元始道心坚定，虽然处于眩晕无比疲倦的状态之中，可还是挺住了。
<br/>　　这一步必须快，且必须挺住。
<br/>　　否则那三分之一元神就会只有湮灭一途，神魂一旦分开，可就没法再融合了。
<br/>　　“进！”
<br/>　　三分之一元神化作流光，直接进入那三生石之中。
<br/>　　当进入三生石那一刹那，那部分元神就仿佛找到了依托，飘荡在外的小船终于回到了港湾，一种安心的感觉充斥了神魂。神魂有了载体……自然就舒服了。
<br/>　　“呼。”元始这才松了一口气。
<br/>　　“《神元九转》！”
<br/>　　元始根本顾不得其他，而是第一瞬见自己的两道元神都开始修炼《神元九转》。
<br/>　　识海中。
<br/>　　小了一号的元神界域仿佛出现了一道巨大的裂口，通过修炼《神元九转》，那部分裂口正在快速被修复，同时，元神界域也变得无比稳定。
<br/>　　四周散发着无尽温暖的神光照耀在元始神魂之上，更是令元始的神魂开始恢复了起来。
<br/>　　而在三生石之中。
<br/>　　也同样有一小号的元始神魂在其中，同样修炼《神元九转》，元神在快速恢复着。恢复的同时，元神也在与三生石契合着，等到哪一天他的元神完全恢复了，那么他便会成为真正的三生石。
<br/>　　两个元神都属于受创状态，都在快速恢复成长，这一次恢复，整整持续了三年时间，两个神魂都变大了不少。
<br/>　　“呼。”元始睁开了双眼。
<br/>　　他的元神已经恢复了七成，而且是因为他的元神已经经过了涅槃，恢复起来才简单许多，要不然，还不知道要花费多少时间呢。元神受损，最是难以恢复。
<br/>　　同时，再恢复元神的期间，元始的阴阳元神相互交织纠缠，化为阴阳鱼轮转不休，他感觉只要自己的元神一恢复，他的阴阳元神就会彻底大成。
<br/>　　果然，在破败之中崛起，经过涅槃，得到的好处更大。
<br/>　　不过，元神刚开始恢复是简单一些，若是元始没有《神元九转》，他能够让神魂变得稳固起来就不错了，更别谈让神魂快速成长了。但他就是有了《神元九转》，故而他不但稳定稳固了神魂，还恢复了之前的七成。
<br/>　　不过也仅仅只是这一次，毕竟是刚刚分离受创，所以能够一次性恢复不少。以后就慢了。元始的两大元神，以后想要恢复到巅峰，恐怕要好久。
<br/>　　不过，这也是没有办法的事，他别的不多，就是时间多，以后慢慢磨吧。
<br/>　　最重要的是，他收获了三生石。三生石中蕴含时间命运因果等法则，只要元始完全炼化之后，他就能参悟这些本源法则，之后再化形而出，他就能掌握时间法则。况且，此地是距离轮回最近的地方，他在此地会有很多机会接触到轮回，至于今后能否领悟领会，那就要看今后的造化了。
<br/>　　这才是元始的真正目的。
<br/>　　让三生石成为他的第三个分身，不得不说，这是一个疯狂的想法。
<br/>　　当然，炼化三生石成为自己的法宝也可以，但这三生石迟早要回到此处的，因为幽冥界不可能少去三生石。况且，炼作法宝，它只能被使用，发出一般功能，并不足以让元始领悟其中的规则。
<br/>　　但，让三生石成为分身就不一样了。
<br/>　　成为分身，那便意味着三生石已经成了一个独特的生灵了，他不再是一件特殊的灵宝。作为生灵，他便能够修炼，能够悟道，进而，他就有机会掌控三生法则。
<br/>　　这才是元始将三生石练成分身而不是法宝的原因。
<br/>　　元始神魂磨合三生石需要不少时间，他不准备在此多留，幽冥界还处于孕育状态，大道法则不显，并没有什么好东西。
<br/>　　看了一眼三生石中正在修复着的元神，两者心念相同，他想什么分身都知道。
<br/>　　不过，他刚要离开，便发现虚空之中出现一处轻微波动，这波动对于正在孕育的幽冥界，微不足道，但，元始还是发现了它。
<br/>　　于是，他便停了下来。

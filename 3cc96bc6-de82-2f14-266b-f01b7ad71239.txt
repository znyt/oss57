　　这一次，终于是轮到了楚风眠。
<br/>　　楚风眠脸色并无任何紧张，看了一眼那身份牌亮起，便是身躯一跃，便是踏入到了这擂台之上。
<br/>　　这还是楚风眠第一次踏入这方擂台之上，刚刚踏入其中的一刻，楚风眠便是感觉到这擂台的与众不同之处。
<br/>　　这擂台，那是这一方广场上最大的擂台，刚刚踏入其中，楚风眠便是感觉到了周围的变化。
<br/>　　周围的空间，顿时便是斗转星移，开始变化，楚风眠身边的景象，也变成了无边无际的混沌。
<br/>　　现在是这擂台上刻有这几位玄妙的禁制，让这周围的景象变化，在外看起来，这只是一座擂台，但是其中，内有乾坤。
<br/>　　在这无边无际的混沌之中，不管是多么强大的力量，都不能波及到外面，在强大的武者，都可以在其中，肆无忌惮的出手。
<br/>　　而在楚风眠面前站着的对手，却是一名红衣男子，这一袭红衣的颜色，如同鲜血一般的闪耀。
<br/>　　楚风眠看向他的一刻，脑海中便是显现出了此人的身份。
<br/>　　地魔门，罗狞。
<br/>　　这一袭红衣，正是地魔门弟子的象征，此人的境界，已经是度过了生死大劫，达到了生死境一重的境界。
<br/>　　在北域百绝榜上，排名第三十四，正好是在曾经楚风眠击败过的司空弘之上。
<br/>　　“小子，没想到居然是让我碰到了你！”
<br/>　　罗狞看着楚风眠，脸上极为自傲，一双眼睛，打量着楚风眠，丝毫没有将楚风眠放在眼中。
<br/>　　“小子，真不幸，你居然是碰到了我，你的一切奇遇，在我的面前，都会破灭，这一次跟我交手，你必败！”
<br/>　　罗狞看着楚风眠，自信的冷笑几声。
<br/>　　“一个地魔门的小魔头，也就在这个时候程程口舌之勇了。”
<br/>　　楚风眠看向这罗狞，摇了摇头道。
<br/>　　他也不知道，这罗狞如此嚣张，到底凭借的是什么。
<br/>　　楚风眠口中淡然道。
<br/>　　“出手吧，再不出手，你就没有任何出手的机会了。”
<br/>　　“狂妄！一个默默无闻的小子，得到了一些奇遇，就以为真的无法无天了？在我们地魔门面前，你的那些奇遇，根本就不值一提！”
<br/>　　罗狞怒喝一声。
<br/>　　他的背后，一把长刀，猛然出窍。
<br/>　　无数刀光，顿时迸发而出。
<br/>　　仅仅便是出窍的一刻，便是令这擂台之上，充满无尽刀锋。
<br/>　　“告诉你，这是我们地魔门的斩绝刀法，专斩天下天才，死在这刀锋下，也算是你的荣幸了！”
<br/>　　罗狞大笑一声，便是手持长刀，猛然落下。
<br/>　　这长刀落下之际，便是直接带动这周围的刀锋，凝聚一起，将所有的力量，集中到一点，直接便是冲着楚风眠的胸口，斩了过去。
<br/>　　如此凶猛的一刀，的确是让无数生死境强者，都不一定一瞬间抵挡的住。
<br/>　　这一刀，毕竟太快的。
<br/>　　刀锋向着楚风眠袭来的一刻，楚风眠甚至还站在原地，一动不动，似乎没有反应。
<br/>　　这一战，几乎是引得无数周围武者，死死的盯着观看，其中的一举一动，他们都是观看的一清二楚。
<br/>　　这可是楚风眠这批“黑马”，在擂台战上的第一战，地魔门的弟子，许多可都是极为凶残之人，他们在北域百绝榜上的排位并不高，但是并不代表着他们的实力弱。
<br/>　　相反，这些人在以命相搏的时候，甚至是要远超其他宗门的天才。
<br/>　　擂台战，是生死之战，也并非是力量比拼。
<br/>　　罗狞在北域百绝榜上排位不高，但是让一些北域百绝榜上排名前二十的武者，面对他，都要极为忌惮。
<br/>　　起码这一刀，是绝对的惊艳一刀，寻常武者没有反应过来，都要会被斩与马下。
<br/>　　“罗狞的实力，又强大了许多，看来再次之前，他只是在隐藏实力，在这武斗大会上，终于是展现出了自己的实力！”
<br/>　　“地魔门的人，在生死之战上，的确极为强大，在地魔门中，每时每刻都会发生生死之战，地魔门武者，哪一个不是沾满了鲜血，经历过无数次的生死之战，这楚风眠的实力虽然强大，却还是有些太稚嫩了。”
<br/>　　“这一战，只怕楚风眠是要败了，他的实力的确强大，但是空有一身蛮力，却并不足以应对这种生死之战，看他现在的样子，都要吓傻了，这一刀之下，就要结束了。”
<br/>　　无数武者，看着到现在楚风眠还没有任何反应，都是摇了摇头。
<br/>　　在他们看来，应该就是楚风眠没有经历过如此凶猛的生死之战，毕竟楚风眠，才是一个新人，刚刚冒出头来，只是得到了一些奇遇，才有着如今的力量。
<br/>　　而北域百绝榜上的武者，哪一个不是经历过无数次的生死之战，他们的排名，都是经历过无数生死，才得到的。
<br/>　　罗狞更是地魔门的人，这一招出手，凶狠无比。
<br/>　　现在的楚风眠，还没有反应，那这一刀之下，楚风眠则是非死即伤。
<br/>　　就在众人议论之际，突然无数人的眼睛，都是瞪大了。
<br/>　　就在这刀锋，在接近到楚风眠胸口的一刻，楚风眠的手，却是动了，他的手掌，直接便是抓了上去，一抓之下，居然是将这刀锋，都给抓在了手中。
<br/>　　就连那罗狞的表情，都是一下子变得极为呆滞，似乎是有些无法理解，眼前这一幕。
<br/>　　他的刀锋，居然是被楚风眠徒手接下了，这刀锋上的力量，让一尊生死境巅峰的强者前来，都不敢徒手去接。
<br/>　　除非是那些真正超越生死的大人物，才能做到这一点。
<br/>　　他的心中一万个想不到，楚风眠居然是会徒手，接下来他的刀锋，甚至是脸色，都是轻松无比。
<br/>　　“该死！”
<br/>　　罗狞的表情，变得无比狰狞，似乎是真正的愤怒了。
<br/>　　“小子，很好，你彻底的激怒了我！现在你已经没有其他的结果了，只有死，以你的鲜血，才可以平息我的怒火！”
<br/>　　罗狞狂吼一声。
<br/>　　突然之间，罗狞的身躯开始猛烈震动起来，他身上的力量，突然直接开始疯狂攀升，足足是攀升了十倍的力量。
<br/>　　就在这一瞬间，罗狞的力量，增长了十倍。

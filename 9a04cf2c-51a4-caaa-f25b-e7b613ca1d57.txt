　　“奇犽，我们比一比，看谁先到达终点吧？”小杰兴致勃勃提议。
<br/>　　“好！”奇犽兴奋的点头。
<br/>　　两人直接无视了跟在身后的苏重，迈开大步就跑了起来。不一会儿，就超过了第一梯队的众人，竟然快速无比的占据了第一位置！
<br/>　　“果然不愧是主角，这身体素质，甚至比成年人还要强大！”苏重感叹。
<br/>　　奇犽从小接受揍敌客训练，有这种体制不奇怪。小杰可一直在鲸鱼岛生活，一直处于散养状态，可身体素质同样强大。
<br/>　　“天赋出众。”苏重感叹，不愧是主角。
<br/>　　“生活环境影响同样不可小看。”苏重仔细思考小杰的背景。
<br/>　　在参加猎人测试前，小杰一直生活在安宁祥和的鲸鱼岛。不同于其他同龄人，小杰从很早就混迹于森林之内。
<br/>　　那里有着各种各样的强悍猛兽，正是因为从小候选与这些猛兽之间，让小杰在不知不觉之间得到了锻炼。
<br/>　　强大的身体素质，宛若野兽一般的敏锐直觉。
<br/>　　“和猛兽追逐嬉戏，将锻炼融入游戏之中，潜移默化的强健体魄淬炼心智。很高明的方法。”苏重面露赞赏。
<br/>　　相比于健身房内的重复训练，这种寓教于乐的方法更能激发主动性。
<br/>　　“可惜，这也只能是主角的待遇了。如果是一般人，早就死在那些凶猛怪兽利爪之下了。”
<br/>　　优哉游哉，苏重一路跟着众人，走出地下隧道，重见天日。
<br/>　　入目所见，一片白色大雾笼罩四周。空气中充斥着一股潮湿味道，各种怪异鸣叫声，从树林内不断传来。
<br/>　　“这里是达梅尼湿地，拥有诸多危险生物，你们要做的很简单。跟上我，安全到达目的地。”萨次不紧不慢一派从容道。
<br/>　　刚刚放松下来的一众人，顿时紧张起来。左瞧瞧又看看，试图找到隐藏在浓雾之后的危险生物。
<br/>　　“你们不要被他骗了，我才是考官，他是变形魔兽变化出来的，你们要小心！”两个伤痕累累的人，突然从众人身后冲出，指着远处的萨次大吼大叫。
<br/>　　一众人顿时犹疑起来，他们也无法判断，到底哪个才是真正的考官。
<br/>　　苏重知道，那两个家伙是变形魔兽人面猿。能够变化形体，模仿人形。如果刚来到这个世界时，他就来参加这次考试，苏重会觉得很有意思。可以他如今的实力，这所谓的测试，总让他感觉儿戏。
<br/>　　而且，两只小小的变形魔兽，竟然敢在他面前乱晃！几乎本能的，一股怒意骤然产生。
<br/>　　吼！
<br/>　　一道手臂粗光柱骤然从苏重背后虚空射出，光柱好似一条岩浆组成的钢鞭，直接对着两只变形魔兽横扫而过。
<br/>　　砰！
<br/>　　两只人面猿根本来不及反应，就被黑红光鞭扫中。恐怖的高温腐蚀力量，瞬间蔓延全身。下一刻就伴随着微风化作一片黑色灰尘。
<br/>　　全场一片安静，全部满脸恐惧的看着苏重。一言不合就动手，而且一动手就打成粉末，这也太凶残了吧！
<br/>　　“奇犽，你哥哥好凶啊！”小杰满脸凝重。
<br/>　　他感知敏锐，就在刚才，苏重给他的压迫感，比森林中最厉害的狐熊还要强大数倍。
<br/>　　西索浑身一抖，饶是他心性邪意，对挑战强者十分狂热，也被苏重这突然一击搞的有些发蒙。
<br/>　　易地而处，西索觉得，自己的下场不一定比那两头人面猿好到那里去。刚才一瞬间，他甚至没反应过来。苏重的攻击速度太快了。
<br/>　　“快点开始吧。”苏重对着全身僵硬的萨次催促道。
<br/>　　“还是进阶的太快了，果然受到了魔龙意志的影响。”他以前可没这种一言不合就开打的霸道。
<br/>　　萨次不再拖延，当先走进浓雾之中。众人紧随其后，知道这里有各种危险魔兽，没人敢大意。
<br/>　　苏重依旧是不紧不慢的吊在最后，也不去管已经开始走入岔道的小杰和奇犽。
<br/>　　这里的各种危险情景，对他来说如同儿戏，但对奇犽、小杰两个初出茅庐的新人来说，是个不错的历练场所。
<br/>　　摇摇锁定萨次，苏重慢悠悠的走着。沿途遇到任何阻碍物，气魄光膜闪烁之间，就被焚烧殆尽。
<br/>　　魔龙法相大成，气魄威力骤然提升。湿地内的恐怖魔兽，对他来说不堪一击。
<br/>　　“到地方了，接下来就不是我负责了。”萨次对身后的苏重道，说完也不管苏重，转身就走，背影里总有种落荒而逃的感觉。
<br/>　　“萨次，这么快就来了。”卜哈拉挺着肥胖的大肚子，笑呵呵道。
<br/>　　卜哈拉身高两米多，是一个不折不扣的大胖子，肚子高高凸起，眼睛都快要被肥肉挤的消失。脸上始终带着憨厚笑容。
<br/>　　“如果你在身后被一个怪物盯着，你也能走那么快。”萨次一脸没好气。
<br/>　　“怪物？这次考生有什么特别的吗？”门琪眼中满是好奇。
<br/>　　她穿着热裤，上衣也极为清凉。头发扎着三个直立辫子，整个人洋溢着一股热情活泼。
<br/>　　两人都是美食猎人，为了各种美味食材，出入深山密林，也活跃于高档餐厅。为的是追求食物味道的极致。
<br/>　　“怎么说呢，这一次，特别的考生有不少。”萨次想到了那两个年轻的小鬼，还有那个满身邪气的魔术师。人群之中，同样也隐藏着一些气质独特之人。
<br/>　　“不过最特别，或者说最危险的一个，就在外面。”那突入起来的攻击。直到现在，他依然不知道那种攻击是什么。
<br/>　　“危险？”门琪跃跃欲试。每一个猎人，都有着旺盛的好奇心。
<br/>　　“我劝你最好不要尝试。”萨次肃然道：“我在他身上，感觉到了危险气息。”
<br/>　　门琪一怔：“你感觉到了危险？”
<br/>　　他们可是职业猎人，掌握着独特的念能力。每一个猎人，可都是能够以一敌百的强大之人。
<br/>　　而向他们这种多年的职业猎人，实力更是强大无比，每人都有着自己底牌。
<br/>　　“虽然只是一瞬间的气息，但却已经给了我一种毛骨悚然的感觉。”萨次皱着眉头。
<br/>　　“他也是念能力者？”门琪了然。
<br/>　　萨次摇头：“不只是念。说不清楚，那种无形的压迫，很奇特。”他没开口说，那种骤然间的压力，他只在协会会长尼特罗身上感受到过。

<br/>				<br/>　　    那狗尾巴草的根茎只是比针稍大一点点，更重要的是，那是根草，不是针啊！可居然那样一根随便一折便断的狗尾巴草却穿刺过了那条毒蛇的身体，这、这得拥有多雄厚的内息才能做得到这一点？<br/>　　<br/>　　    一时间，他看着凤九半响说不出一句话来。原本他想不通被他所忽略的事情也一一的再度浮现在脑海里。<br/>　　<br/>　　    为什么在这布满凶兽的林子里，那些凶兽却在盯着他们看了一会后止步不前掉头就走？为何只要凤九在他们身边，他们就不会遇到危险？为什么他们掉进了这里，还能活到现在？<br/>　　<br/>　　    这一幕幕的脑海中一一掠过，想到这一路上他为他们暗中解决掉的危险，他只感觉心头一震。<br/>　　<br/>　　    难怪，难怪就算掉进这陌生而又充斥着危险的地方他也不害怕不恐慌，在这样的一个环境里，能做到不害怕不恐慌的，那只有一个原因，那就是他有足够的自保能力。<br/>　　<br/>　　    凤九不知他在想什么，只是走上前，取出小刀将毒蛇的蛇胆取了出来，装进了一个小瓶之中，见他还站着没动，她微讶，看了他一眼：“怎么了？”<br/>　　<br/>　　    谢玉堂背着他妹妹，两人就这么看着呆呆的凤九，半响，谢玉堂微张着嘴，本想询问的话语到最后却成了：“我、我有些渴了，想坐下歇会喝口水。”<br/>　　<br/>　　    “嗯，那前面吧！到那树下歇着。”她指着前面一块空旷的地方说着，声音一落，便率先迈步走上前去。<br/>　　<br/>　　    谢玉堂连忙背着他妹妹跟了上去，到了那里才放了她下来，取出水连喝了几口，这才看了凤九一眼后又低下了头，不知在想着什么。<br/>　　<br/>　　    “嗯？”凤九忽的挑了挑眉站了起来。<br/>　　<br/>　　    “怎么了？”谢玉堂见此，也紧跟着站了起来，手中握着剑警惕着，一边将他妹妹护在身后。<br/>　　<br/>　　    “好像有人。”凤九说着，神识释放而出，朝一处地方探去，不多时，收回神识，露出了笑容：“看来，可以离开这里了。”<br/>　　<br/>　　    “有人？是什么人？是不是阮家派来找我们的人？”谢玉堂问着。<br/>　　<br/>　　    “不是，应该是一些来此历练的家族子弟。”<br/>　　<br/>　　    凤九说着，道：“既然有历练的家族子弟来到这里，那就证明这地方除了那传送阵法之外，还是有别的方法离开这里的，走吧！我们过去问问。”<br/>　　<br/>　　    听到他的话，谢玉堂连忙背起他妹妹跟着他往那方向走去。<br/>　　<br/>　　    在离此约莫百米的地方，一名老者和中年男子不约而同的睁开了眼睛，朝一方向看去，同时，中年男子手中做了个手势，原本坐着说话的众人迅速的站了起来，做出防备的姿势。<br/>　　<br/>　　    不多时，当他们看到沙沙声的杂草丛后走出的身影时，不由一怔，其中，前面的一名年轻男子厉声喝着：“什么人！”<br/>　　<br/>　　    看到真的有人，谢家兄妹两人十分欢喜，他们看了凤九一眼，便道：“我们是谢家的人，因为……”话还没说完，就让凤九打断了。<br/>　　<br/>　　    “因为与家族的人走失了，在这里面迷了路。”凤九接过话说着。<br/>

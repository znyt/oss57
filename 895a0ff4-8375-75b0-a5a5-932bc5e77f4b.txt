　　如此数量，哪怕斗皇巅峰强者，都只有转身就逃的份，慕清霄眼中却闪烁着兴奋的光芒。
<br/>　　在他看来，这群岩浆生物，并非生命，而是一颗颗精纯的火属性能量珠。
<br/>　　四百多颗火属性能量珠，其中六名斗王，两名斗皇，斗灵不计其数，珠子中蕴含的能量集中在一起，绝对是一个恐怖的量。
<br/>　　嘴角牵起一丝笑意，慕清霄道：“就是不知道，能不能让自己实力提升一些？”
<br/>　　就在沉吟之际，蜥蜴人已经蜂拥而至，将他团团围住，细小的瞳孔中，泛着残暴与凶戾之色。
<br/>　　抬起头来，慕清霄率先动手，一道道剑气凝聚，以迅雷不及掩耳之势，冲进蜥蜴人群体之中。
<br/>　　伴随着周围岩浆暗流一阵涌动，周围响起一连串沉闷而凄厉的惨叫声。
<br/>　　只见慕清霄穿梭于蜥蜴人群体中，犹如虎入羊群，一个个强大的蜥蜴人，被剑气切成两半，化作精纯的能量珠。
<br/>　　片刻后，岩浆上空浮着一片赤红色能量珠，而剩下的蜥蜴人寥寥无几，瞳孔中满是凶戾，却不敢上前。
<br/>　　虽然蜥蜴人群体数量众多，但对于慕清霄来说，都是一些不堪一击的蝼蚁，唯有两名斗皇级别，值得他瞧上一眼。
<br/>　　“咻！”
<br/>　　就在这时，约莫七八米开外，响起一阵沉闷的破风声。
<br/>　　只见，一只体形硕大的蜥蜴人，狰狞的大嘴前，凝聚成一颗头颅大小的岩浆能量球，其中蕴含着恐怖的力量。
<br/>　　这股力量，堪比斗皇巅峰的全力一击。
<br/>　　慕清霄剑眉一皱，眼中寒芒一闪，心念一动，魂剑已经出现在手中，身形突兀自原地消失。
<br/>　　岩浆空间一阵扭曲，他的身影就已经出现在那名八星斗皇的蜥蜴人身后。
<br/>　　蜥蜴人在岩浆中极其敏锐，察觉到身后的异样，狰狞的大嘴中低吼一声，硕大的拳头，直接向慕清霄怒砸而去。
<br/>　　“蠢货！”
<br/>　　嘴角泛起一丝冷笑，寒芒一闪，魂剑剑刃就穿过蜥蜴人的腰身...
<br/>　　“铮...！”
<br/>　　周围响起一阵沉闷的剑鸣声，随着蜥蜴人身躯一颤，细小的瞳孔中透露着一抹惊恐，生机消散，最后留下一颗赤红的珠子。
<br/>　　珠子明显比斗灵，或者斗王级别的蜥蜴人色泽光亮的多，其中所蕴含的火属性能量，更加的庞大精纯。
<br/>　　见慕清霄一剑将同伴斩杀，另外一只斗皇蜥蜴人见着，细小的瞳孔中满是惊恐，模样极为滑稽。
<br/>　　随后，尾巴一甩，在岩浆中荡起一阵涟漪，转身就要逃走。
<br/>　　“既然来了，就留下吧。”
<br/>　　冷笑一声，空间一阵扭曲，慕清霄就持剑出现在了蜥蜴人的前面，拦住了它的去路。
<br/>　　知道慕清霄不会放过自己，斗皇蜥蜴人细小的瞳孔瞬间变得赤红，张开狰狞的大嘴，尖锐的声波，瞬间传出。
<br/>　　微微一愣，慕清霄感觉脑子恍惚了一秒......
<br/>　　瞬间，一只利爪击破灵气护罩，刺进了他的身体，腹部传来一阵刺痛，金色的血液流淌而下。
<br/>　　嗅到他血液中的气息，蜥蜴人细小的瞳孔中，毫不掩饰的透露出残忍与贪婪，它能感觉出，面前这名人类，血脉极强，若是吃掉...
<br/>　　“铮！”
<br/>　　手起剑落，寒芒一闪，蜥蜴人还未反应过来，粗壮的手臂就被斩断，嘴中发出一道凄厉的惨叫声。
<br/>　　面无表情的将手臂丢下，慕清霄抹去嘴角的血迹，看了眼腹部伤口，视线再次落在惊恐的蜥蜴人身上。
<br/>　　“灵魂攻击，真是没想到，一时大意，竟然中了你的招。”
<br/>　　原本，慕清霄本就是抱着戏耍的态度，与这群蜥蜴人厮杀，回过神想想，实在是最近太安稳了。
<br/>　　狮子搏兔亦用全力，因为不屑，所以轻敌大意，以至于现在受伤，好在蜥蜴人只有七星斗皇的实力，根本难以将他重创。
<br/>　　若是斗宗级别，哪怕慕清霄拥有神凤血脉，恢复力变态，恐怕在这一击下，伤势也不会太轻。
<br/>　　而蜥蜴人的一击，将他及时打醒了过来！
<br/>　　否则，日后遇上其他对手，依旧保持着这种态度，阴沟里翻船的可能性会很大啊。
<br/>　　瞥了一眼惊恐后退的蜥蜴人，慕清霄手猛的一挥，恐怖的剑意搅动岩浆，剑气脱剑而出，在岩浆中留下一道剑痕。
<br/>　　“呃呃...”
<br/>　　脸上的表情凝固，蜥蜴人的脖子出浮现一道剑痕，旋即，身首分离，身躯如水波一样消散，最后留下一颗色泽明亮的赤红色珠子。
<br/>　　当最后一只蜥蜴人死去，这一片区域，再度变得死寂一片。
<br/>　　将魂剑收起，黑色神火自慕清霄腹部冒出，伤势以肉眼可见的速度恢复起来，手一卷，将浮在岩浆中的能量珠，全都收进储存空间中。
<br/>　　几分钟后，伤势完全恢复，慕清霄便向远处的陨落心炎掠去。
<br/>　　就在他接近陨落心炎的时候，脚步一顿，瞥了眼透明的光罩，光罩好似是异火自我保护的方式，却不妨碍他穿过。
<br/>　　穿过结界的时候，他明显感觉到，一股异样的波动自身上扫描而过，随后悄然退去，慕清霄也是顺利进入其中。

<br/>　　目光在其中扫视一圈，慕清霄的视线，最后停留在了中央位置，那悬浮的骨骸上。

<br/>　　只见它们在四周徘徊了片刻，似乎并没有发现光罩所在，最后只能离去。
<br/>　　慕清霄并不准备出去将之击杀，刚才已经占了大便宜，若是将这两只斗宗级别的蜥蜴人击杀，谁知道会不会引来更强的。
<br/>　　比如，斗尊！
<br/>　　收起心思，慕清霄转过身来，迈着步伐，不急不缓的向中央位置的骨骸走去。
<br/>　　靠近骨骸后，先前那若有若无的召唤，果然是来自于面前的陨落心炎，目光扫了一眼骨骸，视线最后停留在其上的无形火焰上。

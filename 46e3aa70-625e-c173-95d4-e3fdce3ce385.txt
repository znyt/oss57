　　苏莫离开火元圣师的宫殿之后，便回到了自己的洞府之中。
<br/>　　有了火元圣师的话，他也稍稍放宽了心，准备安心修炼。
<br/>　　洞府中。
<br/>　　苏莫取出了购买的五门属性功法，详细的参悟了起来。
<br/>　　这五门属性功法，分别为：《十方斩灵功》《生生不息诀》《水仙录》《三千炎焱诀》《地藏神功》。
<br/>　　五门功法囊括了金木水火土五行属性，刚好完全可以替换他修炼的五门帝级功法。
<br/>　　苏莫首先参悟的水属性功法《水仙录》，因为他不确定大混沌五行神诀能否融合五门尊级中品功法，所以要先修炼水属性的功法。
<br/>　　其它的四种属性功法，木火属性和土金属性，四种玄力可以两两融合，唯独水属性不能融合，若是修炼这四种属性中任意一种属性，则会对他的实力有影响。
<br/>　　所以修炼水属性的功法，对于他的实力不会有太大的影响，就算有影响也是实力增加，不会减弱。
<br/>　　尊级的功法非常的玄妙，苏莫静心参悟，整整参悟了三天，才有了大概的理解。
<br/>　　而后，他便开始修炼水仙录。
<br/>　　水仙录分为五层境界，他从第一层开始修炼了起来，功法催动，玄力运转。
<br/>　　体内水属性玄力缓慢的转变，变得更加浑厚、更加强大，这个过程非常的漫长，一日又一日。
<br/>　　过去了大概八日的时间，苏莫内体的玄力便转化的差不多了。
<br/>　　当然，这并不代表他修炼成了水仙录第一层境界，只是完成了功法的转换，算是开始入门了。
<br/>　　“好像没有什么异常？”苏莫眼眸闪亮，因为完成了功法的转换之后，五种功法的玄力，并没有什么冲突，非常的自然。
<br/>　　随即，他小心翼翼的运转了大混沌五行神诀，五种功法顿时齐齐运转了起来，五行玄力如同行云流水，顺畅自然。
<br/>　　“真的可以！”苏莫心中大喜，这说明大混沌五行神诀可以融合尊级中品的功法。
<br/>　　不过，他也没有太过兴奋，现在他也不能绝对的肯定，因为其它四门功法，还只是帝级的功法，没有完成转换。
<br/>　　苏莫松了口气，这绝对是个好苗头，既然水仙录能顺畅的和其它功法一起运行，想必大混沌五行神诀，真的可以完美的融合五门尊级功法。
<br/>　　这样的话，他也不用修炼混天魔功了，大混沌五行神诀融合五门尊级中品功法之后，战斗力绝对恐怖。
<br/>　　随即，苏莫便不再多想，安心的修炼水仙录。
<br/>　　就这样，又安静的修炼了两日之后，任云都的呼唤声突然在外面响了起来：“苏兄！”
<br/>　　苏莫闻言睁开了眼眸，灵识一探，发现任云都正立身在火元峰之前的虚空中。
<br/>　　“任兄，请进来一坐！”苏莫走了出去，向对方发出邀请。
<br/>　　任云都没有拒绝，而后便飞落在了洞府前，和苏莫一起走进了洞府中。
<br/>　　“苏兄，恭喜！”
<br/>　　走进洞府，两人在石桌前相对而坐，任云都满脸笑意的向苏莫抱拳。
<br/>　　“任兄，你这是？”苏莫疑惑，不知道对方恭喜什么？
<br/>　　“呵呵，苏兄，你被排在了真圣榜第一，现在整个古周星都已经传炸了，几乎是无人不知无人不晓，你要名震古灵星河了，难道还不值得恭喜吗？”任云都笑道。
<br/>　　呃~
<br/>　　苏莫闻言无语，若不是这个真圣榜，他哪里会这么被动！
<br/>　　“任兄，这个真圣榜，是将我置在了风口浪尖啊！”苏莫叹了口气道。
<br/>　　“说的也是，现在学府外面还聚集了不少天才，扬言要挑战你呢！”
<br/>　　任云都了然的点了点头，苏莫现在修为毕竟不高，直接排到真圣榜第一，的确是置在了风口浪尖。
<br/>　　“还有这事？”苏莫闻言愕然，学府外面有很多人要挑战自己？
<br/>　　“是啊！这些人都想要挑战你呢！不过他们并不能进入学府中，你一直躲在内府，自然不知道！”任云都道。
<br/>　　“嗯，这些人不必理会！”苏莫无奈的摇了摇头，他可没有心思应付别人的挑战。
<br/>　　“苏兄，我此次来此，是之前受了圣子殿下之命，有一事和你商议！”任云都脸色郑重了下来，沉声说道。
<br/>　　“何事？”苏莫见对方脸色郑重，有些疑惑了起来，难道有什么大事不成。
<br/>　　“你两位朋友的事情，你知道吗？”任云都问道。
<br/>　　“两位朋友？”
<br/>　　苏莫闻言微怔，随即惊疑的问道：“你说的是聂乾和韩盈？”
<br/>　　“嗯！看来你还不知道，他们两人被执法堂关押了！”任云都说道。
<br/>　　“什么？怎么回事？”苏莫闻听此言，顿时脸色一变，站了起来，大声的问道。
<br/>　　聂乾和韩盈被执法堂关押了？
<br/>　　怎么会这样？
<br/>　　难怪两人不在，洞府中都落满了灰尘！
<br/>　　任云都叹了口气，道：“尤天韩说聂乾和韩盈设谋坑死了赵城和郭剑青，向执法堂举证，然后执法堂将他们两人关在了镇魂之地！”
<br/>　　“可恶！”苏莫心中大怒，赵城和郭剑青明明是他杀的，尤天韩居然污蔑到了聂乾和韩盈身上。
<br/>　　这不是摆明了报复他吗？
<br/>　　太史世家派系对付不了他，居然将矛头指向了聂乾和韩盈。
<br/>　　“苏兄，你也不必担心，只是关押五年而已！”任云都安慰道。
<br/>　　“镇魂之地是什么地方？”苏莫沉声问道。
<br/>　　“镇魂之地是学府惩罚弟子的地方，在其中会承受镇魂之苦，每时每刻都如同炼狱一般！”任云都说道。
<br/>　　苏莫闻言脸色阴沉了下来，心中的杀机缓缓涌动，这个惩罚未免太过严重了！
<br/>　　虽然没有生命危险，但承受五年的镇魂之苦，这会把人折磨疯了的。
<br/>　　“苏兄，你切勿冲动，一切等圣子殿下回来再说！”任云都告诫道，他担心苏莫会直接去找尤天韩报仇。
<br/>　　“圣子什么时候回来？”苏莫沉声问道，脸色越发的难看，这尤天韩简直找死，太史世家派系真的是要和他不死不休了吗？
<br/>　　“圣子有事外出了，过几日便能回来！”任云都说道。
<br/>　　苏莫沉默了片刻，随即深深的吸了口气，压制了心中杀机，点了点头，道：“好，我再等几日，等圣子回来！”
<br/>　　他并没有冲动，聂乾和韩盈被关押了，他总不可能直接杀去执法堂吧！
<br/>　　他现在面对的处境颇为艰难，外有巫族的杀机，内有太史世家派系的仇视，只能先等古天意回来再说。

<br/>        其实，他已经猜到了，只是不确定。
<br/>
<br/>        “是筱颖……”傅夜沉黑眸深邃，自言自语地唤了声。
<br/>
<br/>        就在李忆莎未听清楚他后来所说的话时，只见傅夜沉急着连忙站起身。
<br/>
<br/>        李忆莎下意识地问道：“你要去哪？”
<br/>

<br/>
<br/>        李忆莎忽然庆幸自己，能看清楚这个事实。
<br/>
<br/>        与其嫁给不爱自己的男人，倒不如嫁给一个深爱自己的男人。
<br/>
<br/>        李忆莎看着傅夜沉那帅气的背影消失在屏风的转角后，默默地潸然泪下。

<br/>        “你好，这位美丽的小姐，可否与你拼个桌？”

<br/>        李忆莎稍稍抹掉眼泪，循声望去，抬眸看着已经站在桌边等她答复的男人。

<br/>        或许，上帝还是眷顾她的，把她的窗户关上后，又给她开了另一扇心灵之窗。

<br/>        翌日。

<br/>        对于李忆莎悔婚的消息，落以柔得知后，心里虽然开心不已，但是又开始忙得交头接耳。
<br/>
<br/>        之前为婚礼所准备的事，现在得一一由她去跑腿全部取消。
<br/>
<br/>        啊啊啊！真的是太折磨她了！

<br/>        郑忠威给落以柔倒了杯果茶，递给她后，微微耸肩，打趣地说：“十有八九，是被傅老太爷叫回去训话了！毕竟，那位李大小姐悔婚，跟大少爷脱不了干系。”

<br/>        郑忠威端着自己手里茶，大口大口地吞咽后，抬起手来比划道：“你休息一会，记得到我办公室里来拿文件，然后送去财务部。”

<br/>        “好的，师父！”落以柔顿时干劲十足，下意识地望向了大厦窗外。

<br/>        今天她心情真的特别好，但今天却不是一个好天气。

<br/>        雨，还在下……

<br/>        因为乌云密布，城市被阴暗的雾霭给笼罩。

<br/>        绵绵细雨，不停兜头落下，让视野显得朦胧不清。

<br/>        行人脚步匆匆，撑伞踩踏过雨水，车辆行驶于车水马龙的大马路，车轮辗过低洼地方，溅起脏污雨水。
